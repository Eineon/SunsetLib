## 基础

- **临时变量**：许多机制会以［N(编号)］的格式存储可变化的值。若未指定变量的具体值，则由使用者自行决定变量代表的值。不同机制的同编号变量不会相互引用。
- **武器伤害骰**：伤害表述中的［[N]WD］代表将相关武器的伤害骰投掷N次并将出目相加，固定值与额外伤害骰不受影响。
- **主副手**：创建角色时，选择1只手作为主手，其他手为副手。
- **敌对行为**：敌对行为是指迫使他人遭受非自愿危害的行为。例如，将火球术施放到人群中是敌对行为，但是打开门时意外放出了一个可怕的怪物并不是。具体由DM裁定。
- **失去HP**：直接导致「失去HP」的效果不会被视为伤害。
- **损伤**：损伤代表你已失去的HP。消除损伤等同于恢复HP。
- **斩杀**：意义相当于「将**有威胁**的生物的HP降至0时触发效果」。每场战斗中，同一名生物只能触发1次**斩杀**。是否**有威胁**由DM裁定。 \
  当你对一名生物造成至少1点损伤后，若该生物在1轮内被你的盟友**斩杀**，也将其视为被你**斩杀**。
- **咏唱**：记作「**咏唱_N**」，意义相当于「当咏唱值到达N点时，以及之后每高出1点，都会自动触发1次效果」。分别处理不同来源的咏唱值。
- **血术**：记作「**血术_N**」。意义相当于「消耗并投掷N枚 (至少1枚) HD，再失去等于掷骰结果的HP，随后触发效果」，这不会使你的HP低于1。
- **谐振**：以「谐振」作为持续时间的效果将持续10分钟，并会在你参与的遭遇结束时提前结束。
- **架势**：发动标注为「架势」的效果时，之前发动的架势将提前结束。你无法在遭遇之外发动架势。 \
  如果一个架势拥有任何条件，其效果也会在你无法满足条件时提前结束。
- **韧性**：替换原本的临时生命。韧性可以互相叠加，但拥有名称的韧性不会与同名韧性叠加。职业特性与魔法物品提供的韧性默认为增强韧性。
- **底力**：替换原本的英雄激励。你最多能够同时拥有6点底力，可以重掷已经重掷过的检定。当你完成长休时，底力变为3点。 \
  此外，你可以在自身的1次行动检定掷骰后，消耗1点底力，使检定获得［1PD］点加值。只要底力充足，你可以对同一次检定重复使用这个效果。

### 规则调整
- 优先使用5E2024的资源。你可以向DM申请使用旧版资源。
- 无法使用TCE的额外职业选项。
- 使用**格**取代**尺**作为长度单位，1格为5尺，也等同1.5米。
- 掷出N1的检定将受到10点减值；掷出N20的检定将获得10点加值。
- 黑暗视觉的范围等同视觉范围。
- 原本的固定DC提升30%，向上取整。
- 背景与职业在人物第1级提供的技能熟练分别降低2项，不足2项则用工具熟练代替。创建角色时，你可以自选合计4项技能与工具获得熟练。
- 智力值到达13点，以及之后每提升4点，你都可以任选1项技能或工具获得熟练，并掌握1门非秘密语言。
- 移除：先攻检定，死亡豁免，阵营，魔法物品稀有度。

### 运算规则

- **最低值**：数值最低通常为0，不会被降至负值。
- **截尾取整**：计算结果出现小数时，通常会舍去小数的部分。例如：3.7取3，-1.2取-1。
- **运算顺序**：对同一数值进行多次运算时，先进行加减，再进行乘除；先进行整数之间的计算，再进行与分数的计算，最后进行与百分比的计算。
- **分数运算**：数值与分数进行乘除前，先将该数值向下取整至分母的最近倍数。例如：分母为4、数值为10时，先将10取为8，再进行计算，不会产生余数。
- **百分比运算**：数值与百分比进行运算时，先将所有百分比与100%相加，得到**全局系数**，再用全局系数乘以原数值。例如：数值增加20%共2次，全局系数等于［100%+20%+20%］。
- **减半**：对数值的减半不属于百分比运算，并且总是在运算的最后进行。同一数值最多只能被减半1次。

### 掷骰
掷骰记作［[N1]D[N2]］，意义相当于「从［1~N2］中随机抽取1个整数，重复［N1］次」。 \
［N1］代表骰子的枚数，［N2］代表骰子的尺寸。抽取的结果称为**出目**，出目的总和称为**骰值**。

只要所用工具能够保证每个整数在理论上的出现概率近似相等，即可用于掷骰，而不限于实体骰子。 \
在本文档中，**骰子**泛指用于生成随机整数的工具，**掷骰**指进行该随机抽取的行为。

对掷骰公式进行的乘法・除法运算，只会将乘数・除数与骰子的枚数做运算，骰子的出目与骰值不会参与运算。
- **翻面**：将出目改为［尺寸+1-原值］。例如，对1D6掷出的2进行翻面，出目将变为5。
- **重掷**：在看到骰子的出目后，重新投掷骰子，以新结果覆盖旧结果。一枚骰子通常只能重掷1次。 \
  给予重掷的效果若没有标注「可以」重掷，则代表掷骰者**必须**进行重掷。 \
  在重掷多枚骰子中的一部分时，由掷骰者决定重掷哪些骰子。
- **冠名调整值**：拥有名称的调整值不会与同名调整值叠加，包括各项属性的调整值。原本的职业特性与魔法物品提供的加值默认为增强加值。

#### 支援与妨碍
- 支援的强度上限为4。当一次检定拥有支援时，检定者需要在投掷D20后再投掷［[强度]D4］，将结果总和作为支援加值应用在判定值上。
- 妨碍的强度上限为4。当一次检定拥有妨碍时，检定者需要在投掷D20后再投掷［[强度]D4］，将结果总和作为妨碍减值应用在判定值上。
- 同时拥有支援与妨碍时，两者的强度会互相抵消，并减少本次检定的支援与妨碍强度上限。

#### 冠名支援与妨碍
拥有名称的支援/妨碍不会与同名支援/妨碍叠加，强度较低的一方会被压制。

#### 熟练加值与阶位加值
- PB与RB都属于无法与同名调整值叠加的冠名调整值。
- PB是一个固定值，计算方式为［⌈人物等级÷4⌉×2］。RB的数值由给予RB的效果决定。
- 原本使用次数等于PB的特性，使用次数改为3次。如果该特性不会在完成短休时进行恢复，现在则会在完成短休时恢复1次使用次数。
- 当你进行拥有熟练的行动检定时，会获得［人物等级÷2］点RB。
- 如果在一次行动检定中同时拥有技能与工具的熟练，你不会获得优势，改为获得1点支援。若有专精，则改为获得2点。
- 未着装护甲・着装受训护甲时，你可以将PB降低2点加入AC，并获得［人物等级÷2］点RB。
- 如果一个效果原本允许你将PB加入AC，现在只会将AC上的PB增加2点。11级起，改为增加4点。
- 如果一个效果原本允许你加入双倍PB，现在只会将加入的PB增加2点。11级起，改为增加4点。
- **熟练骰**：你的熟练骰尺寸为［PB+2］。效果表述中的「[N]PD」代表投掷N枚熟练骰并将出目相加。

### 属性
创建角色时，各项属性从6开始，你可以将合计40点属性值分配在初始值上，最高18。 \
每次提升人物等级时，你都可以重新分配1点属性值，最高20。 \
属性调整值的计算方式为［属性值÷2-5］。
- **属性提升**：获得属性提升的属性基本值+2。若会令基本值超过20，则改为+1。同时获得多次**属性提升**时，不能重复选择相同的属性。

#### 基本值与当前值
属性的基本值代表初始值加上所有永久存在、且没有生效条件的调整值。 \
属性的当前值代表初始值加上所有调整值。对属性的引用默认为其当前值。 \
其他记述为基本值或当前值的参数，也参照与属性值相同的规则进行区分。

### [行动检定]{#行动检定}
取代原本的属性检定。当你执行【cell:行动】并做出具有风险 (由DM决定) 的行为时，需要采取行动检定来决定结果如何。没有风险的行为可以自由地实现，无需采取检定。 \
行动检定的流程分为以下步骤。
1. PL决定愿景与熟练
2. DM决定难度与标签
3. PL投掷检定

#### 愿景与熟练
PL的愿景应该是一个当PC克服面临的障碍后，将会达成的实在事件。 \
愿景或许会是"进入庄园"或者"看看都有谁在那栋宅邸里徘徊"。在这两种情况下，障碍都是"宅邸警卫巡逻"，警卫是个有风险的障碍，对PC来说可能会很麻烦。 \
通常来说PL的愿景会很明显，但DM还是需要在必要的时候询问清楚。

PL通过演绎角色的行为决定使用哪一种**技能**或**工具**的熟练。如果PC想大打出手，那就用**武器**。如果PC想发号施令，那就用**权威**。PL需要描述相应的行为，才能使用对应的熟练进行检定。 \
这里当然有一些灰色地带，技能和工具之间有所重叠，愿景也能通过多种手段来实现。这是被特意设计成这样的，具体使用哪种方式由PL自己决定。

#### 难度与标签
DM需要根据PL的演绎来决定这次检定的DC，以及是否拥有支援或妨碍、优势或劣势。 \
根据具体的情景，DM还可以为这次行为添加一些标签，来决定它会与其他效果产生怎样的互动。

#### 投掷检定
PL根据DM设定的难度投掷1D20加上相关修正，根据检定结果引发后果。有时DM不会将DC公开，让PL难以确定行为的后果。

### 时计
时计是一种用于追踪特定任务进展的游戏控件，其进度以**刻度**表示，简称为**刻**。 \
时计由DM创建并管理，DM有权决定是否向PL公开其存在。 \
时计拥有以下3种属性。

- **初始位置**：刻度的初始位置可以是固定值，也可以通过掷骰决定。将时计重置为初始位置时，以掷骰决定的初始位置也需要重新掷骰。
- **推进方式**：根据时计的推进方式，时计的刻度将会随之改变。
- **终止位置**：在刻度推进至终止位置的瞬间，将会激活时计的效果。不具**循环**标签的时计会在效果结算后移除。

#### 时计标签

:::{admonition} 循环
:class: seealso
拥有**循环**标签的时计在效果结算后不会移除，而是将刻度重置至初始位置。
:::

:::{admonition} 无序
:class: seealso
拥有**无序**标签的时计刻度不会按照顺序推进，推进者可以自由选择下一刻的位置。 \
如果PL拥有推进无序时计的权限，DM必须对PL公开该时计。
:::

:::{admonition} 指针
:class: seealso
记作「**指针_N**」。**指针**会被放置在时计第N刻的位置。 \
在时计推进至该位置的瞬间，将会触发指针后方所述的效果。指针会在效果结算后移除。 \
记作「**循环指针_N**」的指针因此被移除后，会在时计被重置的同时，再次将自身放置在第N刻的位置。
:::

### 命刻
每当一个具有明确叙事闭环的场景结束时，DM可以宣布为相关角色推进1次命刻。例如，解决一场谜题、完成一次潜入、击败一波敌人、成功交涉获得关键信息 \
你每推进1次命刻，都可以从以下效果中选择1项。
- **回火**：如同完成短休般恢复1项特性的次数・点数，或是触发1项完成短休时生效的特性。
- **自疗**：恢复WPR4倍的HP。

### 整备
整备是一种轻度活动。任何能够打断长休的因素均会打断整备。 \
你可以将一次整备拆分为多次完成，但会失去整备进度中未满1小时的部分。 \
在2次长休之间，你最多只能进行8小时的整备。

#### 学习法术
你可以向掌握某个法术的人请教，或者从法术书或卷轴等魔法文本中学习属于你根源的新法术。 \

### 动作

#### 行动值
你可以在自己的回合，通过消耗（行动值:Action Points, AP）获得行动权。 \
你的最大AP初始为3。你会在最大AP为0的期间陷入失能状态。 \
每当你的开始回合时，如果不处于失能状态，你会恢复已消耗的AP。

#### 行动权
每个你的回合1次，可以消耗不超过最大AP的AP，根据下表获得对应的行动权。 \
你需要消耗行动权才能使用对应的动作。未消耗的行动权会在你的下个回合开始时失效。
| 消耗  | 行动权                                                      |
| :---: | :---------------------------------------------------------- |
|   1   | 1个标准动作，1个反应动作                                    |
|   2   | 1个标准动作，1个附赠动作，1个反应动作                       |
|  3+   | ［消耗-2］个标准动作，1个移动动作，1个附赠动作，1个反应动作 |

:::{admonition} 自由动作
:class: seealso
自由动作是一类特殊的动作，使用自由动作不需要消耗行动权。 \
你可以在大多数情况下使用自由动作，但无法在昏迷・死亡的期间使用。
:::

:::{admonition} 移动动作
:class: seealso
最大速度为0的期间，你无法使用全力・移动动作获得移动力或进行移动。
:::

:::{admonition} 转换行动权
:class: seealso
你可以对行动权进行转换，这是一个自由动作。 \
标准动作的行动权可以转换为移动动作，移动・反应动作的行动权可以转换为附赠动作。 \
行动权无法逆向转换，但可以被多次转换。例如，你可以转换2次标准动作的行动权，用它来执行一个附赠动作。
:::

:::{admonition} 全力动作
:class: seealso
每个你的回合1次，可以消耗1次标准动作与1次移动动作的行动权，获得1个全力动作的行动权。
:::

:::{admonition} 集中动作
:class: seealso
某些复杂效果需要数个回合才能成功执行，这种效果会在所需动作的位置记述「**集中[N]**」。N代表执行该效果所需的进度。 \
宣言集中动作时，你会进入集中状态，持续至效果成功执行，或是宣言另一个集中动作。 \
你需要为此维持专注，并在此期间消耗行动权来累积进度。失去专注将会【word:打断】该效果。 \
每回合1次，你可以通过以下两种方式之一累积进度。只要有1轮没有累积进度，效果就会被【word:打断】。
- **全力动作**：消耗1个全力动作累积2点进度。
- **标准动作**：消耗1个标准动作累积1点进度。

在成功执行之前，该效果会维持在宣言阶段，直到进度到达要求才会进入后续阶段。 \
每次宣言集中动作时，你需要重新开始累积进度。
:::

#### 动作选项

:::{admonition} 打击【标准动作】
:name: 打击
执行打击动作时，你可以指名1个角色，使用当前持用的1件武器对目标发动1次攻击。 \
这需要你进行1次攻击检定，以攻击目标的AC作为DC。
- **成功**：攻击**命中**，对目标造成武器的伤害。如果你的攻击特别成功，还有可能形成【word:重击】。
- **失败**：攻击**失手**，通常不会对目标造成伤害。你可能可以对攻击目标造成少量的【word:擦掠伤害】。
:::

:::{admonition} 冲刺【全力动作】
:name: 冲刺
执行冲刺动作时，你获得［最大速度×2］点移动力，并立即以直线移动，移动结束时失去剩余移动力。 \
如果以此移动至少2格，你可以选择本次移动经过的任意1格作为起点，执行1次【word:打击】。
:::

:::{admonition} 行进【移动动作】
:name: 行进
你获得［最大速度］点移动力，持续1回合。
:::

:::{admonition} 快步【附赠动作】
:name: 快步
如果你的最大速度不为0，也没有陷入倒地，可以进行1格即时移动，无法进入阻碍区域。
:::

:::{admonition} 起身【移动动作】
:name: 起身
你起身脱离【sign:倒地】状态，这会引发借机攻击。
:::

:::{admonition} 回避【标准动作】
:name: 回避
执行回避动作时，你将获得以下增益，持续至你的下个回合开始：你可见的角色对你发动的攻击检定具有劣势，并且你进行的反射豁免具有优势。 \
如果陷入【sign:失能】或最大速度降至0，你会提前失去这些增益。
:::

:::{admonition} 同步【自由动作】
:name: 同步
每回合1次，当你宣言一个标准・移动・附赠动作时，可以执行同步动作，将该动作进行**封装**。你同时只能拥有1个被封装的动作。 \
被封装的标准动作会在封装时消耗标准动作的行动权；封装移动・附赠动作也会消耗标准动作的行动权，但不再消耗与自身动作类型对应的行动权。 \
在任意角色的回合开始・结束时，你可以使用反应释放封装的动作。封装会一直维持，直到被释放、被新的封装覆盖、或你的下个回合开始 (你可以在此之前先将它释放)。
- **释放动作**：被封装的动作数据会固定在结束宣言阶段的那一刻，直到被释放才进入后续阶段。封装与释放动作，不会改变其原本的动作类型。 \
  你无法封装作为衍生动作宣言的动作。例如，通过全力攻击执行的打击动作。
:::

:::{admonition} 延迟【自由动作】
:name: 延迟
执行延迟动作时，你可以将自己的先攻值降至0点。这会延后你的回合，并延长那些持续至你的回合开始・结束的效果。 \
存在多个执行延迟动作的角色时，由原本先攻值**较低**的一方优先开始回合。 \
你无法在先攻值为0或更低的期间执行延迟动作。
:::

:::{admonition} 魔法【可变】
:name: 魔法
执行魔法动作时，你可以施展1道法术，或是使用需要魔法动作的特性或物品。 \
魔法动作的动作类型由对应的法术、特性、或物品决定。若没有进行说明，则视为标准动作。
:::

:::{admonition} 行动【可变】
:name: 行动
执行行动动作时，你针对特定目标取得进展，具体见【word:行动检定】。 \
行动动作的动作类型由DM根据你具体采取的行为决定。
:::

#### 动作阶段
使用动作的全过程分为以下4个阶段：宣言 \--> 执行 \--> 裁定 \--> 结算。

1.  **宣言阶段**
    - 尝试使用动作时，首先需要将动作的名称提交给DM，这一行为称作**宣言**。
2.  **执行阶段**
    - 你需要明确动作的目标，支付相应的代价，并依次将这些信息提交给DM。
    - 如果无法支付代价，动作将被【wrod:打断】。
3.  **裁定阶段**
    - 根据动作的效果，采取必要的检定。无需检定则跳过本阶段。
4.  **结算阶段**
    - 对有效的目标结算动作的效果。
    - 若不存在有效目标，动作将被【wrod:打断】。

:::{admonition} 打断
:class: seealso
:name: 打断
被打断的效果会跳过所有剩余阶段，不进行结算，也不退还已付出的代价。
:::

:::{admonition} 解除
:class: seealso
:name: 解除
被解除的效果会提前结束。**瞬间效果**无法被解除。
:::

#### 指名
进行指名时，你需要在执行阶段将每个对象的称呼提交给DM，作为效果的目标。 \
你使用的称呼不必是对象的真名，但必须足以让DM确定所指的唯一对象。

#### 时点
时点是效果在游戏时间轴上发生的位置，决定了其触发的先后顺序与合法性。 \
所有效果都必须在某个明确的时点进行宣言。你可以自行决定自己制造的效果在同一时点的生效顺序，但不能改变自己与他人制造的效果之间的先后关系。


## 战斗

### 移动
你可以消耗移动力进行移动。在移动过程中，你可以进行步行、跳跃、翻滚、攀爬、游泳、或飞行。 \
如果你没有相应的特殊速度，部分移动模式会消耗更多的移动力，而飞行则完全不可能。

#### 阻碍区域
因物理环境恶劣或魔法现象干扰而导致生物难以自由穿行的区域被称为**阻碍区域**。 \
被体型比你小不足2级的生物占据的方格，对你而言视为**阻碍区域**。 \
当你尝试进入属于阻碍区域的方格时，需要额外消耗1点移动力。

#### 步行
你在地面上移动，步行进入1格需要消耗1点移动力。步行只能进入与当前格子拥有共角的格子。 \
一次连续移动中，你每进入2次对角方格，该次进入需要额外消耗1点移动力。 \
你可以在每个动作之间、甚至每次攻击之间进行步行，但不能在使用动作的过程中步行。 \
如果你步行离开一个生物威胁范围内的方格，将会引发该生物的借机攻击。

#### 跳跃
进行跳跃需要消耗1个附赠动作与2点移动力。你的跳跃长度为［(力量值-9)÷4］格，高度为长度的一半。 \
如果跳跃轨迹包含对角移动，每2个对角方格额外计1格距离。 \
通过跳跃跨越的距离不需要消耗移动力，也不受地面的阻碍区域影响，但会如常引发借机攻击。 \
跳跃时，你可以伸展自己的手臂来触及站立时够不到的位置，但很快就会被重力捕获。 \
跳跃允许你将跳跃高度加在1次攻击或类似行为的触及距离上，这种短暂的变化不足以改变你的威胁范围。

#### 翻滚
在你的回合中，你可以在地面上消耗1点移动力进行翻滚，这会使你移动1格。你无法在阻碍区域翻滚，也无法翻滚进入阻碍区域，因被生物占据而形成的阻碍区域除外。 \
若你在敌人的威胁范围内翻滚，则需要进行1次机敏检定，分别对抗每名敌人的攻击DC，成功则本次移动不会引发对应敌人的借机攻击。 \
若你进行的翻滚没有被借机攻击命中，这次移动可以穿越敌人占据的空间，并且首次进入敌人占据的空间不会占用移动距离。 \
每进行1次翻滚，你在本回合后续进行的所有翻滚都需要额外消耗1格移动力。

#### 攀爬
你在垂直的表面上移动，或者倒挂在天花板上。攀爬的规则与步行类似，但如果你没有攀爬速度，攀爬进入1格需要额外消耗1点移动力。在属于阻碍区域的垂直表面上攀爬时，你的每次移动都需要通过一次成功的行动检定，否则将会坠落。

#### 游泳
你在液体中移动。游泳的规则与步行类似，但如果你没有游泳速度，游泳进入1格需要额外消耗1点移动力。在属于阻碍区域的液体中游泳时，你的每次移动都需要通过一次成功的行动检定，否则将会窒息。

#### 飞行
你在空气中穿梭。飞行的规则与步行类似，并且可以在空中悬停。你无法在没有飞行速度的情况下飞行。 \
飞行期间，如果陷入【sign:失能】、【sign:倒地】、飞行速度为0、或死亡，你将坠落。如果拥有【sign:悬浮】能力，则只会在死亡时坠落。 \
因倒地而坠落时，你可以使用反应进行1次行动检定。成功则从坠落的挤压强度中减去你的飞行速度。

#### 即时移动
记作「**N格即时移动**」，意义相当于「获得N点额外移动力，并立即消耗这些移动力进行移动，移动结束时失去这些移动力」。 \
即时移动时，你可以只消耗其中一部分移动力，或是完全不消耗移动力。即时移动不会引发借机攻击。

#### 强制移动
强制移动通常发生在一名生物改变另一名生物的位置时，移动路径由制造强制移动的生物决定。强制移动与移动速度无关，不会引发借机攻击，也不会消耗移动力。 \
身处固体表面相邻方块的生物可以被强制移动推向固体表面，从而造成挤压。


### 危害
危害是指与环境有关的危险。魔法环境的危害，所造成的伤害可能也是魔法性的，具体由DM决定。

#### 燃烧
燃烧中的角色会在其每个回合开始时受到1D4点能量[火焰]伤害。你可以使用标准动作，在地上打滚来熄灭身上的火焰，并因此陷入倒地状态。 \
燃烧的火焰同样也会在被浇灭、淹灭、扑灭时消失。

#### 坠落
如果你在距离地面至少2格的高度落下，将会在坠落结束时，基于坠落的距离受到挤压。 \
坠落在液体，或者体型比你小不足2级的其他生物身上时，可以使用反应进行1次行动检定。 \
检定成功时，将因坠落而遭受的挤压伤害减半；如果坠落在生物身上，减半的部分由该生物承受，这可能会致其倒地，随后你会落在距离该生物最近的未被占据的空间内。

#### 挤压
当你从半空坠落，或是被强制移动推向固体表面时，将根据承受的挤压强度受到伤害。
- **挤压伤害**：每1点挤压强度会使你受到1D6点物理伤害。如果因挤压承受至少1点损伤，你还会陷入倒地状态。
- **坠落**：挤压强度等于坠落的高度，最高为20。
- **强制移动**：当你已经与固体表面相邻，却仍被强制移动继续推向该表面时，每2格未能完成的推移距离，将会转化为1点挤压强度。

#### 窒息
一个有准备的人类最多可以憋气［1+CON］(最低1) 轮。在战斗外，人类可以憋气10倍长的时间。 \
当剩余憋气时间归零，或者在遭遇呛水及类似情况时，生物将陷入窒息，直到连续1轮自由呼吸。
- **恍惚**：生物会在窒息期间陷入【sign:恍惚】。
- **力竭**：每当生物结束回合时，将会承受1级力竭。这些力竭等级会在窒息结束时自动解除。


### 感官
感官的敏锐程度按由低到高的顺序分为以下3个级别：朦胧、模糊、精确。你不会知晓未被任何感官侦测到的事物是否存在。
- **朦胧感官**：对人类而言最常见的朦胧感官是嗅觉，范围通常为2格。朦胧感官只会让你知晓事物是否存在。
- **模糊感官**：对人类而言最常见的模糊感官是听觉，范围通常为6格。模糊感官能够让你知晓事物占据的方格位置，但不会获知具体位置。
- **精确感官**：对人类而言最常见的精确感官是视觉，范围通常为24格。精确感官侦测到的事物对你可见。

#### 深入视觉
人类的视觉并不仅仅只有24格的可视距离，但在侦测更远距离的事物时，视觉会降为模糊感官。

#### 特殊感官
- **昏暗视觉**：拥有昏暗视觉的生物可以在微光环境中视物，如同在明亮环境中一样，因此忽略微光环境造成的遮蔽。
- **黑暗视觉**：拥有黑暗视觉的生物可以在微光・黑暗环境中视物，如同在明亮环境中一样，因此忽略微光・黑暗环境造成的遮蔽。 \
  这种感官无法分辨黑暗中的颜色，只能看到具有灰度的景象。某些形式的魔法黑暗，例如【黑暗术】法术，能够阻挡普通的黑暗视觉。
- **高等黑暗视觉**：这种魔法性的感官类似黑暗视觉，但能够在黑暗中分辨颜色，也不会被魔法黑暗阻挡。
- **盲视**：盲视允许你将视觉之外的感官作为精确感官。部分盲视拥有特殊规则。
  + **基于振动**：基于振动的盲视只能侦测与你共处同一处表面 (例如地面、一面墙壁或一块天花板，也包括地下) 或液体的事物。
  + **基于心灵**：基于心灵的盲视无法侦测没有心灵的事物，例如物品和【无心智】生物，并且在侦测【无感情】生物时视为模糊感官。

#### 遮蔽
你对拥有轻度遮蔽的生物进行的察觉・攻击检定会受到1点妨碍；若为重度遮蔽，则会受到2点妨碍。 \
当你侦测某件事物时，根据所有你当前能够用于侦测该事物的感官来判断其遮蔽。
- 若该事物仅能被朦胧感官侦测，则对你具有重度遮蔽。
- 若该事物能够被模糊感官侦测，但无法被精确感官侦测，则对你具有轻度遮蔽。
- 若该事物能够被精确感官侦测，则对你不具遮蔽。

#### 位置
如果你不知道攻击目标占据的方格位置，你对其发动的攻击检定具有劣势；此目标对你发动的攻击检定具有优势。

#### 掩护
弱掩护会为AC与反射豁免提供2点状态加值，强掩护为5点。


### 攻击

#### 体术
使用**体术**取代徒手打击。体术是一种具有「轻型」词条的**肉体武器**，造成0档物理伤害。 \
如果不具体术熟练，将无法使用体术制造威胁范围，其攻击检定也不会获得PB。

#### 借机攻击
只要生物移动离开你的威胁范围内的方格，就会引发你的借机攻击。 \
借机攻击每轮可以发动1次，不再占用反应。会让你无法执行反应的效果，同样会让你无法发动借机攻击。你依然可以消耗反应获得1次借机攻击次数。 \
当你无法发动借机攻击时，也无法使用需要消耗借机攻击的效果。

当你发现威胁范围内可见的生物使用全力・标准动作执行【cell:魔法】时，这名生物会引发你的借机攻击。 \
当你发现威胁范围内可见的生物尝试在物理上操作一件物品或比划手势时，这名生物通常也会引发你的借机攻击。例如，捡起武器、打开一扇未上锁的门或容器、将一件物品放入或取出一个容器、将小瓶中的内容物倾倒出来。 \
生物可以进行1次专注检定，对抗你的攻击DC，成功则不会因上述行为引发你的借机攻击。若这名生物同时被多名生物威胁，它只需要进行1次专注检定，分别对抗每名威胁者的攻击DC。

借机攻击形成重击时，触发借机攻击的行动会被【word:打断】。
- **打断移动**：目标立即停止移动，并失去剩余的移动力。
- **打断施法**：目标施法所用的时间被浪费，但不会产生其他消耗。
- **打断操作**：目标用于操作的手无法在当前回合发起其他行为。

#### 拔出武器
拔出武器通常是一个附赠动作，也可以作为一次至少2格的直线移动的一部分。 \
尝试拔出武器时，你的每只空手都可以拔出1件武器。你可以在自己的回合将被拔出的武器重新收起，这无需动作。 \
拔出・收起肉体或白刃武器不会引发借机攻击，其他类型的武器则会。

> **拔出与收起肉体武器** \
> 拔出与收起肉体武器无需动作，并且可以在你的回合外执行，但不能在生物的行动过程中执行。 \
> 拔出肉体武器并不意味着你需要血淋淋地摘下自己的器官。它仅仅代表你亮出尖牙、弹出利爪、或者攥紧久经锻炼的拳头，锋芒毕露，蓄势待发。 \
> 同样，你也可以随时收敛杀意，缩回爪子、闭上嘴巴、或者把拳头揣进口袋。这一切的视觉表现取决于你的创意。 \
> 与人造武器不同，无论是否收起肉体武器，都不会影响你能否持用肉体武器。这意味着你可以在收起肉体武器的状态下用它们制造威胁范围。

#### 突袭
你会在遭到突袭时承受以下效果，持续至你的第一个回合结束。
- **措手不及**：你陷入迟钝状态，并且无法执行反应动作。
- **破绽**：你陷入破绽状态。

#### [重击]{#重击}
如果攻击检定的判定值比DC高5点或更多，这次攻击将形成**重击命中**，额外造成［[攻击PB÷2]D10］点重击伤害，伤害类型与攻击相同。 \
重击不会翻倍伤害骰，并移除所有扩大重击范围的效果。
- **N1与N20**：攻击检定掷出N1/N20不会再自动失手/自动命中并重击。
- **重击确认**：形成重击命中时，攻击目标必须进行1次重击豁免。豁免失败则形成**重击确认**，令重击伤害骰+100%。
- **致命重击**：如果攻击检定的判定值比DC高10点或更多，每枚重击伤害骰都会取最大值。

:::{admonition} 重击豁免
:class: seealso
当你受到重击命中时，必须进行1次对抗此次攻击DC的重击豁免。 \
重击豁免没有关键属性，将［本次攻击挑战的DC-10］作为调整值；若重击命中是一次反击，改为以［你的基本AC-10］作为调整值。
- **失败**：本次攻击将形成**重击确认**，重击伤害骰+100%。
:::

:::{admonition} 免疫重击
:class: seealso
对免疫重击的生物发动的攻击无法形成重击命中。此类生物免疫重击伤害。
:::

#### 威胁范围
持用具有熟练的近战武器期间，你能够制造相当于触及范围的威胁范围。 \
你无法威胁对你具有全掩护的空间。处于失能状态时，你无法制造威胁范围。

#### 反击
当你被可见生物选为攻击目标时，若该攻击对抗你的AC，则可以在攻击者投掷攻击检定后、DM宣布结果前，使用当前持用的1把武器对其发动1次攻击检定承受2点减值的攻击作为反击。 \
反击不会选择攻击目标，也不会对抗攻击者的AC，而是将攻击判定值作为对方攻击检定的DC。 \
即便攻击者不在武器射程内，你也可以进行反击。如果攻击者的攻击失手，则代表反击成功；反之则为反击失败。
- **反击成功**：若攻击者身处近战反击的触及范围、或远程反击的首个射程增量内，你可以令反击命中攻击者并对其造成伤害。 \
  如果使用你的攻击判定值对抗攻击者的攻击判定值，能够满足重击的条件，命中攻击者的反击也会形成重击命中。

你不能在无法发动借机攻击时进行反击，并且针对每次攻击只能进行1次反击。 \
每进行1次反击，直到你的下个回合开始为止，后续所有反击的攻击检定都会受到2点减值。

#### 伤害档位
武器的伤害档位用于决定其伤害骰的基础尺寸与枚数。伤害档位为0档或更低的武器只能造成1点伤害，不会进行伤害检定。
| 档位  |                      伤害骰                       |
| :---: | :-----------------------------------------------: |
|  1~4  |                ［1D[(档位+1)×2]］                 |
|  5~8  |              ［1D10+1D[(档位-3)×2]］              |
| 9~12  |              ［2D10+1D[(档位-7)×2]］              |
|  13+  | ［[(档位-1)÷4]D10+1D[(档位-4×⌊(档位-1)÷4⌋+1)×2]］ |

#### 擦掠伤害
当你使用具有熟练的武器发动的攻击失手时，若判定值比DC低不超过5点，依然可以对目标造成［伤害档位］点擦掠伤害，伤害类型与武器相同。 \
你在反击时无法造成擦掠伤害。若你的攻击被成功反击，你也无法造成擦掠伤害。

#### 攻击DC
攻击产生的效果有时会要求目标进行1次对抗攻击DC的豁免检定。这种DC等于［10+对应攻击的基本修正］。 \
如果你的攻击检定拥有优势，攻击DC也会获得5点加值；若有劣势，则会承受5点减值。 \
仅作用于单次攻击的优势与劣势 (例如**下次攻击检定具有优势**) 不会影响攻击DC。

#### 瞄准空间
当你使用武器发动攻击时，可以不瞄准特定的角色，而是以空间中的任意一点作为目标。此时攻击检定的DC为5。 \
你可以通过这种方式投掷喷溅武器。由于这种武器通常能够对以落点为中心的一片区域产生影响，在特殊情况可能会产生比瞄准特定角色更好的结果。攻击检定失败则意味着武器丢到预料之外的地点，通常不会产生效果，但也可能导致戏剧性的后果，具体由DM决定。 \
对于其他武器，这种攻击方式永远不会意外命中其他角色。

#### 典型豁免
如果一个效果允许你进行拥有「典型」前缀的豁免检定，代表你可以通过豁免检定改变自身遭受的效果。 \
如果你选择不进行豁免检定，则会受到如同豁免失败的效果，但不会形成特殊失败。
- **成功**：你受到一半伤害。若豁免判定值比DC高10点或更多，还会形成特殊成功，使你免受伤害。
- **失败**：你受到全额伤害。若豁免判定值比DC低10点或更多，还会形成特殊失败，使你额外受到［[效果制造者PB]D10］点相同类型的伤害。

#### 双持攻击
每回合1次，你在执行【cell:打击】时，可以在发动攻击前使用附赠动作进行双持攻击，令这次【cell:打击】能够以副手武器额外发动1次攻击。
- 进行双持攻击时，主手发动的攻击检定将承受4点减值，副手发动的攻击检定将承受6点减值。
- 如果副手为徒手・轻型武器，则副手攻击因此承受的减值降低2点。
- 进行双持攻击时，你无法将AKM加入副手攻击的伤害中，除非该调整值为负值。

#### 夹击
你与至少1名盟友满足以下所有条件的期间，你们会对目标敌人形成夹击，此时目标对抗你们发动的攻击具有**破绽**。
1. 你与盟友分别身处目标所在方格的对侧或对角。如果目标占据多个方格，任意方格满足条件即可满足此条件。
2. 目标同时处于你与盟友的威胁范围内。
3. 目标对你与盟友可见。
4. 你能够与盟友交流。


### 战策
若无特别说明，你只能选择触及范围内，对你可见且不具全身掩护的生物作为战策的目标。
- **战策攻击**：一部分战策需要你发动攻击。你可以将通过【cell:打击】或借机攻击发动的任意次武器攻击替换为战策攻击。 \
  若无特别说明，战策攻击不会造成任何直接伤害，例如基础伤害、额外伤害、擦掠伤害、横扫伤害。

#### 推撞
你需要有空手才能发动推撞，以你的徒手触及范围作为推撞范围。 \
使用空手对目标发动1次近战攻击，对抗目标的反射SC。如果目标体型大于你，其反射SC获得优势。
- **成功**：将目标往任意方向直线强制移动最多［1+(判定值-DC)÷5］格。强制移动目标时，你可以跟着目标一起移动，这不会引发目标的借机攻击。
- **失败**：若判定值比DC低6点或更多，你将失去平衡，陷入倒地状态。

#### 擒拿
你需要有空手才能发动擒拿，以你的徒手触及范围作为擒拿范围，目标体型不能比你大超过1级。 \
使用空手对目标发动1次近战攻击，对抗目标的反射SC。以你的空手攻击DC作为擒拿的挣脱DC。
- **成功**：目标陷入受擒状态，此时你可以将其**即时移动**至距离你最近的未被占据的空间。
- **失败**：若判定值比DC低6点或更多，你将陷入破绽状态，持续至你的下个回合开始。
- **紧勒**：你可以使用移动动作，以体术对受擒于你的任意名生物造成［1WD］点伤害，目标可以进行1次对抗你的徒手攻击DC的典型强韧豁免。 \
  紧勒不会进行攻击检定，也不会造成那些要求你攻击命中的附加效果。
- **一手一擒**：每个能够用于擒拿的部位在同一时间最多只能擒拿1名目标。 
- **结束擒拿**：受擒目标可以使用标准动作，进行1次行动检定来对抗挣脱DC，成功则结束受擒状态。 \
  若你陷入失能，或是目标离开擒拿范围，受擒状态也会提前结束。此外，你随时可以释放目标 (无需动作)。

#### 缴械
你尝试击落1个目标身上携带且对你可见的1件物品，目标体型不能比你大超过1级。 \
使用徒手或武器对目标发动1次近战攻击，对抗目标的反射SC。如果目标以至少2只手持握该物品，其反射SC获得优势。
- **成功**：该物品将掉落在原地。如果你有空手，则可以尝试接住该物品。
- **失败**：若判定值比DC低6点或更多，你用于发动缴械的武器将掉落在原地。

#### 虚招
你尝试用误导性动作让能看见或听见你的1个敌人对你真正的攻击毫无防备。你可以使用移动动作发动虚招。 \
进行1次行动检定，对抗目标的被动洞悉。如果以【无心智】生物作为目标，其被动洞悉获得优势。
- **成功**：目标对抗你的下一次近战攻击具有破绽，持续1轮。若判定值比DC高5点或更多，目标会在相同时间内对抗你的所有近战攻击具有破绽。
- **失败**：若判定值比DC低6点或更多，你对抗目标的所有近战攻击具有破绽，持续1轮。

#### 兵胁
你尝试挫败6格内1名能看见或听见你的敌人的士气。你可以使用移动动作发动兵胁。 \
进行1次行动检定，对抗目标的被动洞悉。如果以无法理解你的生物作为目标，其被动察觉获得优势。
- **成功**：目标陷入<轻度屈服：内在失控>状态，强度为［1+(判定值-DC)÷5］。


### 伤害

#### 生命值
你在1级时的最大HP为［体质值+职业生命骰尺寸+CON］点，升级时取满你的职业生命骰。 \
最大HP还会因为你的体型获得修正，具体见【word:体型修正】。

#### 专注
受到伤害不会引发专注检定。但一次性因伤害失去至少［专注修正×2］(最低1) 点HP，将会结束专注。 \
如果处于无法维持专注的状态，你进行的专注检定将会自动失败。

你最多可以同时维持3个专注，无法重复对相同效果维持专注。 \
每多维持1个专注，你的专注检定就会承受5点减值。

#### 浴血
你的浴血阈值等于［⌈最大HP÷2⌉］。你会在损伤超过浴血阈值期间处于浴血。

#### 伤害类型
将原本的伤害类型整合为以下伤害类型。有时伤害类型会获得一些伤害标签，这些标签本身没有机制上的作用，主要用于决定与其他效果的交互。
| 类型  |                   例子                   |
| :---: | :--------------------------------------: |
| 物理  |             钝击、挥砍、穿刺             |
| 能量  | 强酸、寒冷、火焰、闪电、音波、黯蚀、光耀 |
| 虚弱  |     毒素、流血、磨损、枯竭、精神压力     |
| 力场  |            空间撕裂、奥术力量            |
| 纯化  |      生命或死亡力量、神圣或亵渎力量      |

#### 伤害相性
生物对不同类型的伤害具有相性，共分为五类：湮灭、易伤、通常、抗性、免疫。
- **湮灭**：受到拥有湮灭的伤害时，必须进行1次DC为［伤害值÷2］(最低10) 的体质检定。 \
  失败则本次伤害值获得［最大HP］点加值；成功则只获得［PB÷2×5］点加值。
- **易伤**：你每次承受的拥有易伤的伤害值，都会获得［PB÷2×5］点加值。
- **抗性**：你每次承受的拥有抗性的伤害值，都会受到［PB÷2×5］点减值。

#### 伤害减免
拥有（伤害减免:Damage Reduction, DR）的角色能够降低自身所受的攻击伤害。 \
如果一次攻击以角色的AC作为DC，这次攻击对任何角色造成的武器伤害、擦掠伤害与横扫伤害都会降低对应角色DR强度的数值。 \
如果武器攻击的武器伤害被DR免除，这次攻击也不会使对应角色承受任何基于攻击命中・造成伤害的非自愿效果。


### 垂危
你的垂危阈值等于［最大HP-人物等级×3］。你会在损伤超过垂危阈值期间处于**垂危**，面临遭受**处决**的风险。

#### 处决
作为全力动作，你可以对1格内可见的1名垂危生物进行处决。
- 在敌人的威胁范围内进行处决会引发其借机攻击。
- 进行处决时，你需要使用武器发动1次攻击，此次攻击会自动命中并重击确认，目标无法进行重击豁免。
- 目标因处决承受至少1点损伤时，若HP未降至0，则必须进行1次对抗你的攻击DC的体质检定，失败则HP降至0。

#### 濒死
当你的HP降为0且没有立即死亡时，将会承受1级**濒死**状态，持续至你完成长休或是死亡。如果你的基本体质21+，濒死的持续时间缩短至1小时。 \
你的死线基础为3。濒死级别超过死线时，你会死亡。

濒死级别每次增加后，直到伤势稳定或濒死解除为止，你的损伤无法被治疗。

### 谱系
取代原本的生物类型，所有生物都属于以下谱系之一。
- **凡**：具备基础认知能力与自我意识的非魔法生命体。涵盖人类在内的大部分动物。
- **械**：通过人工技术构建的生命体，通常缺乏情感与欲望，依赖预设逻辑运行。
- **妖**：源于神秘本质的原生超自然存在，例如妖精、龙类以及其他魔法生物。
- **异**：源于神秘本质的异界超自然存在，例如天使、恶魔以及其他跨位面生物。

#### 原型
除了谱系之外，生物还会拥有多种原型。以下列出了一系列不具任何效果的特征原型，主要用于决定生物与其他效果的交互方式。
- **异怪**：异怪通常具有奇异的生理结构，千奇百怪的特殊能力，怪异的思维方式，或者这三者的任意组合。
- **动物**：动物代表非人的活体生物，通常为不具有魔法属性也不具有语言和文化天赋的脊椎动物。
- **构装体**：构装体是活化的物体，或人工建造的生物。
- **龙类**：龙类是有着古老起源的有鳞生物，他们通常有翅膀、具有魔法和非凡的威压。
- **精类**：精类是与妖精荒野或自然力量有关的生物，他们通常具有人类的外形。
- **人类**：人类通常只有少量超自然能力，或者完全没有。他们大多数可以用语言交谈，普遍拥有较为发达的社会结构。
- **魔兽**：魔兽是有着怪异起源的非自然生物，他们通常具有超自然能力，但有时候仅仅是在外表或习性上奇特而已。
- **泥怪**：泥怪是无具体形状或易变形的生物，通常没有心智。
- **界外体**：界外体至少有一部分本质是主物质位面以外的某个位面的产物。一些生物可能原本属于其他原型，当他们达到更高 (或更低) 层次的精神状态时变成了界外体。
- **植生体**：植生体是有意识的植物或真菌生物，他们和普通的植物与真菌不一样。
- **亡灵**：亡灵曾经是活物，但在死去以后被神术或异能力量活化，成为了不死者。

- 名称调整：人类 ==> 长身人，兽人 ==> 异人。

### 缺陷
缺陷经历是角色的内在枷锁，源于创伤、执念或致命弱点。 \
创建角色时，你可以获得最多3项缺陷。DM随时可以要求你修改缺陷。

当你或DM认为你的某项缺陷符合当前情况，将会触发你的缺陷，为你提供1点底力。 \
触发缺陷会为你带来麻烦。如果你需要为之进行检定，本次检定将自动失败；否则，将产生一些由DM决定的不利影响。例如，因"赌徒心态"在谈判中孤注一掷，反而彻底激怒了对方。

#### 改变缺陷
如果你在游戏过程中发现之前获得的缺陷不再合适，你可以在间章向DM描述角色的转变，随后修改、删除或添加你希望的缺陷。

#### 5级：踏入传奇
你获得以下特性。每项特性在使用后，你需要完成短休・长休才能再次使用。此后每提升6级，分别增加1次每项特性的使用次数。
- **传奇抗性**：当你被攻击命中，或是豁免失败时，可以消耗1点底力使攻击失手，或是豁免成功。你无法对被你反击的攻击使用传奇抗性。
- **传奇动作**：当其他生物结束回合时，你可以消耗1点底力发动下列选项之一。你不能在无法执行反应时使用传奇动作。你与其他拥有传奇动作的盟友，每轮使用传奇动作合计不能超过4次。
  + **即时行动**：你执行1个标准动作。若执行【cell:魔法】并消耗法术位，其环阶不能超过［最大环阶-2］(最低1) 环。
  + **即时移动**：你进行［⌈最大速度÷2⌉］格即时移动。

### 武器熟练度
根据你的（武器熟练度:Weapon Proficiency Rank, WPR），拥有熟练的武器攻击检定会获得［WPR÷2］点RB。 \
高阶WPR职业的WPR等于［职业等级］级，中阶WPR职业的WPR等于［⌈职业等级×$\frac{3}{4}$⌉］级，低阶WPR职业的WPR等于［⌈职业等级÷2⌉］级。 \
如果进行兼职，分别计算每个职业的WPR并加在一起，最后将小数向上取整。

#### 全力攻击
WPR5级起，作为全力动作，你可以执行1次【cell:打击】，并从以下效果中选择1项发动。
- **连击**：这次【cell:打击】能够额外发动1次攻击，但其攻击检定受到2点减值。WPR11级/17级起，能够再发动第2次/第3次额外攻击，其攻击检定受到4点/6点减值。
- **痛击**：这次【cell:打击】中你选择的1次攻击的武器伤害增加［[(WPR+1)÷6]WD］。你可以在这次【cell:打击】之前或之后进行［⌈最大速度÷2⌉］格即时移动。

你不会再获得其他来源的【（额外攻击:Extra Attack）】以及类似特性，改为在相同等级获得1项满足条件的战斗专长。

### 豁免
所有属性豁免合并为以下3种基础豁免。影响特定属性豁免的效果不会影响基础豁免。 \
每次豁免掷骰前，你需要从豁免的关联属性中选择1项作为关键属性。无论是否被选择，调整值为负的关联属性都会作为额外的调整值加在豁免检定上。
- **反射豁免**：取代力量豁免与敏捷豁免，关联属性为STR与DEX。
- **强韧豁免**：取代体质豁免与魅力豁免，关联属性为CON与CHA。
- **意志豁免**：取代智力豁免与感知豁免，关联属性为INT与WIS。

你会获得所有基础豁免的熟练，并将PB降低2点加入基础豁免，再获得基于职业豁免倾向的RB。 \
强豁免的RB等于［职业等级÷2］，弱豁免的RB等于［(职业等级+1)÷3］。 \
如果进行兼职，分别计算每个职业的豁免RB并加在一起，最后将小数舍去。 \
只要你拥有豁免倾向为强的职业，对应豁免获得的RB便会增加2点。

#### 豁免等级
豁免的（豁免等级:Saving Class, SC）等于［10+对应豁免总修正］。SC有时会代替AC作为攻击检定的DC。 \
如果你的豁免检定拥有优势，其SC也会获得5点加值；若有劣势，则会承受5点减值。 \ \
仅作用于单次豁免的优势与劣势 (例如**下次豁免检定具有优势**) 不会影响SC。
若一个效果令豁免具有劣势，并同时使对抗该豁免SC的检定获得优势，SC不会在面对该检定时承受上述减值。

##### 解散魔宠
你可以执行【cell:魔法】暂时解散你的魔宠，将其送入一个口袋位面。 \
随后，你可以执行【cell:魔法】使其重新出现于6格内你可见且未被占据的空间中。 \
当魔宠的HP降至0时，它会被永久解散。你也可以执行【cell:魔法】主动将其永久解散。 \
当魔宠通过任意方式解散时，它会将自己穿着或携带的任何物品留在自己曾占据的空间中。 \
作为一个消耗10分钟与1把金币的仪式，你可以获得一只新的魔宠，之前的魔宠将被永久解散。在2次长休之间，这个仪式最多只能举行1次。

##### 专属能力
为你的魔宠从宠物能力、魔宠能力和主人能力中选取合计2项能力。你可以在完成长休时重新选取能力。 \
你不能替换掉魔宠的天生能力。例如，你不能不给猫头鹰魔宠选择飞行能力。

#### [宠物能力]{#宠物能力}
- **两栖**：宠物同时拥有步行与游泳速度，都等于它的最高步行或游泳速度。宠物可以在空气和水中呼吸。

- **飞掠**：宠物获得6格飞行速度。宠物飞行离开敌人的威胁范围时不会引发借机攻击。

- **敏锐嗅觉**：宠物获得2格基于嗅觉的盲视，并且基于嗅觉的行动检定获得1点支援。
- **敏锐听觉**：宠物获得4格基于听觉的盲视，并且基于听觉的行动检定获得1点支援。
- **快速移动**：选择宠物的1种移动速度给予2点内在加值。
- **心灵手巧**：宠物可以用最多两条肢体像手一样发起行为。
- **健壮**：宠物的最大HP每等级增加3点。在投掷HD为其恢复HP时，可以取满HD。

- **魔兽**：魔宠的谱系变为「妖」，将【动物】原型替换为【魔兽】。
- **泥怪**：魔宠的谱系变为「异」，将【动物】原型替换为【泥怪】，并获得【无心智】原型。 \

- **反射闪避**：魔宠获得【反射闪避】专长。3级起，再获得【精通反射闪避】专长。忽略这些专长的先决条件。
- **魔法抗性**：魔宠对抗法术和其他魔法效果时进行的豁免检定具有优势。
- **抗性**：从以下伤害类型中选择2项：强酸、寒冷、火焰、闪电、雷鸣、毒素、光耀、暗蚀。魔宠获得所选伤害类型的抗性。 \
  如果魔宠拥有【天生武器】能力，它可以使用所选伤害类型之一作为天生武器造成的伤害类型。
- **隐形**：从智力、感知、魅力中选择1项作为SKM。魔宠可以对自身施展【隐形术】法术，且无需法术成分与法术位。
- **灵光乍现**：从智力、感知、魅力中选择1项作为SKM。2次长休之间最多2次，魔宠可以施展【神导术】且不会给予临时免疫。

#### 主人能力
- **吸纳魔宠**：你的魔宠可以变成你肉体上的一处印记，通常看上去像是一处胎记，纹身，或宝石，略微像是魔宠的通常模样。当处于变身状态，魔宠不能行动，除非变回魔宠。它不会被区域效果影响，必须被单独指定为目标时才能影响到它，这需要知晓它是一个生物。生物需要执行【搜索】动作成功通过1次对抗你法术DC的调查或察觉检定才能意识到它其实是一只魔宠。你的魔宠仍然可以通过心灵连接来交流其感受。从魔宠和变身形态之间变化需要你维持1分钟的专注。

- **爆裂魔宠**：从以下伤害类型中选择1项：强酸、寒冷、火焰、闪电、雷鸣。 \

- **法力储存 (先决：主人最大环阶4+)**：你获得1枚［最大环阶-3］环的法术位，完成长休时恢复。


## 装备

### 金币
**金币**代表你在冒险中积累的财富，可用于购买物品、消耗品与装备。 \
金币以三种单位计量：**把**、**袋**、**箱**。每把金币的价值约为10GP，换算关系如下所示。你不必在意单价不足10GP的非魔法物品储备。
- 10 把 = 1 袋
- 10 袋 = 1 箱

当某一单位的标记槽填满时，需要向上进位并清零当前单位。
- 例如：持有 9 把时再获 1 把 → 转为 1 袋，把数归零。
- 持有 9 袋时再获 1 袋 → 转为 1 箱，袋数归零。

若金币总量超过1箱，需要先将超额部分存入安全地点，才能继续获得金币。 \
金币的价值是抽象的，无需精确计算每一枚。若你想打赏侍者、向许愿井投币，或进行其他小额支出，只需示意即可，不必拘泥具体数额。


### 武器

#### 武器类型
常见的武器分为以下4种类型。肉体武器之外的武器又被统称为**人造武器**。
- **肉体武器**：作为角色身体一部分而存在的武器。这种武器无法直接购买。拥有天生武器的角色对自身所有肉体武器具有熟练。
- **白刃武器**：所有需要依靠角色自身力量进行近战格斗的冷兵器。部分白刃武器十分适合投掷，可以用于发动远程攻击。
- **射击武器**：利用机械结构、火药爆炸或能量释放来发射弹药的远程武器。使用射击武器通常需要消耗弹药，装填・取出弹药会引发借机攻击。
- **喷溅武器**：能够对一定区域造成范围破坏的特殊武器。喷溅武器通常是消耗品。从容器中取出喷溅武器会引发借机攻击。

#### 武器模板
取消原本的所有非魔法人造武器，你现在通过购买武器模板设计自己的武器。

|   模板   | 伤害  | 价格  |
| :------: | :---: | :---: |
| 简易武器 |  2档  |   F   |
| 军用武器 |  3档  |  1把  |

购买模板时，你需要决定它属于**白刃武器**还是**射击武器**。若为射击武器则获得「弹药_10」词条。 \
你可以为武器模板添加任意项改装。通常不能重复添加相同的改装，也无法添加会将伤害降至0档的改装。
- **投掷**：仅限白刃武器。武器的伤害档位-1，并获得「投掷_4」词条。支付1把金币，令对轻型武器添加的此改装无需降低伤害档位。
- **触及**：仅限白刃武器。武器的伤害档位-2，获得「触及」词条。
- **残暴**：仅限白刃武器。支付2袋金币，可以使用武器代替空手发动【推撞】战策，并且为此进行的攻击检定获得专精。
- **绞缠**：仅限白刃武器。支付2袋金币，可以使用武器代替空手发动【擒拿】战策，并且为此进行的攻击检定获得专精。
- **华丽**：仅限白刃武器。支付2袋金币，使用武器发动【缴械】或【虚招】战策时，你为此进行的检定获得专精。
- **灵巧**：仅限白刃武器。支付2把金币，武器获得「灵巧」词条。
- **助进**：仅限射击武器。支付2把金币，武器获得「助进」词条。
- **远射**：仅限射击武器。武器的伤害档位+1，使用该武器攻击5格内的角色时攻击检定具有1点妨碍。
- **轻型**：武器的伤害档位-1，并获得「轻型」词条。无法同时添加重型或双手改装。
- **重型**：武器的伤害档位+1，并获得「重型」词条。无法同时添加轻型改装。
- **繁琐**：武器的伤害档位+1，并获得「繁琐」词条。
- **双手**：武器的伤害档位+1，并获得「双手」词条。无法同时添加轻型或多用改装。
- **可靠**：武器的伤害档位-1，使用该武器的攻击检定+1。
- **强力**：武器的伤害档位+1，使用该武器的攻击检定-1。
- **多用**：支付2把金币，武器获得「多用_1」词条。无法同时添加双手改装。
- **增距**：支付2把金币，如果武器拥有射程增量，则射程增量+50%。可以重复添加此改装，最多4次。
- **圣化**：支付5袋金币，武器的伤害类型变为纯化，并为该武器添加【sign:圣洁】或【sign:邪秽】标签。

完成改装之后，再为武器添加1种满足条件的精通词条，武器就此设计完成。此时武器的伤害每比模板高1档，你就需要支付1把金币。 \
只有以下改装可以添加在设计完成的武器上：残暴、绞缠、华丽、灵巧、助进、增距、圣化。

#### 武器词条

- **灵巧**：你可以使用敏捷作为灵巧武器的攻击属性。
- **轻型**：轻型武器很适合用于进行双持攻击。
- **重型**：如果你的力量值低于13，无法使用重型武器；如果低于15，使用重型武器发动的攻击检定具有1点妨碍。 \
  被共持的重型武器在判断上述条件时，视你的力量值比实际高2点。
- **繁琐**：如果你的敏捷值低于13，无法使用繁琐武器；如果低于15，使用繁琐武器发动的攻击检定具有1点妨碍。 \
  被共持的繁琐武器在判断上述条件时，视你的敏捷值比实际高2点。
- **触及**：使用触及武器时的触及范围增加1格。这同样会增加你的威胁范围。
- **多用**：记作「**多用_N**」，你以共持的多用武器造成的武器伤害获得N点加值。擦掠伤害也会获得这项加值。
- **投掷**：记作「**投掷_N**」，你可以投掷此类武器作为远程攻击，射程增量为N格，并且可以在远程攻击时的伤害检定中加入STR。
- **弹药**：每次使用弹药武器发动攻击都需要消耗1发弹药。若记作「**弹药_N**」，则代表其可以发射弹药进行远程攻击，射程增量为N格。
- **装具**：装具武器能够稳定安装在你身上，因此不会占用你的手，也无法被缴械。但想要使用装具武器发动攻击，依然需要你用手持用它。
- **助进**：如果你的STR为正值，可以将［STR-2］(最低0) 加入使用助进武器发动的远程攻击的伤害检定 (若原本不行)；否则加入你的完整STR。


#### 精通词条

- **擦掠**：删除该词条。
- **横扫**：仅限重型武器。每回合1次，使用横扫武器发动的攻击命中时，你可以对攻击目标，以及距离其1格内你选择的任意个生物造成［AKM-2］点横扫伤害，伤害类型与武器相同。 \
  如果本次攻击重击确认，你可以使所有受到横扫伤害的生物往远离你的方向强制移动1格。
- **迅击**：仅限轻型武器。进行双持攻击时，如果你仅以讯击武器发动主手与副手攻击，则无需消耗附赠动作。
- **推离**：使用推离武器发动的攻击命中时，你可以使用此武器代替空手，以相同的攻击判定值对目标发动1次**推撞**。
- **削弱**：使用削弱武器发动的攻击命中时，直到你的下一回合开始前，目标进行的下次攻击检定的妨碍强度获得1点状态加值。
- **缓速**：使用缓速武器发动的攻击命中时，目标的速度受到2点状态减值，持续至你的下个回合开始。如果本次攻击重击确认，目标还会在此期间陷入禁足状态。
- **失衡**：使用失衡武器发动的攻击命中时，以攻击判定值对抗目标的反射SC，成功则其陷入倒地状态。
- **侵扰**：使用侵扰武器发动的攻击命中时，直到你的下一回合结束前，你对目标进行的下次攻击检定的支援强度获得1点状态加值。

#### 稳定装置
你可以将稳定装置作为配件安装在重型射击武器上。 \
完成部署的稳定装置能够为你免去重型词条对力量值的需求。

:::{admonition} 便携武器架
便携武器架是一种标准的稳定装置，价值可以忽略不计。 \
部署便携武器架可以作为拔出武器的一部分，收起它则是一个附赠动作。 \
你必须先收起便携武器架，才能将被安装的武器移动到另一个位置。
:::

#### 武器伤害
若无特别说明，所有武器都造成物理伤害。 \
无论攻击属性，通过武器发动的近战攻击，伤害检定只会加入你的STR。原本改变武器攻击属性的效果，不会改变伤害检定中使用的属性。 \
除非拥有「投掷」或「助进」词条，否则通过武器发动的远程攻击不会加入属性调整值。

#### 射程增量
武器的远程攻击最远能够瞄准6个射程增量内的目标。 \
你与目标的距离每超过1个射程增量，就会在远程武器攻击检定上承受2点射程减值。

#### 持用武器
你的每只手同时只能持用1件物品。拥有肉体武器的肢体没有持用任何物品的期间，可以视为持用着对应的肉体武器。
- **共持**：用至少2只手持用，且其中包含主手的武器握法被称为共持。被共持的武器攻击视为主手攻击。肉体武器无法共持。
- **副手攻击**：以副手使用近战武器发动攻击时，将加入伤害检定的属性调整值降低2点。


### 护甲
- **防具加值**：着装受训护甲的期间，如果你的力量值到达护甲列出的力量，AC与敏捷上限会获得［基本WPR÷4］点加值。
- **检定减值**：沉重的护甲会妨碍你的行动，DM可以在合适的时机将护甲的检定减值添加在你的行动检定上。如果你的力量值到达护甲列出的力量，该减值减半。
- **速度减值**：你的所有速度都会承受护甲列出的速度减值。如果你的力量值到达护甲列出的力量，该减值减半。
- **防御力与耐久**：你可以使用附赠动作，或是在被攻击命中时使用反应，消耗1点护甲的耐久，将其防御力作为状态加值添加在DR上，持续至你的下个回合开始。护甲已消耗的耐久会在你完成长休・短休时恢复。 \
  着装受训护甲的期间，该护甲的耐久获得［(基本WPR+1)÷6］点加值。

#### 护甲词条
- **嘈杂**：即便力量值到达要求，护甲对隐匿的行动检定造成的检定减值不会因此减半。
- **适应**：在决定**防具加值**与耐久时，可以使用你的人物等级作为基本WPR。
- **柔韧**：护甲对运动的行动检定造成的检定减值减半。若已被减半，改为不会在此时承受护甲的减值。
- **壁垒**：着装该护甲的期间，如果你的DEX是负值，不会加入AC与反射豁免。

如果你对盾牌受训，则AC会在持用盾牌期间获得2点状态加值。你不会在失能时获得这项加值。 \

无心智的生物没有智力属性，并免疫影响心灵的效果，例如拥有心灵标签的伤害、魅惑・恐慌・屈服状态。
:::

:::{admonition} 无感情
:class: hint
:name: 无感情
无感情的生物免疫影响情绪的效果，例如魅惑・恐慌状态。
:::

:::{admonition} 不定型
:class: hint
:name: 不定型
不定型的生物对依赖特定解剖结构的伤害拥有抗性，并免疫伤害之外的其他效果，例如武器的重击・偷袭伤害。 \
只要此类生物没有携带或穿着任何物品，就能跻身通过最窄1寸宽的空间。
:::

:::{admonition} 爪牙
:class: hint
:name: 爪牙
爪牙难以对等级相同或更高的PC构成威胁。 \
PC对爪牙发动的攻击形成重击，或是爪牙的典型豁免形成特殊失败时，爪牙将立即死亡。
:::
