# 法术

## 基础调整
- 通过职业特性获得的始终准备法术视为对应职业的**本职法术**。
- 你必须空着主手才能完成法术的姿势・材料成分。
- 你可以执行【魔法】动作施展施法时间为附赠动作的法术。
- 作为法术效果的一部分而发动的武器攻击同时视为法术攻击，并且可以使用SPR代替APR决定RB。
- 移除：寻获魔宠、法术反制。

### 施法关键属性
当你获得第2个拥有施法关键属性的职业时，从两者的施法关键属性中选择1项，作为所有相关职业的施法关键属性。 \
每当你提升人物等级时，都可以从当前所有相关职业中重新选择施法关键属性。

### 法术成分
你可以使用与法术根源对应的法器作为额外的（器材:Focus）成分，代替不会被消耗、或没有标注价格的材料成分。 \
只要你用至少1只手持握器材，就可以满足器材成分。如果这只手是主手，你还可以在满足姿势・材料成分时将其视为空手。

### 辨识施法
当你发现一名生物尝试施法时，可以使用与其根源对应的技能进行1次属性检定。这无需动作，但不能在无法执行反应时进行。 \
若你在对应技能上不具熟练，则无法进行此属性检定。 \
若该法术不属于任何根源，则由DM决定技能。
| 根源  | 技能  |
| :---: | :---: |
| 奥术  | 奥法  |
| 神术  | 宗教  |
| 异能  | 神秘  |
| 原能  | 自然  |

这一属性检定的DC基础为10，再加上［法术环阶×2］点RB。检定成功则知晓即将被施展的是什么法术。 \
若被辨识的法术不是你的已知法术，根据法术的稀有度，此次属性检定的DC将获得加值。
| 稀有度 | 加值  |
| :----: | :---: |
|  常见  |  +0   |
|  罕见  |  +2   |
|  稀有  |  +5   |
|  传说  |  +10  |

#### 辨识持续中的魔法
你可以通过10分钟的专注尝试辨识一个持续中的魔法效应，包括一件物品或一个地点，这需要进行1次与辨识施法相同的属性检定。若辨识对象没有环阶，则由DM决定其RB。 \
若你的属性检定失败，直到完成长休为止，无法再尝试辨识相同的目标。


## 新增规则

### 魔法根源
施法者可以从四种不同的法术列表中准备法术，每个列表代表一种魔法根源：奥术、神术、异能、原能。 \
当你通过职业特性获得魔法根源时，会将该根源中的所有常见法术加入对应职业的已知法术列表。 \
如果你从多个职业获得魔法根源，需要分别处理这些职业的已知法术列表。
- **奥术**：奥术根源包含所有术士与法师法术，使用奥术法器作为施法法器。
- **神术**：神术根源包含所有牧师与圣武士法术，使用圣徽作为施法法器。
- **异能**：异能根源包含所有吟游诗人与魔契师法术，使用奥术法器作为施法法器。
- **原能**：原能根源包含所有德鲁伊与游侠法术，使用德鲁伊法器作为施法法器。

### 法术强化
SPR5级、11级、17级起，你准备的所有法术分别会获得1次强化。 \
在决定以种族特性准备的法术获得的法术强化次数时，将你的人物等级作为SPR。

「**强化_N**」后记述的效果会在法术获得N次强化后激活。 \
戏法原本在第5级、11级、17级获得的强化分别替换为<**强化_1**>、<**强化_2**>、<**强化_3**>效果。

### 本能法术
你始终准备着本能法术，将本能法术视为所有为你提供魔法根源的职业的**本职法术**。 \
施展本能法术时，将你的人物等级作为SPR。若你没有施法关键属性，则将魅力作为本能法术的施法关键属性。 \
通过原本的种族・专长获得的始终准备法术皆属于本能法术。

### 降环施法
当你用比法术本身环阶更低的法术位施展法术时，所施展的法术将被视为一个更低环阶的法术。 \
若对不具「**降环施法**」条目的法术进行降环施法，法术将不会生效。

### 反制
被成功反制的效应将会无效。进行反制时，反制者需要进行1次相关的D20检定，对抗被反制效应的DC。
- **反制法术**：以法术反制其他效应时，你的反制检定调整值为［PB+SKM］，并获得［SPR÷2］点RB。 \
  你施展的法术遭到反制时，反制检定将会挑战你的法术DC。
- **反制特性**：以法术之外的效应反制其他效应时，你的反制检定调整值为［PB+CKM］，并获得［人物等级÷2］点RB。 \
  你造成的法术之外的效应遭到反制时，反制检定将会挑战你的职业DC。

你能否反制成功，取决于反制检定的结果与反制等阶的差距。 \
若以法术反制其他效应，或是反制法术，将法术环阶作为反制等阶。 \
若为其他情况，则将［人物等级÷2］作为反制等阶。
- **成功**：若［目标反制等阶<=你的反制等阶+1］，则反制成功。 \
  若判定值比DC高10点或更多，你的反制等阶将获得2点加值。
- **失败**：若［目标反制等阶<你的反制等阶］，则反制成功。 \
  若判定值比DC低10点或更多，反制将自动失败。

- **操作**：目标与环境进行简单的互动。例如，发出尖叫、松开握住的物品、打开一扇未上锁的门或容器、将一件物品放入或取出一个打开的容器、将小瓶中的内容物倾倒出来。

目标完整执行你的命令后，将结束自己的回合。 \
目标在执行命令的过程中受到任何伤害，可以立即重新进行1次意志豁免，成功则驱散此<屈服：支配>状态。

<**强化_1/2/3**>：你的命令可以包含最多2项/3项/4项行动，可以重复选择相同的行动。

##### 变巨术/缩小术
*二环 变化* \
**根源**：奥术、异能、原能 \
**施法时间**：标准动作 \
**施法距离**：6格 \
**法术成分**：V、S、F (法器) \
**持续时间**：专注，至多1小时 \
在法术持续时间内，法术使施法距离内一个你可见的生物或无主物品的大小发生改变（见下可选效应）。 \
如果目标为非自愿生物，其可以进行一次强韧豁免，成功则法术仅持续至你的下个回合开始。 \
如果目标是生物，则其着装物和携带物的尺寸也随之变化。随后任何从该生物身上脱离的物品都会立即恢复为正常大小。投掷武器和弹药在命中目标或失手后会恢复为正常大小。
- **变巨**：目标的高度、长度、宽度加倍，重量变为8倍，体型增大1级（例如从中型变为大型），其发动武器攻击时的伤害骰级别获得2点内在加值。
- **缩小**：目标的高度、长度、宽度减半，重量变为$\frac{1}{8}$，体型缩小1级（例如从中型变为小型），其发动武器攻击时的伤害骰级别受到2点内在减值。

<**强化_2**>：最大专注时间延长至8小时。 \
<**强化_3**>：你可以一直为本法术维持专注。

**升环施法**：
- **4环**：你可以使目标的体型额外增加/缩小1级，并使伤害骰级别因此承受的内在加值/减值提升至4点。

##### 解除魔法
*二环 防护* \
**根源**：奥术、神术、异能、原能 \
**施法时间**：可变 \
**施法距离**：6格 \
**法术成分**：V、S \
**持续时间**：立即 \
你干扰法术或者效果中的魔法力量。选择以下方式之一来施展解除魔法。
- **反制型解除**：反应，当你成功辨识施法距离内的生物施展的法术时可用。 \
  进行1次对抗被辨识法术的反制，成功则将目标施展的法术无效。
- **目标型解除**：以一个标准动作，选择施法距离内一道法术、一件物品或生物身上被成功辨识的一个魔法，进行1次对抗目标的反制，成功则提前结束其效应。 \
  若你不知道自己尝试解除的魔法效应的正确名称，你的反制检定将具有劣势。 \
  若目标是一件魔法物品，或持续时间为「永久」，则将其魔法效应压制10分钟。 \
  若目标是一件神器或类似物品，你的反制将自动失败。 \
  对你自己施展的法术进行的反制将会自动成功。
- **区域型解除**：5环起，以一个全力动作，选择施法距离内你可见的一点，对半径4格球状区域内的每个魔法分别进行1次反制，其他方面如同**目标型解除**。

<**强化_1/2/3**>：施法距离提升至12格/24格/视野内。

- **失败**：目标陷入<**缓慢_1**>状态，持续1分钟，在此期间无法执行反应。 \
  若为特殊失败，<**缓慢**>强度提升至2级，目标还会在此期间陷入迟钝状态。

**升环施法**：
- **6环**：你可以选择最多10名生物作为目标。


### 新增法术

#### 零环

##### 法剑术
*零环 塑能* \
**根源**：奥术、神术、异能、原能 \
**施法时间**：附赠动作 \
**施法距离**：自身 \
**法术成分**：V、S、F (一把你熟练的武器) \
**持续时间**：1分钟 \
你将1发充能注入作为器材成分的武器中，持续至法术结束。 \
法术持续时间内，当你使用该武器发动攻击时，可以消耗至少1发充能使本次攻击获得以下增益。
1. 你可以使用施法属性作为攻击属性。
2. 本次武器攻击同时视为法术攻击，并且可以使用SPR代替APR决定RB。

若本次攻击命中，可以再从以下效果中选择1项发动。X等于你为此消耗的充能。若对一名生物发动多个持续效果，新的效果会覆盖旧的效果。
- **轰雷**：额外造成［[X-1]D8］点雷鸣伤害，你可以将基础伤害的类型改为雷鸣。\
  直到你的下个回合开始为止，目标下次进行移动时，将受到［[X]D8］点雷鸣伤害。
- **翠炎**：额外造成［[X-1]D8］点火焰伤害，你可以将基础伤害的类型改为火焰。 \
  可以选择距离目标X格内你可见的其他生物作为第二目标，使用本次攻击判定值对抗第二目标的反射SC，成功则对其造成［[X-1]D8+SKM］点火焰伤害。
- **仇杀**：额外造成［[X-1]D8］点暗蚀伤害，你可以将基础伤害的类型改为暗蚀。 \
  直到你的下个回合开始为止，目标下次发起敌对行为时，将受到［[X]D8］点暗蚀伤害。
- **克敌**：额外造成［[X-1]D8］点光耀伤害，你可以将基础伤害的类型改为光耀。 \
  直到你的下个回合结束为止，你对目标发动的下次攻击获得1点支援，并在命中时额外造成［[X]D6］点光耀伤害。

当你再次施展本法术或丢掉受术武器时，注入其中的充能将提前消散。 \
若以【魔法】动作施展本法术，作为施法的一部分，你可以立即使用作为器材成分的武器发动1次攻击。

<**强化_1/2/3**>：注入的充能提升至2发/3发/4发。


#### 一环

##### 聚能成刃
*一环 塑能* \
**根源**：奥术、神术、异能、原能 \
**施法时间**：附赠动作 \
**施法距离**：自身 \
**法术成分**：V、S \
**持续时间**：专注，至多10分钟 \
你将魔力锻造为剑刃，在空手中创造1把灭除之刃。你的回合结束时，如果灭除之刃不在你手中，则会随之溃散。你可以用附赠动作重新将其唤出。 \
灭除之刃视为你拥有熟练的军用近战武器，整体形状和构造可以由你选择。 \
施展本法术时，从下列选择1项作为灭除之刃的基本数据。随后再选择钝击、挥砍、穿刺之一，作为灭除之刃的基础伤害类型。
- **Ⅰ型**：拥有「灵巧，轻型，投掷 (射程4)」词条，基础造成［⌈环阶÷2⌉D6］点伤害。
- **Ⅱ型**：拥有「灵巧，投掷 (射程2)」词条，基础造成［⌈环阶÷2⌉D8］点伤害。
- **Ⅲ型**：拥有「灵巧，触及，双手」词条，基础造成［⌈环阶÷2⌉D10］点伤害。

灭除之刃没有也无法改变伤害骰级别。你可以通过【攻击】或【魔法】动作以其发动攻击，并获得以下增益。
1. 你可以使用施法属性作为攻击属性。
2. 使用灭除之刃发动的武器攻击同时视为法术攻击，并且可以使用SPR代替APR决定RB与全力攻击效果。

<**强化_2**>：增加1枚灭除之刃的基础伤害骰。

**升环施法**：
- **2环**：施展本法术时，你可以从以下效果中选择1项激活。
  + **灼热**：可以选择火焰作为灭除之刃的伤害类型。 \
    Ⅰ型/Ⅱ型/Ⅲ型的基础伤害变为［[环阶+1]D4］/［[环阶+1]D6］/［[环阶+1]D8］。 \
    此外，灭除之刃会提供2格的明亮光照以及额外2格的微光光照。你可以使用附赠动作缩小或还原灭除之刃的光照范围。
  + **逐影**：可以选择心灵作为灭除之刃的伤害类型。 \
    以灭除之刃攻击身处微光・黑暗中的生物时，你进行的攻击检定具有优势。
  + **殷红**：可以选择暗蚀作为灭除之刃的伤害类型。 \
    每回合1次，你以灭除之刃发动的攻击命中时，可以迫使目标进行1次意志豁免，失败则在接下来的1轮内对你陷入恐慌状态。 \
    此外，当灭除之刃溃散时，你可以迫使源自其1格光环区域内的所有生物进行1次反射豁免，失败则受到其基础伤害的一半。
- **3环**：施展本法术时，你可以同时创造2把灭除之刃，并分别决定每把灭除之刃的数据。这些灭除之刃会造成低1环的伤害。
