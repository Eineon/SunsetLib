# 法术

## 基础调整
- 你必须空着主手才能完成法术的姿势・材料成分。
- 作为法术效果的一部分而发动的武器攻击同时视为法术攻击，并且可以使用SPR代替APR决定等级加值。
- 移除：寻获魔宠

### 专注
专注豁免改为单独的一种豁免检定，其豁免调整值为PB加上等于［APR÷2］与［SRP÷2］中较高一方的等级加值，并使用CON作为关键属性。 \
受到伤害不会再引发专注豁免，改为一次性承受至少［(专注SC-10)×2］(最低1) 点伤害则打断专注。

你最多可以同时维持3个专注。每多维持1个专注，你的专注豁免就会承受5点减值。


## 新增规则

### 法术强化
当SPR到达5级、11级、17级起，你准备的所有法术与已知戏法分别会获得1次强化。

<**强化_N**>后记述的效果会在法术获得N次强化后激活。 \
戏法原本在第5级、11级、17级获得的强化分别替换为<**强化_1**>、<**强化_2**>、<**强化_3**>效果。


## 法术列表

### 个别调整
使用下列法术替换同名法术。

#### 戏法

##### 神导术
*戏法 预言（神术、异能、原能）* \
**施法时间**：附赠动作 \
**施法距离**：6格 \
**法术成分**：V、S \
**持续时间**：2轮 \
你请求超自然存在的引导，选择施法距离内你可见的1名生物作为目标。 \
直到法术结束为止，目标进行1次D20检定的掷骰前，可以激活本法术，使这次检定获得1点支援。 \
本法术会在激活后提前结束。激活本法术的生物将在相当于<**谐振**>的时间内临时免疫本法术。 \
在战斗之外，你可以使用反应施展本法术。

##### 橡棍术
*戏法 变化（奥术、原能）* \
**施法时间**：附赠动作 \
**施法距离**：自身 \
**法术成分**：V、S \
**持续时间**：8小时 \
你将魔力灌入手中持握的一根木棍中，使其成为你拥有熟练的临时武器，基础造成2D4点钝击伤害。 \
你在法术持续时间内使用该武器发动的攻击获得以下增益。
1. 你可以使用施法属性作为攻击属性。
2. 本次攻击可以使用SPR代替APR决定等级加值，此时该武器攻击同时视为法术攻击。
3. 本次攻击造成伤害时，你可以选择将基础伤害类型改为力场，或是维持原本的伤害类型。

当你再次施展本法术或丢掉该武器时，灌入其中的魔力将提前消散。

<**强化_1/2/3**>：该武器的伤害骰级别获得2点/4点/6点增强加值。

##### 心灵之楔
*戏法 惑控（奥术、异能）* \
**施法时间**：标准动作 \
**施法距离**：12格 \
**法术成分**：V、S \
**持续时间**：1分钟 \
指定施法距离内一个你可见的生物，你试图暂时刺入目标的心灵，迫使其进行1次意志豁免。 \
豁免失败将受到1D6点心灵伤害，并在法术剩余持续时间内，进行的下1次基础豁免承受1点妨碍。

<**强化_1/2/3**>：本法术的伤害提升至2D6/3D6/4D6，并且能够影响下2次/3次/4次基础豁免。


#### 一环

##### 灾祸术
*一环 惑控（神术、异能）* \
**施法时间**：标准动作 \
**施法距离**：自身 \
**法术成分**：V、S \
**持续时间**：1分钟 \
你将疑虑充满3格光环范围内所有敌人的心灵，迫使每名敌人进行1次意志豁免。 \
豁免失败则身处本法术范围内的期间，发动的攻击检定承受1点妨碍。 \
在法术终止前，你可以使用移动动作将本法术的光环范围扩大1格，并迫使范围内尚未受到影响的敌人进行意志豁免。

##### 祝福术
*一环 惑控（神术、异能）* \
**施法时间**：标准动作 \
**施法距离**：自身 \
**法术成分**：V、S \
**持续时间**：1分钟 \
你祝福自己与3格光环范围内的所有盟友，使你们发动的攻击检定获得1点支援。 \
在法术终止前，你可以使用移动动作将本法术的光环范围扩大1格。

##### 魔化武器
*一环 变化（奥术、神术、异能、原能）* \
**施法时间**：附赠动作 \
**施法距离**：触碰 \
**法术成分**：V、S \
**持续时间**：1分钟 \
你触碰任意件武器，为这些武器的伤害骰级别分配［环阶］点增强加值。你再次施展该法术时，先前的效应将提前结束。

<**强化_1/2/3**>：持续时间延长至10分钟/1小时/8小时。

##### 援助术
*二环 防护（神术、异能、原能）* \
**施法时间**：标准动作 \
**施法距离**：6格 \
**法术成分**：V、S、M（一小片白布） \
**持续时间**：1分钟 \
选择施法距离内最多3名生物。每名目标在自己的回合开始时获得［5×(环阶-1)］点增强障壁，持续至其下个回合开始。

<**强化_1/2/3**>：持续时间延长至10分钟/1小时/8小时。

##### 变巨术/缩小术
*二环 变化（奥术、异能、原能）* \
**施法时间**：标准动作 \
**施法距离**：6格 \
**法术成分**：V、S、M（一撮铁粉） \
**持续时间**：专注，至多1小时 \
在法术持续时间内，法术使施法距离内一个你可见的生物或无主物品的大小发生改变（见下可选效应）。 \
如果目标为非自愿生物，其可以进行一次强韧豁免，成功则法术仅持续至你的下个回合开始。 \
如果目标是生物，则其着装物和携带物的尺寸也随之变化。随后任何从该生物身上脱离的物品都会立即恢复为正常大小。投掷武器和弹药在命中目标或失手后会恢复为正常大小。
- **变巨**：目标的高度、长度、宽度加倍，重量变为8倍，体型增大1级（例如从中型变为大型），其发动武器攻击时的伤害骰级别获得2点内在加值。
- **缩小**：目标的高度、长度、宽度减半，重量变为$\frac{1}{8}$，体型缩小1级（例如从中型变为小型），其发动武器攻击时的伤害骰级别受到2点内在减值。

<**强化_2**>：最大专注时间延长至8小时。 \
<**强化_3**>：你可以一直为本法术维持专注。

**升环施法**：
- **4环**：你可以使目标的体型额外增加/缩小1级，并使伤害骰级别因此承受的内在加值/减值提升至4点。


### 新增法术

#### 戏法

##### 法剑术
*戏法 塑能（奥术、神术、异能、原能、邪狱使）* \
**施法时间**：附赠动作 \
**施法距离**：自身 \
**法术成分**：V、S、M（一把你熟练的武器） \
**持续时间**：1分钟 \
你将1发充能注入作为法术成分的武器中，持续至法术结束。 \
法术持续时间内，当你使用该武器发动攻击时，可以消耗至少1发充能使本次攻击获得以下增益。
1. 你可以使用施法属性作为攻击属性。
2. 本次武器攻击同时视为法术攻击，并且可以使用SPR代替APR决定等级加值。

若本次攻击命中，可以再从以下效果中选择1项发动。X等于你为此消耗的充能。若对一名生物发动多个持续效果，新的效果会覆盖旧的效果。
- **轰雷**：额外造成［[X-1]D8］点雷鸣伤害，你可以将基础伤害的类型改为雷鸣。\
  直到你的下个回合开始为止，目标下次进行移动时，将受到［[X]D8］点雷鸣伤害。
- **翠炎**：额外造成［[X-1]D8］点火焰伤害，你可以将基础伤害的类型改为火焰。 \
  可以选择距离目标X格内你可见的其他生物作为第二目标，使用本次攻击判定值挑战第二目标的反射SC，成功则对其造成［[X-1]D8+施法属性调整值］点火焰伤害。
- **仇杀**：额外造成［[X-1]D8］点暗蚀伤害，你可以将基础伤害的类型改为暗蚀。 \
  直到你的下个回合开始为止，目标下次发起敌对行为时，将受到［[X]D8］点暗蚀伤害。
- **克敌**：额外造成［[X-1]D8］点光耀伤害，你可以将基础伤害的类型改为光耀。 \
  直到你的下个回合结束为止，你对目标发动的下次攻击获得1点支援，并在命中时额外造成［[X]D6］点光耀伤害。

当你再次施展本法术或丢掉受术武器时，注入其中的充能将提前消散。

<**强化_1/2/3**>：注入的充能提升至2发/3发/4发。


#### 一环

##### 聚能成刃
*一环 塑能（奥术、神术、异能、原能）* \
**施法时间**：附赠动作 \
**施法距离**：自身 \
**法术成分**：V、S \
**持续时间**：专注，至多10分钟 \
你将魔力锻造为剑刃，在空手上创造1把灭除之刃。你的回合结束时，如果灭除之刃不在你手中，则会随之溃散。你可以用附赠动作重新将其唤出。 \
灭除之刃视为你拥有熟练的军用近战武器，整体形状和构造可以由你选择。 \
施展本法术时，从以下选择1项作为灭除之刃的基本数据。随后再选择1种力场之外的伤害类型，作为灭除之刃的基础伤害类型。
- **Ⅰ型**：拥有「灵巧，轻型，投掷 (射程 6/24)」词条，基础造成［⌈环阶÷2⌉D6］点伤害。
- **Ⅱ型**：拥有「灵巧，投掷 (射程 4/12)」词条，基础造成［⌈环阶÷2⌉D8］点伤害。
- **Ⅲ型**：拥有「双手，触及」词条，基础造成［⌈环阶÷2⌉D10］点伤害。

灭除之刃没有也无法改变伤害骰级别。你可以通过【攻击】或【魔法】动作以其发动攻击，并获得以下增益。
1. 你可以使用施法属性作为攻击属性。
2. 使用灭除之刃发动的武器攻击同时视为法术攻击，并且可以使用SPR代替APR决定等级加值与全力攻击效果。

<**强化_2**>：增加1枚灭除之刃的基础伤害骰。

**升环施法**：
- **2环**：施展本法术时，你可以从以下效果中选择1项激活。
  + **灼热**：Ⅰ型/Ⅱ型/Ⅲ型的基础伤害变为［[环阶+1]D4］/［[环阶+1]D6］/［[环阶+1]D8］。 \
    此外，灭除之刃会提供2格的明亮光照以及额外2格的微光光照。你可以使用附赠动作缩小或还原灭除之刃的光照范围。
  + **逐影**：以灭除之刃攻击身处微光・黑暗中的生物时，你进行的攻击检定具有优势。
  + **殷红**：每回合1次，你以灭除之刃发动的攻击命中时，可以迫使目标进行1次意志豁免，失败则在接下来的1轮内对你陷入恐慌状态。 \
    此外，当灭除之刃溃散时，你可以迫使源自其1格光环区域内的所有生物进行1次反射豁免，失败则受到其基础伤害的一半。
- **3环**：施展本法术时，你可以同时创造2把灭除之刃，并分别决定每把灭除之刃的数据。这些灭除之刃会造成比实际低1环的伤害。
