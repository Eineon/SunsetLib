# 真血卿 Lucard

*于赤红满月映照之下，踏起靡丽荣华的舞步*

## 真血卿核心特质
- **主要属性**：魅力
- **生命值骰**：D6
- **职业熟练**：中阶APR、中阶SPR
- **豁免倾向**：反射_弱、强韧_强、意志_弱；选择一项弱豁免提升为强豁免
- **武器熟练**：简易武器、2种不具「重型」词条的军用武器
- **护甲受训**：轻甲、盾牌
- **额外熟练**：1项技能、1项工具

在浩瀚夜空的怀抱中，民众安然入眠。苍天亦恪守誓言，将恩泽赐予大地。 \
过去野蛮的战争已不复存在，民心得以被安宁的生活所满足。 \
然而在这长夜深处，盘踞着不散的灾祸。

邪魅的妖精必须被驱除。 \
复苏的亡灵必须被安息。 \
堕落的骑士必须被讨灭。 \
恶阳的碎片必须被毁弃。

骑士的敌人无穷无尽。 \
正因如此，阁下也必须奋起成为骑士。 \
护卫苍生，印证荣光，告慰逝者…… \
在真祖的注视之下，将阁下的传奇铭刻于这片大地。

## 成为真血卿

### 作为1级角色
- 获得「真血卿核心特质」中的所有能力
- 获得真血卿的1级特性

### 作为兼职角色
- 获得「真血卿核心特质」中的以下能力：生命值骰，职业熟练，豁免倾向，轻甲和盾牌的护甲受训，1项技能或工具熟练
- 获得真血卿的1级特性

### 选择子职业
阁下可以选择1种满足条件的真血卿子职。此后阁下将获得所选子职的所有能力，只要其所需等级不超过阁下的职业等级。 \
若所选子职的先决条件失效，阁下将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：诫命 Commandment
*吞噬自我的羽翼，驯服心中的野兽。* \
得益于真祖之血的洗礼，阁下获得以下效应。
- 阁下需要谨守【骑士道】(见下)，否则将面临【堕落】的威胁。
- 阁下的外表不再成长・衰老，但仍存在寿命；每过去3年，阁下只会损失1年寿命。
- 阁下可以通过精神上的满足代替每日的饮食需求，具体由DM裁定。
- 阁下即便不眠不休，也不会承受任何不利影响。阁下仍需要通过睡眠完成长休。
- 阁下的最大HP增加［职业等级×(施法属性值-9)÷4］(最低0) 点。
- 阁下对抗疾病与**环境影响 Environmental Effects**的强韧豁免获得［SKM］(最低1) 点加值。
- 阁下免疫【祈愿术】法术之外的命能・虚能・治疗・复活・唤起亡灵效应，除非该效应直接来自阁下自身的职业特性，具体由DM裁定。
- 阁下会对侦测亡灵存在的法术及类似效应显示为亡灵。

> **》骑士道 Knight's Path** \
> 阁下在蒙受洗礼时立下的三大誓言被称为「骑士道」。 \
> 当阁下做出违背骑士道的行为时，DM可以给予阁下焦渴作为惩罚。 \
> 实施未被骑士道约束的恶行会被视为「不德」，虽会遭到唾弃，但并不增加焦渴。

#### 骑士名声 Knight's Renown
从以下特性中选择1项获得。每次提升职业等级时，阁下都可以更换所选特性，或是更换在所选特性中做出的选择。
- **胜利**：获得军用武器熟练与中甲受训。随后选择2种已熟练的武器，阁下可以使用其精通词条；完成长休时，阁下可以更换所选的武器。
- **优雅**：使用已熟练武器进行攻击时，阁下可以使用职业属性作为攻击属性。通过【绽裂】发动的法术攻击伤害获得［SKM］(最低1) 点加值。
- **勇敢**：获得中甲与重甲的护甲受训。阁下可以使用反应动作重掷1次属性・豁免・命中阁下的攻击检定并使用新的结果，本特性在2次长休之间最多可以使用2次。
- **诚实**：选择合计2项技能与工具获得熟练。阁下回忆生物信息的智力检定、以及针对生物的感知检定获得［SKM］(最低1) 点加值。
- **激情**：阁下能够以附赠动作执行【协助】或【影响】动作。阁下可以会在针对非敌对生物的社交相关检定中获得专精。
- **仁爱**：阁下可以使用【影响】动作安抚2格内任意名能够听见或看见阁下的自愿生物的情绪。进行1次技能与DC由DM决定的魅力检定，成功则解除恐慌・魅惑状态之一，并给予［能级+SKM］点内在障壁。
- **丑闻**：阁下对6格内的可见生物造成伤害时，可以发动<**血术_1**>，治疗自身等于其中1名生物所受伤害一半的HP，溢出的治疗会变为矜持，这无需动作。为此发动<**血术**>不会失去HP。本特性在获得后无法更换。

### 1级：恩典 Grace
*体内流淌的真祖之血，允许阁下以意志扭曲现实，施行凡人视为奇迹之举。* \
阁下的能级将会决定许多职业特性的效果，并随着职业等级逐步提升，如「真血卿特性」表中的**能级**一列所示。

#### 行仪 Cortege
阁下掌握着名为行仪的特殊法术。施展行仪不需要消耗法术位，但任何尝试储存其魔法能量的行为均会失败 (例如【法术卷轴】和【吸收权杖】，不过【吸收权杖】以及其他类似效应可以如常消除行仪的效果)。 \
行仪的施法关键属性为魅力，没有法术成分，但会在施展过程中产生辐射12格的华丽视听效果，并预示其影响规模与方位。 \
阁下会自动习得所有满足先决条件的行仪。

> **》交响 Symphonic** \
> 每当阁下开始回合时，会获得2点交响。5级、11级、17级起，分别会再多获得2点交响。未消耗的交响会在阁下的回合开始时消失。 \
> 以一个【魔法】动作，这也被称为【交响】动作，阁下可以选择**任意**个施法时间为「交响_N」的行仪依次施展，并分别为每个行仪消耗N点 (若为X，最低为1) 交响，交响不足将导致施法失败。 \
> 当这种行仪的施法时间被改变时，施展该行仪无需消耗交响，但也无法通过【交响】动作施展。

> **》幽影幻象 Shadow Illusion** \
> 幽影幻象拥有真实物件一般的物理性质，如同【造物术 Creation】法术的创造物。

#### 心境 Aspect
心境分为2种类型：基于积极情绪的「热忱」，与基于消极情绪的「焦渴」。 \
完成短休时，阁下可以消耗任意点热忱，以消除等量的焦渴。 \
完成长休时，阁下会失去所有热忱，并消除等量的焦渴。 \
若在长休中与自愿生物进行至少1小时的亲密互动 (由DM裁定)，则会在完成长休时重新获得［能级÷2］点热忱。 \
第2级起，阁下可以在<激奏>期间开始・结束自己的回合时引导任意点热忱，用以压制等量的焦渴，持续至失去<激奏>。
- **压制心境**：被压制的心境视为不存在，但持续时间仍然如常计算，也可以因效果而消除 (无法作为代价被消耗)。
- **引导热忱**：被引导的热忱将会受到压制。失去被引导的热忱时，引导该热忱而产生的效果会立即结束。 \
  阁下可以在自己的回合开始・结束时主动解除任意点热忱受到的引导，这也会结束因此产生的效果。

> **》矜持 Pride** \
> 若阁下拥有至少1点热忱，则每休息10分钟，便会获得［能级］点矜持，持续至失去所有热忱。 \
> 阁下最多能够同时拥有［热忱值(最高5)×能级］(最低1) 点矜持，可以通过以下方式运用。
> - **整洁仪容**：若拥有至少1点矜持，阁下的身体和装备便会在需要时保持清洁和干燥，代谢产生的废物也会被无害分解。 \
    灰尘和水分会立即脱离；即使被完全弄脏，阁下的外表也会在1分钟内恢复到原始状态。本效应只会产生卫生层面的影响，对有害效应基本没有防护作用，具体由DM裁定。
> - **镇定自若**：若阁下未陷入失能，则可以在受到伤害时消耗任意点矜持，抵消自身承受的等量伤害，这不需要任何动作。

##### 堕落 Falling
若焦渴值到达5点，阁下将承受1级<濒临堕落>。焦渴值降至4点及以下后，会解除1级<濒临堕落>。 \
每次在<濒临堕落>时承受焦渴，阁下都需要进行1次**堕落豁免** (DC以增加后的焦渴值为准)。
- **濒临堕落**：本状态拥有级别。每承受1级本状态，阁下都会再承受1个**异相**。<濒临堕落>一旦到达4级，阁下将<完全堕落>为由DM控制的怪物。 \
  承受本状态时，若阁下的焦渴值低于5点，则立即提升至5点，此时不会再因焦渴值到达5点而重复承受<濒临堕落>。 \
  只要阁下的焦渴值为5点及以上，<濒临堕落>便不会降至0级。
- **堕落豁免**：DC为［5+焦渴值］，与任何属性值无关的豁免检定。任意来源的堕落豁免都拥有以下基础效果：
  + 豁免失败时，将承受1级<濒临堕落>。掷出N1会自动失败。
  + 豁免成功时，将解除1级<濒临堕落>。掷出N20会自动成功。

> **》异相 Sigil of Fallen** \
> 任何异相都是铭刻在灵魂上的屈辱烙印。阁下每承受1级<濒临堕落>，都需要再承受1个异相。若经过DM许可，阁下随时可以承受异相。
> - 异相不会在失去<濒临堕落>时消失，也不会产生游戏机制方面的效应。
> - 异相无法通过任何方式伪装。
> - 即便更换躯体，已有的异相也会以恰当的方式重现，或是变异为其他异相。
> - 完成短休时、或是作为长休的一部分，阁下可以消耗5点热忱消除1个异相。

### 1级：逸话 Exemplum
*阁下从历史、传说与个人经历中汲取教训，凝结为独特的智慧和能力。* \
阁下可以将**秘闻**分配在逸话上，若到达「秘闻」条目中要求的数值，并满足所有先决条件，则会习得该逸话。若一个逸话可以重复习得，则将其视为不同的逸话分别分配秘闻。若逸话的效果要求豁免，则使用阁下的行仪DC。 \
「真血卿特性」表中的**秘闻**一列显示了阁下每职业等级能够分配的秘闻总数。每次提升职业等级时，阁下都可以重新分配秘闻，这会使阁下失去秘闻不再到达要求的逸话。

## 能力选项

### 行仪

#### 特殊规则
- **有效环阶**：行仪的有效环阶为［⌈职业等级÷2⌉］环。阁下可以主动降低行仪的环阶，最低1环。
- **施法频率**：拥有「施法频率」条目的行仪会受到条目中记述的施展次数限制。
- **主动解除**：阁下可以使用附赠动作或移动动作主动解除任意个由自身施展且仍在持续的行仪。

##### 先决条件
阁下无法习得未满足先决条件的行仪。若已习得的行仪先决条件失效，阁下将暂时失去该行仪，直到条件被重新满足。

##### 标签
- **心能**：这种行仪不会受到【法术反制】、【解除魔法】法术及类似效应影响，但依然无法在【反魔法场】法术及类似区域内生效。
- **未见**：这种行仪不会在施展过程中产生视听效果。
- **激奏**：这种行仪只能在<激奏>期间施展。若失去<激奏>，行仪会被压制，直到重新获得<激奏>。
- **演绎**：阁下必须通过角色扮演向DM描述自身施展这种行仪时采取的行动。若DM认为该行动不可能，或与行仪不匹配，行仪将施展失败。

##### 持续时间
- **同调**：施展行仪时，阁下需要为行仪指定1个同调位建立连结，这会将原本与之建立连结的效应 (例如同调物品的魔法效应) 压制至行仪结束。
- **热忱**：记作「热忱_N」。施展行仪时，阁下需要引导N点热忱维持行仪。

##### 具现
阁下可以在自己的回合开始・结束时主动解除自身造成的任意个具现效应。
1. **从者具现**
  - 记作「从者具现：(从者级别)级(从者类型)」。
  - 从者是一种抽象的妖精，外形由阁下自行决定，可以表现为集群，也可以是单一的个体，其他方面的数据则由DM裁定。
  - 从者分为以下6种类型，并拥有级别；在战斗中，从者视为施加在阁下身上的**状态**，相同类型的从者状态不会叠加。
  - 相同类型的从者每轮只能使用1次。使用从者的效果无需动作，但无法在阁下失能时使用。
    + **尖兵**：阁下可以在造成伤害时，对其中1名角色额外造成「1D［尖兵级别×2］」点相同类型的伤害。
    + **近卫**：阁下可以在受到伤害时，将自身承受的伤害减轻「1D［近卫级别×2］」点。
    + **猎手**：阁下可以在先攻顺序轮到20时 (出现等值时视为后手)，选择自上一轮先攻顺序轮到20以来被阁下伤害过・伤害过阁下的1名角色作为目标，对其造成「1D［猎手级别×2］」点相同类型的伤害。
    + **乐师**：阁下可以在先攻顺序轮到20时 (出现等值时视为后手)，使自身获得「1D［乐师级别×2］」点矜持。
    + **名家**：阁下可以使1名可见角色进行的1次D20检定获得「1D［名家级别×2］」点加值；阁下可以在掷骰后，DM宣布结果前决定是否使用这个效果。
    + **弄臣**：阁下可以使1名可见角色进行的1次D20检定承受「1D［弄臣级别×2］」点减值；阁下可以在掷骰后，DM宣布结果前决定是否使用这个效果。
  - 只要位于相同位面，阁下就能以心灵感应的方式与从者互相交流，从者会完全服从阁下的命令。
  - 从者通常只拥有最低限度的自我意识，若阁下不下达命令，从者便不会行动。
  - 阁下一旦死亡，所有从者都会随之消失。
2. **眷属具现**
  - 记作「眷属具现：(名称)」。
  - 眷属是一种具象的妖精，数据由将其具现的效果决定。
  - 阁下只能同时维持具现1个名称相同的眷属。
  - 若无特别说明，引用其他生物数据的眷属，依然保留眷属的默认生物类型。
  - 眷属停留在意识中时，不会满足任何要求其位于阁下特定距离内的条件。
  - 只要位于相同位面，阁下就能以心灵感应的方式与眷属互相交流，眷属会以自身的判断执行阁下的命令；若不下达命令，眷属会根据自身的意志行动。
  - 阁下一旦死亡，所有眷属都会随之消失。
3. **领域具现**
  - 记作「领域具现：(范围)」。
  - 若同一区域受到多个不可叠加且环阶相同的领域影响，双方皆会在重叠的区域中被压制。
  - 只要进入领域范围内的方格，就会触发以「进入领域」作为条件的效果。

> **》消灭从者** \
> 使阁下承受伤害的其他生物可以消耗［5X］点伤害，消灭合计X级任意类型的从者。

- **血族飨宴**：每轮1次，目标受到伤害时，阁下可以选择距离其6格内可见的最多［SKM］(最低1) 名自愿生物作为第二目标，恢复「1D［2+能级］」点HP。若目标已连入阁下的心灵网络，则阁下可以从心灵网络的成员中选择第二目标。持续至魅惑结束，或是阁下主动解除 (无需动作)。


### 厄夜卿 Clan of the Tepes

**先决**：阁下必须满足以下所有条件才能选择本子职。
- 拥有异相。
- 力量或敏捷值到达13。
- 拥有下列熟练：运动、威吓。
- 纹章包含「意象伤害：穿刺」或「意象伤害：挥砍」。

#### 3级：穿刺凯歌 Merciless Crucifixion
阁下每回合可以使用1次本特性。 \
当阁下发动的攻击、或是施展无需豁免的法术造成穿刺・挥砍伤害时，可以迫使攻击目标进行1次强韧豁免。豁免失败则陷入<轻度侵蚀：流失>状态。 \
阁下也可以在施展需要豁免且造成穿刺・挥砍伤害的法术时使用本特性，使对该法术豁免失败的生物陷入<轻度侵蚀：流失>状态，此时不会重复进行豁免检定。

##### 流失
目标结束回合时，会失去［1D[2+能级]］点HP。若目标CR比阁下职业等级低2级或更多，则会失去双倍HP。 \
若在拥有本状态期间再次承受本状态，则会立即失去1次HP。 \
每轮1次，目标会在自己的回合中满足以下任意条件时立即失去1次HP。
- 发动徒手・武器攻击。
- 受到穿刺・挥砍伤害。
- 因移动而消耗的移动力到达2格。

#### 3级：鲜红独舞 Bloodstained Massacre
阁下获得以下增益。
1. 阁下提前获得【慑敌凝视】特性，并且可以在**斩杀**时立即使用而无需动作。 \
   9级起，阁下在使用【慑敌凝视】时，可以选择最多［SKM］(最低1) 名目标。若在**斩杀**时使用，则可以将范围内所有可见的敌人作为目标。
2. 第6级起，阁下可以使用附赠动作发动<**血术_1**>，来激活以下效应，持续时间如同「谐振」。
   - 被阁下赋予<侵蚀：流失>状态，且意识到自身距离阁下不超过12格的生物，将会陷入恐慌状态。
   - 阁下可以定位12格内同时被阁下赋予<侵蚀：流失>与恐慌状态的生物，如同盲视。
3. 第14级起，阁下可以在使用增益②时改为发动<**血术_2**>，来额外添加以下效应。
   - 阁下对浴血生物使用【穿刺凯歌】特性时，其进行的豁免检定自动失败。
   - 阁下使用【穿刺凯歌】特性时，可以承受1点焦渴，使赋予的状态变为<重度侵蚀>。
   - 承受<重度侵蚀：流失>的生物每次结束回合时，<重度侵蚀：流失>都会获得1点咏唱。
   - <**咏唱_3**>清空咏唱，并迫使该生物进行1次强韧豁免。豁免失败则立即死亡。豁免成功则本回合结束时因<侵蚀：流失>而失去的HP提升至［[能级]D12］点。 \
     无论豁免是否成功，直到阁下完成1次长休为止，赋予该生物的<侵蚀>不会再因此获得咏唱。

- **威慑**：第6级起可用，处于阁下赋予的有害状态影响下的生物，若在开始回合时距离阁下不超过2格，则其在当前回合只能选择在移动、标准动作、附赠动作中执行一种。14级起，本增益的范围提升至6格。
