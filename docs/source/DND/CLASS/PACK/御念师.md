# 御念师 Kineticist

## 御念师核心特质
- **主要属性**：感知
- **生命值骰**：D8
- **职业熟练**：中阶APR、中阶SPR
- **豁免倾向**：反射_弱、强韧_强、意志_强
- **武器熟练**：简易武器
- **护甲受训**：轻甲

## 成为御念师

### 作为1级角色
- 获得「御念师核心特质」中的所有能力
- 获得御念师的1级特性

### 作为兼职角色
- 获得「御念师核心特质」中的以下能力：生命值骰，职业熟练，豁免倾向
- 获得御念师的1级特性

### 1级：原能念力
御念师能够使用原力——这是一种特殊的法术，不过它会汲取御念师体内的精神力，可以随意使用。 \
任何尝试储存原力魔法能量的行为均会失败 (例如【法术卷轴】和【吸收权杖】，不过【吸收权杖】以及其他类似效应可以如常消除原力的效果)。 \
除非另有说明，使用原力均为【魔法】动作。
- **关键属性**：本职业的施法关键属性为**感知**。
- **法术成分**：除了注能原力之外，所有原力都拥有姿势成分。注能原力没有任何法术成分。
- **首要元素**：选择1种元素获得专攻，它将成为你的首要元素。你会习得【四象法门】与2道匹配该元素性质的戏法，具体由DM裁定，这些戏法对你而言视为原力。2级起，习得该元素的护身原力。
- **通用原力**：「御念师特性」表中的**原力**一列显示了你在每个等级能够习得的通用原力总数。每次提升职业等级时，你都可以更换1项该原力。
- **级别与环阶**：你最高能够习得［⌈职业等级÷2⌉］级原力。在使用原力时，其有效法术环阶为［⌈职业等级÷2⌉］环。

#### 注能原力
「御念师特性」表中的**原力**一列显示了你在每个等级能够习得的注能原力总数。每次提升职业等级时，你都可以更换1项该原力。 \
注能原力分为**性质注能**与**塑形注能**。你可以在施展念袭时激活注能来调整效果，无法同时激活多个相同类型的注能。 \
如果同时激活多个需要豁免的注能，目标需要先进行塑形注能的豁免，其次进行性质注能的豁免。若目标豁免成功并免受塑形注能的效果，那么它可以免受整个念袭的效果。 \
如果同时激活多个调整伤害的注能，优先应用性质注能所带来的调整。

#### 超载
御念师能够超负荷地运用身体，并引发比平时更强大的力量，冒险跨过安全的界限并承受力量的反噬。 \
完成长休时，你可以消除当前承受的任意点超载。
- **限制**：「御念师特性」表中的**超载**一列显示了每个等级你每轮能够主动进行的最大超载点数。
- **超载阈值**：你的超载阈值为［CON+PB÷2］(最低1)，代表能够安全进行超载的极限。每点超过阈值的超载都会使你承受<轻度侵蚀：疲劳_PB>，无法通过消除超载之外的方式解除。 \
  不具备【原能念力】特性的生物在承受超载时，其超载阈值视为0。

### 1级：念力凝聚
每回合1次，若你空着主手，则可以使用移动动作进行念力凝聚，直到你的下个回合开始为止，你可以使自身施展的1道念袭或通用原力产生的超载降低1点。 \
你也可以使用整轮动作进行念力凝聚，直到你的下个回合结束为止，你可以使自身施展的1道念袭或通用原力产生的超载降低2点，持续至你的下个回合结束。

你需要为念力凝聚维持专注。若被迫打断专注，你还会承受其原本能够降低的超载。 \
以念力凝聚降低超载的原力持续时间若到达10分钟，则会变为「谐振」。 \
念力凝聚无法将超载降至低于0，并会以你为中心在4格光环区域创造出异常醒目且肉眼可视的能量或物质。

### 1级：储念器
御念师对自己身体与元素念力进行了长时间的研究，这让你可以在体内构筑出内在储念器用以存贮额外的能量。 \
当你需要承受超载时，可以将其中的一部分或全部填充在储念器中，使你免受承受超载的不利影响。 \
在决定职业增益与原力效果时，填充在储念器中的超载依然视为由你承受。 \
「御念师特性」表中的**超载**一列显示了每个等级储念器能够填充的超载数量。 \
完成长休・短休时，你可以消除储念器中填充的任意点超载。 \
以储念器承受超载的原力持续时间若到达10分钟，则会变为「谐振」。

### 2级：超念
当你在自己的回合施展念袭时，可以通过承受超载，如同塑法专长般调整念袭。
- **强效**：你可以承受1点超载，对念袭造成1次类似【强效法术】专长的效果。
- **顽强**：7级起，你可以承受2点超载，对念袭造成1次类似【顽强法术】专长的效果。
- **瞬发**：12级起，你可以承受3点超载，对念袭造成1次类似【瞬发法术】专长的效果。
- **双发**：17级起，你可以承受4点超载，将念袭发动2次。所有调整 (例如注能和其他超念) 都会应用到两发念袭上，但你仅需要支付单次的超载。

### 3级：元素满溢
无论是否进行超载，你的身体都会充满所选择的能量，这会让你发出盈盈的火光、毛孔渗出水珠或者经受其他与元素相应的效果。 \
你施展原力时的SPR会获得等于当前超载值的加值，最高［职业等级÷3］点，原力造成的伤害也会获得这项加值。 \
你可以使用整轮动作抑制元素满溢带来的视觉效果与增益。在你下次使用原力时，视觉效果与增益都会立刻恢复。

### 5级：元素扩展
选择任意1种元素获得专攻，包括你的首要元素。13级时再选择1种，无法选择6级时选择的元素，除非其为首要元素。每次你都会习得2道匹配所选元素性质的戏法，具体由DM裁定，这些戏法对你而言视为原力。
- 若你第一次获得该元素的专攻，能够从中习得的最大原力级别将比首要元素低2级。
- 若你第二次获得该元素的专攻，可以从中习得1项通用或注能原力作为奖励原力。
- 若你第三次获得该元素的专攻，施展出自该元素的原力时的法术攻击・伤害检定、法术DC、SPR各获得1点加值。

### 6级：注能熟稔
每回合1次，你可以将本回合为激活注能而承受的超载总值降低［职业等级÷3-1］点，最低0。

### 10级：精通增幅
你为激活念袭原力的<**超载增幅**>而承受的超载降低1点，最低0。 \
16级起，你为激活护身原力的<**超载增幅**>而承受的超载降低1点，最低0。

### 11级：超聚合
使用移动・整轮动作进行的念力凝聚，能够额外降低1点超载。此外，你可以使用附赠动作进行念力凝聚，直到你的下个回合开始为止，你可以使自身施展的1道念袭或通用原力产生的超载降低1点。

### 18级：超念大师
选择1种超念选项，将其产生的超载降低1点，最低0。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

### 20级：万力全能
你获得以下增益。
1. 你可以额外进行1点超载，以不具备专攻的元素作为主导元素施展念袭。
2. 以【魔法】动作承受1点超载，你能够选择将任何已知的原力变为任意一个类别相同的其他原力 (例如将**顽石之肤**变为**水之护罩**)，持续至完成长休。这会忽略任何关于元素的要求或者制约 (但并不能无视其他先决条件和制约)。


## 元素与原力

### 元素

#### 地元素
地元素是秩序、坚韧与不可动摇的象征。它代表着山峦的脊梁、大地的脉动以及万物赖以生长的根基。

#### 水元素
水元素是变化、适应与无形渗透的化身。它既能温柔地滋养万物，也能狂暴地冲垮堤岸，其形态随容器而变，其力量随环境而生。

#### 火元素
火元素是激情、毁灭与纯粹能量的极致体现。它是锻造的烈焰，也是燎原的星火，蕴含着瞬间爆发、净化一切的狂野力量。

#### 风元素
风元素是自由、速度与无形穿透的主宰。它是拂过山巅的清风，也是撕裂天际的飓风，代表着无拘无束的移动和难以捕捉的轨迹。


### 原力

#### 关键术语
- **元素标签**：所有原力都拥有地、水、火、风中的一项或多项作为元素标签。施展念袭时，你只能为其激活主导元素相同的注能。
- **超载增幅**：记作「**超载增幅_N**」。施展原力时，你可以进行N点 (若为X，最低为1) 超载来激活后方记述的效果。

##### 主导元素
施展原力时，你需要从该原力的元素标签中选择1项作为其主导元素。如果你不具备该元素的专攻，则无法施展对应原力。 \
部分原力的伤害类型由其主导元素决定。施展这样的原力时，你需要从下表的对应伤害类型中选择1种。
| 主导元素 |     伤害类型     |
| :------: | :--------------: |
|    地    | 强酸、力场、暗蚀 |
|    水    | 强酸、寒冷、心灵 |
|    火    | 寒冷、火焰、光耀 |
|    风    | 闪电、雷鸣、毒素 |

#### 念袭原力
所有念袭都会散发塑能学派灵光，也会被侦测出属于念袭。 \
念袭分为2种：能量念袭与物理念袭。你会自动习得所有念袭。 \
你可以使用【魔法】动作选择6格内的1点作为源点施展念袭。念袭会影响源点所在方格内的所有生物与物件。 \
施展念袭时，如果源点位于你的触及范围内，将念袭视为近战攻击，反之则视为远程攻击。

##### 能量念袭
*一级 念袭 (地、水、火、风)* \
**超载**：无 \
对目标发动1次法术攻击，对抗其反射SC，命中则造成［[原力环阶]D6］点基础伤害，再加上［SKM-2］(最低0) 。伤害类型由主导元素决定。 \
<**超载增幅_2**>：基础伤害提升至［[原力环阶×2]D6］。

##### 物理念袭
*一级 念袭 (地、水、火、风)* \
**超载**：无 \
对目标发动1次法术攻击，命中则造成［[原力环阶]D8］点基础伤害，再加上SKM。选择钝击、穿刺或挥砍作为伤害类型。 \
你必须选择能够造成所选类型伤害 (具体由DM裁定) 的主导元素，如果没有可选元素，你无法施展本原力。6级起，任意主导元素都可以令本原力造成力场伤害。 \
<**超载增幅_2**>：基础伤害提升至［[原力环阶×2]D8］。


#### 护身原力
所有护身原力都会散发防护学派灵光，也会被侦测出属于护身原力。

##### 顽石之肤
*一级 护身 (地)* \
**超载**：无 \
你获得［N1］点增强障壁，并优先于其他障壁承受伤害。N1等于［职业等级÷2］。 \
完成短休时，如果你的增强障壁低于［N1］点，则变为［N1］点。 \
如果命中你的一次攻击的基础伤害被增强障壁完整承受，你不会再受到这次攻击的其他伤害与附带效应，如同这次攻击未命中。 \
6级起，<**超载增幅_X**>：X最高为［(职业等级-2)/4］。N1提升至［X÷2×职业等级］点。当你开始回合时，增强障壁会增加［(X+1)×2］点，最高提升至［N1］点。

##### 水之护罩
*一级 护身 (水)* \
**超载**：无 \
若未着装护甲，你可以使用［10+DEX+SKM］作为基础AC。 \
6级起，<**超载增幅_X**>：X最高为［(职业等级-2)/4］。当你发现自己成为攻击的目标时，若未着装护甲，则可以使用反应使所有选择你作为目标的攻击检定承受X点妨碍，持续至你的下个回合开始。

##### 炎之血肉
*一级 护身 (火)* \
**超载**：无 \
你获得［N1］点增强障壁，并优先于其他障壁承受伤害。N1等于［职业等级÷2］。 \
完成短休时，如果你的增强障壁低于［N1］点，则变为［N1］点。 \
拥有增强障壁时，在1格内以近战攻击命中你的生物会受到［⌈职业等级÷4⌉D2］点火焰伤害，擒抱・受擒于你的生物会受到双倍的火焰伤害。 \
6级起，<**超载增幅_X**>：X最高为［(职业等级-2)/4］。上述火焰伤害的伤害骰尺寸提升X级。

##### 风卷之衣
*一级 护身 (风)* \
**超载**：无 \
若未着装护甲，你可以使用［10+DEX+SKM］作为基础AC。 \
6级起，<**超载增幅_X**>：X最高为［(职业等级-2)/4］。若未着装护甲，你的步行速度获得X点增强加值，并且可以悬浮在距离表面最高1格的空中。你还会在自己的回合获得3格外加［3X］点增强加值的飞行速度，回合结束时若距离表面超过1格则会坠落。


#### 注能原力
所有注能原力都会散发变化学派灵光，也会被侦测出属于注能原力。

##### 增距
*一级 塑形注能 (地、水、火、风)* \
**超载**：1 \
念袭射程提升至24格。

##### 连击
*三级 塑形注能 (地、水、火、风)* \
**超载**：1 \
每回合1次，你在自己的回合激活本注能时，念袭一共能够发动［⌈环阶÷2⌉］次攻击，为每次攻击分别选择目标进行攻击・伤害检定，每次攻击检定都会受到2点减值。在决定念袭的基础伤害时，将其视为2环。 \
**特殊**：你可以同时激活本注能与【增距】注能，并分别进行超载。

##### 聚念成刃
*一级 塑形注能 (地、水、火、风)* \
**超载**：1 \
将这次念袭锻造为1把念刃，而非发动攻击。本原力如同【聚能成刃】法术，但不会获得<**强化**>与**升环施法**。 \
你可以一直为本原力维持专注。如果你消除为此承受的超载，专注会提前结束。 \
使用1环念袭的基础伤害作为念刃的基础伤害，伤害类型由主导元素决定。念刃造成的伤害不会获得【元素满溢】特性提供的加值。 \
6级起，<**超载增幅_X**>：X最高为［(职业等级-2)/4］。你获得以下增益。
1. 改为使用［1+X］环念袭的基础伤害作为念刃的基础伤害。
2. 你可以同时创造2把念刃，并分别决定每把念刃的数据。这些念刃会造成低1环的伤害。


#### 通用原力

##### 元素低语
*一级 通用 (地、水、火、风)* \
**超载**：0 \
你与元素碎片结成了友谊。你获得【寻获魔宠】作为奖励专长。 \
你必须为魔宠选择【界外种】作为天生能力，它会获得与主导元素相同的原型。 \
3级起，<**超载增幅_1**>：选择合计2项魔宠能力与主人能力。只要你拥有因此承受的超载，魔宠就视为具备所选能力。
