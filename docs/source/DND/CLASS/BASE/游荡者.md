# 游荡者

## 游荡者核心特质
- **主要属性**：敏捷
- **生命值骰**：D8
- **职业熟练**：中阶APR
- **豁免倾向**：反射_强、强韧_弱、意志_强
- **武器熟练**：简易武器、具有「轻型」或「灵巧」词条的军用武器
- **护甲受训**：轻甲
- **额外熟练**：2项技能、盗贼工具

## 成为游荡者

### 作为1级角色
- 获得「游荡者核心特质」里的所有能力
- 获得游荡者的1级特性

### 作为兼职角色
- 获得「游荡者核心特质」里的以下能力：生命值骰，职业熟练，豁免倾向，轻甲的护甲受训，1项技能熟练，盗贼工具熟练
- 获得游荡者的1级特性

### 选择子职业
你可以选择1种满足条件的游荡者子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：游荡者诡计
你拥有数枚尺寸为D6的诡计骰，可以用于激活诡计选项。已消耗的诡计骰会在你的回合开始时恢复。 \
「游荡者特性」表中的**诡计骰**一列显示了你每职业等级拥有的诡计骰总数。
- **激活诡计**：当你使用精准武器发动的攻击命中时，若本次攻击检定具有优势、攻击目标被夹击、或攻击目标对你处于破绽状态，则可以为本次攻击激活最多1项诡计。
- **精准武器**：精准武器包括你拥有熟练，并且能够使用敏捷作为攻击属性的武器，即使并没有使用敏捷作为攻击属性。

#### 诡计
你在一开始拥有下列诡计选项。
- **偷袭**：本次攻击额外造成等于X枚诡计骰的偷袭伤害，伤害类型与攻击相同。你可以消耗最多X枚诡计骰，使偷袭伤害增加相同枚数的诡计骰。 \
  X代表的数值如「游荡者特性」表中的**偷袭**一列所示。你可以为一次攻击同时激活本诡计与其他诡计。
- **强袭**：消耗最多［职业等级÷3］(最低1) 枚诡计骰，直到你的下个回合结束为止，使用精准武器对目标发动攻击时APR获得等量加值。你可以为一次攻击同时激活本诡计与其他诡计。

6级起，你获得下列诡计选项。
- **淬毒**：消耗1枚诡计骰，迫使目标进行1次强韧豁免，失败则令目标陷入中毒状态，持续时间如同「谐振」。
- **摔绊**：消耗1枚诡计骰，迫使目标进行1次反射豁免，失败则令体型比你大不超过1级的生物陷入倒地状态。
- **撤步**：消耗1枚诡计骰，你进行**X格即时移动**，X等于你最大速度的一半 (向上取整)。

14级起，你获得下列诡计选项。
- **恍惚**：消耗2枚诡计骰，迫使目标进行1次反射豁免，失败则令目标的下一回合只能选择下列行为中的一项：移动、执行一个标准动作或执行一个附赠动作。
- **眩目**：消耗2枚诡计骰，迫使目标进行1次强韧豁免，失败则令目标陷入目盲状态，持续至其下个回合结束。
- **击昏**：消耗6枚诡计骰，迫使目标进行1次强韧豁免，失败则选择令其陷入昏迷或麻痹状态，持续一分钟或受到任何伤害。

### 1级：武备精通
选择合计2种武器与护甲激活精通词条。11级起，再选择2种武器或护甲激活精通词条。完成长休时，你可以重新选择合计2种武器与护甲。 \
若从多个职业获得武备精通，将［⌈游荡者等级÷2⌉］加上其他职业等级决定本特性的效果。

### 1级：专精
选择2项已熟练的技能获得专精。

### 2级：战斗风格
你获得1项满足条件的战斗专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

### 2级：灵巧动作
你精妙的思维与身手使你的行动无比迅速。你能够以附赠动作执行【疾走】、【撤离】或【躲藏】动作之一。

### 2级：反射闪避
你获得【反射闪避】作为奖励专长。7级起，你还会获得【精通反射闪避】作为奖励专长，忽略其先决条件。 \
若从多个职业获得本特性，将游荡者等级加上其他职业等级决定获得【精通反射闪避】的等级。

### 3级：诡术战策
当你通过精准武器发动**虚招**或**胁迫**战策时，可以为此消耗1枚诡计骰，使战策目标不会因【无心智】或无法理解你而在被动察觉上获得优势，并将这枚诡计骰作为内在加值加在你为此进行的属性检定上。

### 4级：技能提升
选择1项技能获得熟练，若已熟练则获得专精。此后每提升4级，再选择1项技能。

### 5级：直觉闪避
当一个你能看见的攻击者用一次攻击检定命中你时，你可以用你的反应将这次攻击对你造成的伤害减半。

### 7级：可靠才能
每当你进行属性检定，并且可以运用你的某项技能熟练或工具熟练时，你可以将D20投出的9及以下的结果视为10。 \
若拥有多个包含本特性的职业，将游荡者等级加上其他职业等级决定获得本特性的等级。

### 10级：奖励专长
你获得1项满足条件的专长。

### 11级：进阶游荡者诡计
你可以为一次攻击同时激活2项诡计。

### 15级：身心俱悉
游荡者强韧豁免的倾向变为强。

### 18级：飘忽不定
你飘忽不定的身形让敌人攻击时无从下手。只要你并未失能，以你为目标的攻击检定无法具有优势。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

### 20级：幸运一击
你留了一手非凡技巧，让你可以随时把握自己的成功之际。当你在一次D20检定中失败时，你可以将结果改为20。 \
此特性一经使用，直至完成短休或长休你都无法再次使用。


## 子职业

### 诡术师

#### 3级：施法
*调整同名特性* \
你改为从奥术法术表中准备法术，并将智力加入游荡者的主要属性。 \
6级起，你获得以下诡计选项。
- **策法**：你可以消耗3枚诡计骰，执行1个【魔法】动作。若为此消耗法术位，其环阶不能超过［最大环阶-2］(最低1) 环。 \
  每额外消耗3枚诡计骰，你就可以再以此执行1个【魔法】动作，或令以此施展的1道法术获得1次强化。


### 刺客

### 3级：暗杀
*替换同名特性的**惊扰突袭***
- **无声行动**：在投掷先攻后，战斗第一回合开始前，你可以使用反应拔出武器并移动最多等于你速度一半 (向上取整) 的距离。作为该反应的一部分，你可以在移动结束后执行【躲藏】动作。
- **诡秘突袭**：当你使用精准武器对未意识到你存在，或视你为盟友的生物发动攻击时，将你的职业等级添加在此次攻击检定中，并且对其使用的诡计骰尺寸提升至D10。
- **夺命攻击**：你可以在5级使用**击昏**诡计选项，并且只消耗3枚诡计骰。为了以这种方式使用该诡计，你必须在发动攻击前观察目标至少3轮。 \
  观察目标时，你可以进行其他行动，但注意力必须保持在对方身上，而且不被对方发现敌意。

### 9级：专业渗透
*调整同名特性* \
**机动瞄准**替换为获得【精通战策】作为奖励专长。

### 13级：淬毒武器
*替换同名特性* \
当你使用**淬毒**诡计选项使目标陷入中毒时，额外造成2枚诡计骰的毒素伤害，目标在中毒期间结束回合还会再次受到该伤害。该毒素无视抗性。

### 17级：致命袭杀
*替换同名特性* \
当你使用精准武器对未意识到你存在，或视你为盟友的生物发动攻击时，只要命中便形成重击。
