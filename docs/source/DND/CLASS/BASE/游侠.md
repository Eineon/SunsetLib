# 游侠

## 游侠核心特质
- **主要属性**：敏捷与感知
- **生命值骰**：D10
- **职业熟练**：中阶WPR、中阶SPR
- **豁免倾向**：反射_强、强韧_强、意志_弱
- **武器熟练**：简易武器、军用武器
- **护甲受训**：轻甲、中甲、盾牌
- **额外熟练**：1项技能

## 成为游侠

### 作为1级角色
- 获得「游侠核心特质」里的所有能力
- 获得游侠的1级特性

### 作为兼职角色
- 获得「游侠核心特质」里的以下能力：生命值骰，职业熟练，豁免倾向，军用武器熟练，轻甲、中甲和盾牌的护甲受训，1项技能熟练
- 获得游侠的1级特性

### 选择子职业
你可以选择1种满足条件的游侠子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：施法
你学会运用自然世界的魔法本源进行施法。
- **魔法根源**：本职业的魔法根源为**原能**。
- **关键属性**：本职业的施法关键属性为**感知**。
- **准备法术**：你可以从已知法术中准备不超过最大环阶的法术，它们会成为游侠的本职法术。 \
  「游侠特性」表中的**术法**一列显示了你每职业等级能够准备的1环及以上法术总数，这也被称为术法。
- **调整法术**：完成长休时，你可以重新准备1道本职术法。

### 1级：原始意识
当你到达特定职业等级起，就始终准备着下表中对应的法术。 \
提升职业等级时，你可以将其中1道法术替换为已知法术中的另一道仪式法术。

### 1级：宿敌
以一个附赠动作消耗1点技力，你可以标记1个生物，持续至你完成长休。拥有此标记的生物称为**宿敌**。
- 你对宿敌发动的武器攻击，WPR获得［职业等级÷4］点加值。
- 你命中宿敌的武器攻击，会额外造成1D4点相同类型的伤害。5级/11级/17级起，该伤害提升至2D4/2D6/2D8。
- 你寻找、躲避宿敌以及回忆其信息的行动检定具有优势。
- 你同时能够维持最多［1+职业等级÷4］个标记。

### 1级：武备精通
选择合计2种武器激活精通词条。此后每提升5级，再选择2种武器激活精通词条。完成长休时，你可以重新选择合计2种武器。 \
若从多个职业获得武备精通，将游侠等级加上其他职业等级决定本特性的效果。

### 2级：战斗风格
你获得1项满足条件的战斗专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

### 2级：屠灭众敌
使用【宿敌】特性时，你可以为它维持专注，如同专注于本职法术，持续1分钟。专注结束时，标记也会提解除。 \
在此期间，宿敌的HP降至0点时，你可以使用附赠动作将1个标记转移给另一个你可见的生物。如果此时不是你的回合，可以预支你的下个回合的附赠动作。

4级起，使用【宿敌】特性时，你可以同时标记最多2个生物。此后每提升4级，分别可以再多标记1个生物。

### 2级：熟练探险家
得益于你的旅途，你获得以下增益。
- **专精**：选择1项已熟练的技能获得专精。
- **语言**：你习得语言表中的2门语言。每次提升职业等级时，你都可以再从语言表中习得1门语言。

### 2级：专精地形
每当你能够获得专精时，可以改为从下表中选择1种地形获得专精。 \
当你回忆专精地形的信息时，进行的行动检定会获得熟练。若已有熟练则获得专精。 \
你还会在身处专精地形期间获得以下增益。
1. 你进行的先攻检定具有优势。
2. 你获得以下技能的熟练：隐匿、察觉、生存。对于已有熟练的技能，改为获得专精。
3. 你对抗自然环境影响的豁免检定具有优势，并且能够无视其带来的劣势。
4. 你的队伍旅行时不会因困难地形而变慢。
5. 除非受魔法效应影响，否则你的队伍不会迷路。

### 3级：致命猎杀
每轮1次，你可以在执行【攻击】动作时额外发动1次武器攻击，但只能以宿敌作为攻击目标。 \
本特性只能使用1次，在你完成长休・短休时恢复使用次数。你也可以消耗3点MP立即恢复使用次数 (无需动作)。 \
8级起，你可以重掷1次通过本特性发动的攻击检定，并选用较高的结果。 \
20级起，通过本特性发动的攻击命中时，你可以迫使目标进行1次对抗职业DC的强韧豁免，失败则选择令其立即死亡或陷入1小时的昏迷。无论豁免结果如何，直到你完成1次长休为止，无法再迫使相同生物进行该强韧豁免。

### 6级：越野
只要未着装重甲，你的步行速度获得2点增强加值。此外，你获得等于步行速度的攀爬・游泳速度。 \
10级起，你可以在空气和水中呼吸，并且免疫燃烧与窒息危害。

### 6级：不知疲倦
原初的力量现在会帮你重整旗鼓，重新踏上旅途。你因它获得以下增益。
- **增强韧性**：每回合1次，你可以消耗1点技力，在自己的回合获得［2PD+SKM］(最低1) 点增强韧性。
- **减少力竭**：10级起，完成短休时，你的力竭等级减少1级 (若有)。

### 9级：专精
选择2项已熟练的技能获得专精。

### 13级：本能专注
当你对本职法术的专注即将被打断时，可以使该法术无需专注地延长至你的下个回合结束。 \
你可以使用附赠动作再次将该法术延长至你的下个回合结束，最多延长10分钟。同时只能通过本特性延长1道法术的持续时间。

### 14级：自然面纱
你祈唤自然精魂，魔法性地将身形遮蔽隐蔽。 \
以一个附赠动作消耗1点技力，你可以让自己进入隐形状态，持续到你的下一回合结束。

### 18级：野性感官
你与自然的链接给予了你【盲斗】专长，并且获得的盲视范围提升至3倍。 \
若你已有此专长，则可以立即将其重训为另一个满足条件的专长。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

## 变体

### 6级：战斗流派
*替换【越野】或【不知疲倦】特性* \
你获得1项满足条件的战斗专长，10级起再获得1项。 \
此外，你可以在到达能够获得【自然面纱】或【野性感官】特性的等级时，将对应特性替换为获得1项满足条件的战斗专长。
