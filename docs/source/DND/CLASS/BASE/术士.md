# 术士

## 术士核心特质
- **主要属性**：魅力
- **生命值骰**：D6
- **职业熟练**：低阶APR、高阶SPR
- **豁免倾向**：反射_弱、强韧_强、意志_强
- **武器熟练**：简易武器
- **护甲受训**：无

## 成为术士

### 作为1级角色
- 获得「术士核心特质」里的所有能力
- 获得术士的1级特性（见术士特性表）

### 作为兼职角色
- 获得「术士核心特质」里的以下能力：生命值骰，职业熟练，豁免倾向
- 获得术士的1级特性（见术士特性表）

### 选择子职业
你可以选择1种满足条件的术士子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：施法
你从你的天生魔法汲取魔力用于施展法术。
- **魔法根源**：本职业的魔法根源由你选择的术法传承决定。
- **关键属性**：本职业的施法关键属性为**魅力**。
- **特殊法器**：若你空着主手，则可以使用自己的肉体作为本职法术的施法法器。
- **准备法术**：你可以从已知法术中准备不超过最大环阶的法术，它们会成为术士的本职法术。 \
  「术士特性」表中的**戏法**一列显示了你每职业等级能够准备的0术法术总数，这也被称为戏法。施展戏法不需要消耗法术位。 \
  「术士特性」表中的**术法**一列显示了你每职业等级能够准备的1环及以上法术总数，这也被称为术法。
- **调整法术**：每次提升职业等级时，你都可以调整以此准备的法术：重新准备1道戏法，再重新准备1道术法。

### 1级：先天术法
你过去经历的某件事在你身上留下了不可磨灭的印记，为你注入了难以控制的涌动魔力。以一个附赠动作，你可以将魔力释放而出，持续1分钟。在此期间你获得以下增益：
1. 对抗你的本职法术的基础豁免受到1点妨碍。
2. 你施展的本职法术的攻击检定获得1点支援。

### 2级：魔力泉眼
你的一部分最大MP将被转换为泉涌，如「术士特性」表中的**泉涌**一列所示。 \
你可以通过以下方式使用泉涌。已消耗的泉涌会在完成长休时恢复。
- **创造法术位**：你可以消耗泉涌代替MP创造法术位。
- **施展传承法术**：你可以消耗X点泉涌，以X环施展1道传承法术，并且无需消耗法术位。X最高为［职业等级÷2］。
- **迅捷塑法**：每回合1次，你可以消耗1点泉涌代替移动动作激活塑法专长。

当你为一场重要的战斗骰先攻或是处于由DM和你先前决定的高压环境时，你的魔力将开始涌动，投掷1D4并触发对应后果。
|  1D4  | 后果                                                                                                      |
| :---: | :-------------------------------------------------------------------------------------------------------- |
|   1   | 获得等于上限一半的临时泉涌，持续3轮，并触发狂野魔法浪涌。                                                 |
|  2~3  | 获得等于上限一半的临时泉涌，持续3轮。如果没有在过期前耗尽临时泉涌，你将在这段时间结束时触发狂野魔法浪涌。 |
|   4   | 获得等于上限的临时泉涌，持续1分钟。如果没有在过期前耗尽临时泉涌，你将在这段时间结束时触发狂野魔法浪涌。   |

使用这些临时泉涌制造的效应 (包括使用以其创造的法术位施展的法术) 持续时间若不为立即，则会在到达过期时间时提前结束。

#### 狂野魔法浪涌
触发狂野魔法浪涌时，你需要在狂野魔法浪涌表上掷骰以确定随机魔法效应。 \
若掷出的魔法效应是一道法术，该法术因其过于狂野而无法受塑法专长影响。如果它正常情况下需要专注，则此时它无需专注。该法术生效至其持续时间结束。

### 2级：超魔法
你获得1项满足条件的塑法专长。此后每提升4级，分别再获得1项满足条件的塑法专长。你也可以选择【额外聚能】代替塑法专长。

### 4级：源流术法
你可以将1道传承法术替换为相同环阶且经过DM允许的任意法术。此后每提升4级，你都可以再替换1道传承法术。

### 7级：术法化身
你可以使用附赠动作消耗2点泉涌激活【先天术法】特性，并且无需消耗使用次数。 \
此外，在【先天术法】特性激活期间，你每回合可以额外使用1次**迅捷塑法**。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

### 20级：术法复苏
【先天术法】特性激活期间每回合1次，你可以在激活塑法专长时不消耗聚能点。
