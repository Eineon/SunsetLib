# 术士

## 术士核心特质
- **主要属性**：魅力
- **生命值骰**：D6
- **职业熟练**：低阶APR、高阶SPR
- **豁免倾向**：反射_弱、强韧_强、意志_强
- **武器熟练**：简易武器
- **护甲受训**：无

## 成为术士

### 作为1级角色
- 获得「术士核心特质」里的所有能力
- 获得术士的1级特性

### 作为兼职角色
- 获得「术士核心特质」里的以下能力：生命值骰，职业熟练，豁免倾向
- 获得术士的1级特性

### 选择子职业
你可以选择1种满足条件的术士子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：施法
你从你的天生魔法汲取魔力用于施展法术。
- **魔法根源**：本职业的魔法根源由你选择的魔力传承决定。
- **关键属性**：本职业的施法关键属性为**魅力**。
- **特殊法器**：若你空着主手，则可以使用自己的肉体作为本职法术的施法法器。
- **准备法术**：你可以从已知法术中准备不超过最大环阶的法术，它们会成为术士的本职法术。 \
  「术士特性」表中的**戏法**一列显示了你每职业等级能够准备的0环法术总数，这也被称为戏法。施展戏法不需要消耗法术位。 \
  「术士特性」表中的**术法**一列显示了你每职业等级能够准备的1环及以上法术总数，这也被称为术法。
- **调整法术**：每次提升职业等级时，你都可以调整以此准备的法术：重新准备1道戏法，再重新准备1道术法。

### 1级：先天魔力
你过去经历的某件事在你身上留下了不可磨灭的印记，为你注入了难以控制的涌动魔力。 \
以一个附赠动作，你可以将魔力释放而出，持续1分钟。在此期间你获得以下增益：
1. 对抗你的本职法术的基础豁免受到1点妨碍。
2. 你施展的本职法术的攻击检定获得1点支援。

### 2级：魔力泉眼
你汲取体内的魔力之源，从中涌现的魔力可以修改你的法术来适应自己的需要。 \
你可以消耗X点技力，以X环施展1道传承法术，并且无需消耗法术位。X最高为［职业等级÷2］。

当你为一场重要的战斗骰先攻或是处于由DM和你先前决定的高压环境时，你的魔力将开始涌动，投掷1D4并触发对应后果。
|  1D4  | 后果                                                                                                                  |
| :---: | :-------------------------------------------------------------------------------------------------------------------- |
|   1   | 你获得［最大环阶-1D4+1］(最低1) 点泉涌技力，持续3轮，并触发狂野魔法浪涌。                                               |
|  2~3  | 你获得［最大环阶-1D4+1］(最低1) 点泉涌技力，持续3轮。如果没有在过期前耗尽泉涌技力，你将在这段时间结束时触发狂野魔法浪涌。 |
|   4   | 你获得［最大环阶］点泉涌技力，持续1分钟。如果没有在过期前耗尽泉涌技力，你将在这段时间结束时触发狂野魔法浪涌。             |

泉涌技力只能通过术士职业特性使用。消耗泉涌技力制造的效应，若持续时间不为立即，则会在到达过期时间时提前结束。

#### 狂野魔法浪涌
触发狂野魔法浪涌时，会由DM在狂野魔法浪涌表上掷骰以确定随机魔法效应。 \
若掷出的魔法效应是一道法术，该法术因其过于狂野而无法受塑法专长影响。若法术拥有效应区域，以你占据的空间作为源点。

### 2级：超魔法
你获得1项满足条件的塑法专长。此后每提升4级，分别再获得1项满足条件的塑法专长。你也可以选择【精通塑法】或【额外聚能】代替塑法专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。
- **迅捷超魔**：每回合1次，你可以消耗1点技力代替移动动作激活以此获得的专长。

### 4级：源流术法
从任意法术列表选择1道不超过最大环阶且经过DM允许的术法作为你的源流术法。 \
你始终准备着源流术法，并且可以将其视为你的传承法术。 \
此后每提升4级，你分别可以再选择1道新的源流术法。

### 7级：魔力复苏
你可以使用附赠动作消耗2点技力激活【先天魔力】特性，并且无需消耗使用次数。 \
此外，在【先天魔力】特性激活期间，你每回合可以额外使用1次**迅捷超魔**。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

### 20级：魔力化身
【先天魔力】特性激活期间每回合1次，你可以在激活塑法专长时不消耗聚能点。
