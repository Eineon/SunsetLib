# 圣武士

## 圣武士核心特质
- **主要属性**：力量或魅力
- **生命值骰**：D10
- **职业熟练**：中阶APR、中阶SPR
- **豁免倾向**：反射_弱、强韧_强、意志_强
- **武器熟练**：简易武器、军用武器
- **护甲受训**：轻甲、中甲、重甲、盾牌

## 成为圣武士

### 作为1级角色
- 获得「圣武士核心特质」里的所有能力
- 获得圣武士的1级特性

### 作为兼职角色
- 获得「圣武士核心特质」里的以下能力：生命值骰，职业熟练，豁免倾向，军用武器熟练，轻甲、中甲和盾牌的护甲受训
- 获得圣武士的1级特性

### 选择子职业
你可以选择1种满足条件的圣武士子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：施法
你已经学会了如何通过祈祷与冥想来施展法术。
- **魔法根源**：本职业的魔法根源为**神术**。
- **关键属性**：本职业的施法关键属性为**魅力**。
- **准备法术**：你可以从已知法术中准备不超过最大环阶的法术，它们会成为圣武士的本职法术。 \
  「圣武士特性」表中的**术法**一列显示了你每职业等级能够准备的1环及以上法术总数，这也被称为术法。
- **调整法术**：每次完成长休时，你都可以重新准备1道以此准备的术法。

### 1级：奇迹
神祇的祝福赐予了你不可思议的奇迹法术。 \
若从多个职业获得奇迹，将［⌈圣武士等级÷2⌉］加上其他职业等级决定你能够准备的最大奇迹级别与奇迹强度。
- **准备奇迹**：你可以将满足条件的奇迹视为牧师的已知法术进行准备。 \
  8级与16级起，你分别可以选择1道满足条件的奇迹始终准备。每次提升职业等级时，你可以重新选择1道奇迹。
- **奇迹成分**：所有奇迹都拥有「F (圣徽)」成分，无法免除。

#### 神力源泉
经由神祇的祝福，你获得了愿力能量池，如同牧师的同名特性。将［圣武士等级÷2］作为牧师等级决定本特性的效果。 \
若从多个职业获得神力源泉，将［⌈圣武士等级÷2⌉］加上其他职业等级决定本特性的效果。你必须为所有神力源泉选择同一种性质。

### 1级：武备精通
选择合计2种武器与护甲激活精通词条。此后每提升5级，再选择2种武器或护甲激活精通词条。完成长休时，你可以重新选择合计2种武器与护甲。 \
若从多个职业获得武备精通，将圣武士等级加上其他职业等级决定本特性的效果。

### 2级：战斗风格
你获得1项满足条件的战斗专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

### 2级：崇圣灵光
只要你未失能・死亡，就会散发覆盖源自你的3格光环区域的灵光。 \
若拥有【圣洁】原型，你会散发祝福灵光；若拥有【邪秽】原型，你会散发灾祸灵光。 \
灵光没有明显的视觉效应，但会被魔法侦测出属于祝福还是灾祸灵光。任何与你信仰相同的信徒身处灵光内时，总能立即知晓你是这一信仰的圣武士。
你可以使用附赠动作压制・恢复灵光。18级起，崇圣灵光的光环范围提升至6格。

#### 祝福灵光
灵光显现期间，你获得以下增益。
- **破敌灵光**：你对灵光内拥有下列原型的生物发动的武器攻击，APR获得［职业等级÷4］点加值：邪秽、天族、邪魔、亡灵。
- **守护灵光**：6级起，你与灵光内盟友的基础豁免获得［(施法属性值-9)÷4］点内在加值。
- **勇气灵光**：10级起，你与灵光内盟友免疫恐慌状态。已陷入的恐慌状态会在身处灵光期间被压制。

#### 灾祸灵光
灵光显现期间，你获得以下增益。
- **破敌灵光**：你对灵光内拥有下列原型的生物发动的武器攻击，APR获得［⌈职业等级÷4⌉］点加值：圣洁、天族、邪魔、亡灵。
- **堕落灵光**：6级起，你的基础豁免获得［(施法属性值-9)÷4］点内在加值；灵光内信仰与你不同的生物的基础豁免受到［(施法属性值-9)÷4］点内在减值。
- **绝望灵光**：10级起，你免疫恐慌状态，已陷入的恐慌状态会在灵光显现期间被压制；灵光内信仰与你不同的生物对抗恐慌的豁免检定具有劣势。 \
  对于免疫恐慌的生物，会改为使其失去对恐慌的免疫。此类生物离开灵光后，其遭受的恐慌状态会被压制，直到重新进入灵光。

### 3级：圣武斩
你始终准备着【至圣斩】法术与【制裁斩】奇迹。

### 3级：神圣感知
你始终准备着【神圣感知】奇迹。

### 3级：信实坐骑
你始终准备着【寻获坐骑】法术。你可以消耗1点愿力，以［⌈职业等级÷2⌉］环施展【寻获坐骑】，无需法术位。以这种方式施法时，你的坐骑会表现出与你的神力源泉相符的性质。

### 4级：祷念
神圣之力充盈你身。你拥有1点祷念，此后每提升8级再获得1点。你可以将祷念分配在武器、防具 (包括护甲与盾牌) 或是圣徽上，并在持握期间获得对应增益。 \
X等于为该物品分配的祷念点数。完成短休・长休时，你可以重新分配祷念。
- **武器**：你使用此武器发动的攻击命中崇圣灵光内的生物时，可以额外对其造成［[X]D8］点伤害，伤害类型与你的愿力性质相同。你可以选择以光耀代替正能量伤害，以暗蚀代替负能量伤害。
- **防具**：你会在自己的回合开始时获得［[2X]PD+SKM］点内在障壁，持续至你的下个回合开始。
- **圣徽**：当你施展施法时间为**标准动作**、仅影响你自己、并且不属于敌对行为的本职法术时，可以消耗1点愿力使用附赠动作施展此法术。 \
  若为本次施法消耗法术位，其环阶不能超过［最大环阶-2×(3-X)］(最低1) 环。

### 7级：神选勇士
你获得1项满足条件的战斗・导能专长。17级起，再获得1项满足条件的战斗・导能专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

### 9级：弃绝众敌
以一个【魔法】动作消耗1点愿力，你让敌人被恐惧吞没。你高举圣徽或武器，选择位于你12格内最多［SKM］(最低1) 名可见生物作为目标。 \
迫使每名目标进行1次意志豁免，失败则陷入恐慌状态，持续1分钟或在其受到任何伤害时提前结束。因此陷入恐慌的生物自己的每个回合只能执行下列选项之一：1次移动、1个标准动作、1个附赠动作。

### 13级：崇圣专注
施展1道需要维持专注的本职法术时，如果目标符合你的**破敌灵光**条件，并且是该法术的唯一目标，你可以免除专注，该法术将生效至其持续时间结束。 \
你同时只能通过本特性免除1道法术的专注。

### 14级：神契
如果你拥有正能量源泉，你始终准备着［基本SKM］(最低1) 道【恩惠】奇迹。 \
如果你拥有负能量源泉，你始终准备着［基本SKM］(最低1) 道【暴行】奇迹。 \
在决定上述奇迹获得的法术强化次数时，将圣武士SPR视为高阶。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

在决定下列奇迹获得的法术强化次数时，将圣武士SPR视为高阶。

##### 制裁斩
*二级 塑能* \
**施法时间**：响应，你对崇圣灵光内的生物发动武器攻击时可用 \
**持续时间**：立即 \
在攻击检定掷骰前，你可以消耗1点愿力以［能级］环施展【至圣斩】，无需动作与法术位。 \
以此施展的法术伤害骰尺寸增加2级，伤害类型变为与你的愿力性质相同。你可以选择以光耀代替正能量伤害，以暗蚀代替负能量伤害。 \
若攻击失手，法术能量不会消散，而是缠绕在武器上，持续至回合结束。
- 在你的武器攻击下一次命中崇圣灵光内的生物时，缠绕的能量将倾注于目标，造成上述伤害。
- 若你使用此武器攻击崇圣灵光外的生物，缠绕的能量将提前消散。

:::{admonition} 注意
:class: tip dropdown
缠绕的能量不会叠加，也不会转移至其他武器上。
:::

##### 神圣感知
*二级 预言* \
**施法时间**：附赠动作 \
**持续时间**：谐振，直至失能 \
你可以消耗1点愿力，通过崇圣灵光侦测其范围内的敌人。 \
祝福灵光能够侦测拥有【邪秽】原型的生物，灾祸灵光能够侦测拥有【圣洁】原型的生物。 \
除此之外，任何灵光都能侦测拥有下列原型的生物：天族、邪魔、亡灵。
- 你会时刻得知被侦测的生物占据的方格位置，但不会获知具体位置。
- 在决定能否发动要求目标对你可见才能使用的效应时，视被侦测的生物对你可见。
