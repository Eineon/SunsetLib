# 圣武士

## 圣武士核心特质
- **主要属性**：力量或魅力
- **生命值骰**：D10
- **职业熟练**：中阶APR、中阶SPR
- **豁免倾向**：反射_弱、强韧_强、意志_强
- **武器熟练**：简易武器、军用武器
- **护甲受训**：轻甲、中甲、重甲、盾牌

## 成为圣武士

### 作为1级角色
- 获得「圣武士核心特质」里的所有能力
- 获得圣武士的1级特性

### 作为兼职角色
- 获得「圣武士核心特质」里的以下能力：生命值骰，职业熟练，豁免倾向，军用武器熟练，轻甲、中甲和盾牌的护甲受训
- 获得圣武士的1级特性

### 选择子职业
你可以选择1种满足条件的圣武士子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

##### 1级：施法
你已经学会了如何通过祈祷与冥想来施展法术。
- **魔法根源**：本职业的魔法根源为**神术**。
- **关键属性**：本职业的施法关键属性为**魅力**。
- **准备法术**：你可以从已知法术中准备最多［⌈(职业等级+SKM)÷2⌉］(最低1) 道1环及以上，且不超过最大环阶的法术，这也被称为术法，它们会成为圣武士的本职法术。
- **调整法术**：每次完成长休时，你都可以重新准备1道以此准备的术法。

##### 1级：审判
你可以使用附赠动作消耗［⌈职业等级÷4⌉］点MP，对6格内可见的1名敌人降下审判，持续至战斗结束，或目标的HP降至0。 \
你最多能够同时审判［SKM］(最低1) 名生物。15级起，你可以同时对多名敌人降下审判，并分别为每名目标消耗MP。
- 你对审判目标发动攻击时，APR获得［⌈职业等级÷4⌉］点加值。
- 当位于你威胁范围内的审判目标执行【撤离】动作，或它的攻击命中除你以外的生物时，你可以立刻对该目标发动1次借机攻击。
- 只要审判目标与你身处同一个位面，你就能时刻得知目标占据的方格位置，但不会获知具体位置。
- 对抗不可见的审判目标时，它对你发动的攻击检定不会因此获得优势。
- 6级起，在对抗【守护灵光】区域内的审判目标发动的攻击时，你与灵光内盟友的AC也会因此获得内在加值。

### 1级：武备精通
选择合计2种武器与护甲激活精通词条。此后每提升5级，再选择2种武器或护甲激活精通词条。完成长休时，你可以重新选择合计2种武器与护甲。 \
若从多个职业获得武备精通，将圣武士等级加上其他职业等级决定本特性的效果。

### 2级：战斗风格
你获得1项满足条件的战斗专长。此后每提升5级，再获得1项满足条件的战斗专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

### 2级：神力源泉
经由神祇的祝福，你获得了一个能量池，如同牧师的同名特性。将［圣武士等级÷2］作为牧师等级决定本特性的效果。 \
若从多个职业获得神力源泉，将［圣武士等级÷2］加上其他职业等级决定本特性的效果。你必须为所有神力源泉选择同一种性质。

### 2级：接触导能
你获得【接触导能】作为奖励专长。

### 3级：引导神力
你能从外层位面引导神圣能量来引动魔法效应。你获得以下引导神力选项。如果某个引导神力的效应要求豁免，使用你的法术DC作为豁免DC。
- **制裁斩**：你始终准备着【至圣斩】法术。 \
  对审判目标发动近战武器攻击时，你可以消耗1点愿力，在掷骰前以［⌈职业等级÷2⌉］环施展【至圣斩】，无需动作与法术位。 \
  若攻击命中，此法术的伤害检定获得［职业等级］点加值。若攻击失手，法术能量也会随之消散。 \
  此法术的伤害类型变为与你的神力源泉性质相同，但无法造成治疗。
- **神圣感知**：以一个附赠动作消耗1点愿力，你可以扩展你的意识侦测天族、邪魔以及亡灵生物。持续10分钟内或直至你陷入失能状态，你能感知到自身周围12格内这些生物的位置，以及它们的对应原型。你也可以侦测任何受到【圣居】一类的法术祝福或亵渎的地点或物件。
- **神力掌控**：以一个附赠动作消耗1点愿力，你可以触碰圣徽进行祈祷，以此恢复［(职业等级-1)÷2］点MP。本选项在2次长休之间最多只能使用3次。 \
  若从多个职业获得本选项，则共享使用次数，并将［圣武士等级-1］加上其他职业等级决定恢复的MP点数。

### 5级：信实坐骑
你能唤来异界坐骑的协助。你始终准备着【寻获坐骑】法术。此外，你可以不消耗法术位地施展该法术一次，而后你必须完成一次长休才能再次这么做。

### 6级：守护灵光
你散发出无形的保护性的灵光，覆盖源自你的2格光环区域。在你陷入失能状态期间，此灵光失效。 \
你与灵光内盟友的基础豁免获得［(当前施法属性值-9)÷4］(最低1) 点内在加值。

### 9级：弃绝众敌
以一个【魔法】动作消耗1点愿力，你让敌人被恐惧吞没。你高举圣徽或武器，选择位于你12格内最多［SKM］(最低1) 名可见生物作为目标。 \
迫使每名目标进行1次意志豁免，失败则陷入恐慌状态，持续1分钟或在其受到任何伤害时提前结束。因此陷入恐慌的生物自己的每个回合只能执行下列选项之一：1次移动、1个标准动作、1个附赠动作。

### 10级：勇气灵光
你与你的盟友在守护灵光内具有对恐慌状态的免疫。若陷入恐慌的盟友进入灵光，该状态在灵光内不会造成影响。

### 11级：崇圣打击
神圣之力充盈你身，连带你的武器也承载着超自然力量。当你使用近战武器进行攻击检定并命中目标时，目标额外受到1D8点伤害。 \
此额外伤害的伤害类型与你的神力源泉性质相同，但无法造成治疗。

### 13级：审判专注
施展1道需要维持专注的圣武士法术时，如果审判目标是该法术的唯一目标，你可以免除专注，效果会一直持续至对目标的审判结束。 \
同时只能通过本特性免除1道法术的专注。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。
