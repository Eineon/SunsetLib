# 奇械师

## 奇械师核心特质
- **主要属性**：智力
- **生命值骰**：D8
- **职业熟练**：中阶APR、中阶SPR
- **豁免倾向**：反射_弱、强韧_强、意志_强
- **武器熟练**：简易武器
- **护甲受训**：轻甲、中甲、盾牌
- **额外熟练**：1项技能、盗贼工具、修补工具、1项工匠工具

## 成为奇械师

### 作为1级角色
- 获得「奇械师核心特质」里的所有能力
- 获得奇械师的1级特性

### 作为兼职角色
- 获得「奇械师核心特质」里的以下能力：生命值骰，职业熟练，豁免倾向，轻甲、中甲和盾牌的护甲受训，1项技能熟练，修补工具熟练
- 获得奇械师的1级特性

### 选择子职业
你可以选择1种满足条件的奇械师子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：施法
你已经了解到如何通过物品引导魔法能量。
- **魔法根源**：本职业的魔法根源为**奥术**与**原能**。
- **关键属性**：本职业的施法关键属性为**智力**。
- **必备工具**：你可以使用工匠工具作为本职法术的施法法器，并且可以通过操作工匠工具来满足本职法术的姿势成分。 \
  此外，你也将以下工具视为工匠工具：草药工具、制毒工具、易容工具、盗贼工具、领航工具。
- **戏法**：你可以从已知法术中准备0环法术，这也被称为戏法。施展戏法不需要消耗法术位。 \
  「奇械师特性」表中的**戏法**一列显示了你每职业等级能够准备的戏法总数。完成长休时，你可以重新准备戏法。
- **法术构型**：你知晓已知法术中所有0环法术的构型，并掌握其中6道你选择的1环法术构型。此后每提升1级，再从已知法术中习得2道［⌈职业等级÷2⌉］(最高8) 环以内的法术构型。
- **解析构型**：当你发现1道［⌈职业等级÷2⌉］(最高8) 环以内的已知法术时，可以消耗［(环阶+1)×环阶×5］分钟与［环阶］袋金币，习得该法术的构型。这可以与【学习法术】同时进行。
- **蓝图**：你可以将法术构型绘制为蓝图。绘制戏法蓝图是一个【研究】动作，绘制术法蓝图则需要［术法环阶］分钟。你也可以通过1小时的整备一次性绘制所有蓝图。 \
  蓝图只能被你自己或者施展了【鉴定术】的生物阅读。只要携带着蓝图，就将该法术视为奇械师的本职法术。

#### 奇械
你可以执行【魔法】动作，阅览随身携带的1张蓝图，制造1台被称为奇械的复杂机械装置，用于实现此蓝图中记载的法术。 \
每控制1台奇械都需要占用1点带宽，你可以主动切断对任意奇械的控制 (无需动作)，失去控制的奇械将立即无害自毁。 \
你会时刻得知受控奇械与自身之间的相对距离。距离你超过20格的奇械将失去控制。 \
你无法在带宽占用到达上限时制造新的奇械。「奇械师特性」表中的**带宽**一列显示了你每职业等级的带宽额度。

#### 启动奇械
只要你愿意，任何智力4+的生物都可以尝试启动受你控制的奇械，如同施展蓝图中记载的法术。一台奇械可以被多次启动。
- **启动者**：将启动者作为法术的施法者。但在决定法术效果时，使用你的SPR与SKM。
- **法术位**：法术所需的法术位由你提供。若未能提供法术位，奇械将启动失败，启动者消耗的动作也会浪费。
- **成分**：法术额外获得「F (此奇械)」成分。启动者需要提供法术的所有成分，并且必须持握并操作奇械来满足法术的S与F成分，无法免除。 \
  你以及其他奇械师可以用持握工匠工具的手持握奇械来满足这项额外的F成分。
- **专注**：若法术要求专注，由启动者维持专注并进行专注豁免。
- **持续时间**：持续超过一瞬间的法术，会在奇械被摧毁时提前结束。在持续期间再次激活相同奇械时，之前的效果也会提前结束。

### 1级：注法
你知晓一些用来制造魔法物品的特殊法术。
- **准备注法**：「奇械师特性」表中的**注法**一列显示了你每职业等级能够准备的注法总数。完成长休时，你可以重新准备注法。 \
  你始终准备着所有满足条件的0级注法，且不计入你能够准备的注法总数中。
- **注法成分**：所有注法都拥有「S、F (工匠工具)」成分，无法免除。你必须通过操作工匠工具满足这些成分。

#### 灵感
你获得［⌈职业等级÷4⌉+CKM］(最低1) 点灵感，可以通过消耗灵感施展注法。已消耗的灵感会在完成长休时恢复。 \
「奇械师特性」表中的**激励骰**一列显示了你每职业等级的激励骰尺寸。效果表述中的「**激励_N**」代表投掷N枚激励骰并将骰值相加。 \
若从多个职业获得灵感，将奇械师等级加上其他职业等级决定灵感点数与激励骰尺寸。

### 2级：魔法导师
你获得1项满足条件的魔导专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

### 2级：万事通
你获得以下增益。
- **专精**：选择1项已熟练的技能获得专精。
- **博学**：你获得【即兴发挥】作为奖励专长。为回忆知识而进行不具熟练的属性检定时，你可以加入2点PB。13级起，改为加入4点PB。 \
	若从多个职业获得博学，将奇械师等级加上其他职业等级决定本特性的效果。

### 3级：逆向工程
使用已熟练的工具进行属性检定时，你总是可以使用施法属性作为关键属性。 \
15级起，你可以无视同调或使用魔法物品时所需的任何职业、种族、法术和等级条件。

### 4级：黄金数
你使用已熟练的工具进行的属性检定获得［职业等级÷4］点内在加值。

### 5级：灵光一闪
每谐振1次，你使用已熟练的工具进行的属性检定掷出16或更高，会获得［(职业等级+1)÷6］点灵感，持续时间如同「谐振」。此灵感只能用于施展注法。 \
此外，你随时可以消耗2点MP换取1点灵感，这无需动作。

### 6级：熟练匠师
你对如何使用和制作魔法物品有着深刻的理解。 \
制作一件非魔法物品、或是稀有度为普通或非普通的魔法物品时，你需要消耗的时间为通常的$\frac{1}{4}$，金钱为通常的$\frac{1}{2}$。 \
此外，你可以通过8小时的整备，将1件魔法物品重铸为价值相近 (由DM判断)，并且你能够制作的其他魔法物品。 \
10级、14级、18级起，你分别会获得2个额外的同调位。

### 7级：校准工程
你获得1项满足条件的塑法专长。17级起，再获得1项满足条件的塑法专长。 \
每回合1次，你可以消耗1点灵感代替移动动作激活以此获得的塑法专长。你可以在此时消耗那些通常只能用于施展注法的灵感。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

### 9级：可靠才能
每当你进行属性检定，并且可以运用你的某项技能熟练或工具熟练时，你可以将D20投出的9及以下的结果视为10。 \
若拥有多个包含本特性的职业，将［奇械师等级-2］(最低0) 加上其他职业等级决定获得本特性的等级。

### 13级：瞬间工程
每回合1次，你可以消耗1点灵感，在自己的回合执行1个额外的【魔法】或【操作】动作。此【魔法】动作只能用于施展注法或激活魔法物品。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

### 20级：奇械之魂
完成短休・长休时，你可以对自身施展【触发术】，并且无需法术成分，也无需为其与凭依法术消耗法术位。 \
你总是可以选择【防死结界】作为凭依法术，即使你无法施展它。 \
凭依法术被触发时，你可以占用2点带宽避免触发术终止，令其能够在相应状况下次出现时再次触发。因此占用的带宽会在你完成短休・长休时恢复。


## 注法
在决定注法获得的法术强化次数时，将奇械师SPR视为高阶。

### 关键术语
- **强度**：注法拥有［⌈职业等级÷2⌉］级强度。在对抗【解除魔法】以及其他类似效应时，使用强度作为注法的法术环阶。
- **附魔**：附魔注法可以为拥有镀装级别的物品添加附魔。一件物品最多能够同时拥有［其镀装级别-1］项附魔。

### 注法条目
- **级别**：你能够准备［⌈职业等级÷2⌉］级以内的注法。注法视为戏法，施法时无需消耗法术位。
- **学派**：所有注法都会散发魔法学派灵光，也会被侦测出属于注法。
- **先决**：你只能准备满足先决条件的注法。若条件失效，你将暂时失去该注法，直到条件被重新满足。
- **持续时间**：施展持续超过一瞬间的注法后，如果再次施展相同注法，之前施展的注法会提前结束。你可以主动解除持续中的注法，这无需动作。

#### 持续时间
- **同调**：记作「**同调_N**」。维持本注法需要占用N个同调位，在注法持续期间压制原本与这些同调位建立连结的效应 (例如同调物品的魔法效应)。
- **带宽**：记作「**带宽_N**」。维持本注法需要占用N点带宽。
