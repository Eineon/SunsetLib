# 牧师

## 牧师核心特质
- **主要属性**：感知
- **生命值骰**：D6
- **职业熟练**：低阶APR、高阶SPR
- **豁免倾向**：反射_弱、强韧_弱、意志_强
- **武器熟练**：简易武器
- **护甲受训**：轻甲

## 成为牧师

### 作为1级角色
- 获得「牧师核心特质」里的所有能力
- 获得牧师的1级特性

### 作为兼职角色
- 获得「牧师核心特质」里的以下能力：生命值骰，职业熟练，豁免倾向，轻甲的护甲受训
- 获得牧师的1级特性

### 选择子职业
你可以选择1种满足条件的牧师子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：施法
你通过祈祷，冥想与奉献习得如何施法。
- **魔法根源**：本职业的魔法根源由你选择的术法传承决定。
- **关键属性**：本职业的施法关键属性为**魅力**。
- **特殊法器**：若你空着主手，则可以使用自己的肉体作为本职法术的施法法器。
- **准备法术**：你可以从已知法术中准备不超过最大环阶的法术，它们会成为牧师的本职法术。 \
  「牧师特性」表中的**戏法**一列显示了你每职业等级能够准备的0环法术总数，这也被称为戏法。施展戏法不需要消耗法术位。 \
  「牧师特性」表中的**术法**一列显示了你每职业等级能够准备的1环及以上法术总数，这也被称为术法。
- **调整法术**：每次完成长休时，你都可以重新准备任意道以此准备的法术。
- **术法转换**：完成短休时，你可以重新准备最多［⌈职业等级÷2⌉］道本职术法。

### 1级：奇迹
神祇的祝福赐予了你不可思议的奇迹法术。 \
若从多个职业获得奇迹，将牧师等级加上其他职业等级决定你能够准备的最大奇迹级别与奇迹强度。
- **准备奇迹**：你可以将满足条件的奇迹视为牧师的已知法术进行准备。 \
  4级起，以及之后每提升4级，你分别可以选择1道满足条件的奇迹始终准备。每次提升职业等级时，你可以重新选择1道奇迹。
- **奇迹成分**：所有奇迹都拥有「F (圣徽)」成分，无法免除。

#### 神力源泉
经由神祇的祝福，你获得了愿力能量池。选择**正能量**与**负能量**之一作为愿力的性质。一旦做出选择，你就无法在没有神力介入的情况下更改神力源泉。 \
你的能量池中流淌着2点愿力。你可以消耗愿力施展奇迹，已消耗的愿力会在完成短休・长休时恢复。6级与18级起，你额外获得1点愿力。 \
若从多个职业获得神力源泉，将牧师等级加上其他职业等级决定本特性的效果。你必须为所有神力源泉选择同一种性质。
- **正能量源泉**：你始终准备着【引导能量】奇迹，并获得【圣洁】原型。
- **负能量源泉**：你始终准备着【引导能量】奇迹，并获得【邪秽】原型。

### 1级：圣职
你让自己投身于以下一种由你自己选择的神圣职能：
- **保护者**：为战斗做足训练，你获得军用武器熟练与中甲、重甲和盾牌受训。将牧师生命骰尺寸提升至D8、APR提升至中阶、强韧倾向提升至强。如果你拥有牧师之外的职业等级，则不会因此获得重甲受训。
- **修道者**：你获得1项技能与草药工具的熟练。若你已有草药工具的熟练，则自选1项工具获得熟练。 \
  你可以从已知法术中额外准备1道戏法。此外，你与神性的神秘链接使你在使用奥秘或宗教技能进行属性检定时，可以使用施法属性作为关键属性。 \
  2级起，你获得【万事通】特性，如同吟游诗人的同名特性。若从多个职业获得博学，将牧师等级加上其他职业等级决定博学的效果。

### 2级：引导神力
你获得1项满足条件的塑法・导能专长。此后每提升5级，分别再获得1项满足条件的塑法・导能专长。你也可以选择【精通塑法】或【额外聚能】代替塑法・导能专长。 \
每回合1次，你可以消耗1点愿力代替移动动作激活以此获得的专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

### 7级：受祝击
神圣的力量在战斗中注入你。你从以下特性中选择其一获得。
- **神圣打击**：当你发动的武器攻击命中时，可以对目标额外造成1D8点伤害，伤害类型与你的愿力性质相同。一名生物每回合最多承受1次此伤害。你可以选择以光耀代替正能量伤害，以暗蚀代替负能量伤害。
- **强力施法**：当你施展的本职戏法造成基础伤害时，你可以对目标额外造成［SKM］点相同类型的伤害。

### 10级：神圣干预
你能呼唤你所信奉的神祇或神系来代你降下干预。以一个【魔法】动作，你选择1道5环以内的已知法术 (施法时间不能为反应)，并将该法术作为执行此动作的一部分以5环施展，无需消耗法术位也无需对应的施法材料。直到你完成一次长休前，你都无法再次使用此特性。

### 14级：精通受祝击
你所选择的受祝击变得更加强大。
- **神圣打击**：神圣打击的额外伤害提升至2D8。
- **强力施法**：当你施展的本职戏法造成基础伤害时，你可以为自己或12格内可见的另一个生物注入活力，使其获得［SKM×2］(最低2) 点增强障壁。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

### 20级：进阶神圣干预
你能够呼唤更加强大的神圣干预。当你使用神圣干预时，你可以在选择法术时选择祈愿术。如果你这样做，你只有在完成2D4次长休后才能再次使用神圣干预。

## 奇迹

### 关键术语
- **强度**：奇迹拥有［⌈职业等级÷2⌉］级强度。在对抗【解除魔法】以及其他类似效应时，使用强度作为奇迹的法术环阶。

### 奇迹条目
- **级别**：你能够准备［⌈职业等级÷2⌉］级以内的奇迹。奇迹视为戏法，施法时无需消耗法术位。
- **学派**：所有奇迹都会散发魔法学派灵光，也会被侦测出属于奇迹。
- **先决**：你只能准备满足先决条件的奇迹。若条件失效，你将暂时失去该奇迹，直到条件被重新满足。
- **施法时间**：此处列出了施展奇迹所需使用的动作。 \
  如果记述为「响应」，则会作为其他动作的一部分施展，不能在无法执行反应时施展。
- **施法频率**：此处列出了奇迹在特定时间内能够施展的次数上限。没有次数限制的奇迹不会显示本条目。

#### 一级

##### 引导能量
*一级 死灵* \
**施法时间**：可变 \
**持续时间**：立即 \
你消耗1点愿力引导神力源泉中的能量。如果是正能量源泉，给予目标［强度］点命能；如果是负能量源泉，则给予相同点数的虚能。 \
根据目标的原型，每点能量都会为其恢复1D8点HP，或是造成1D8点伤害。
- **命能生物**：拥有【命能治疗】原型的生物会被命能治疗，被虚能伤害。获得虚能时必须进行1次强韧豁免，失败则受到负能量伤害，成功则伤害减半。
- **虚能生物**：拥有【虚能治疗】原型的生物会被命能伤害，被虚能治疗。获得命能时必须进行1次强韧豁免，失败则受到正能量伤害，成功则伤害减半。

本奇迹的许多元素由你施展它所用的动作决定。
- **附赠动作**：施法距离为触碰。
- **标准动作**：施法距离为6格，造成的治疗与伤害增加［SKM］(最低0) 点，并且治疗与伤害的骰子尺寸提升至D12。
- **整轮动作**：奇迹会影响源自你的6格光环区域内的所有命能与虚能生物。

##### 神言斥责
*一级 惑控* \
**施法时间**：标准动作 \
**持续时间**：1分钟 \
消耗1点愿力，你展现圣徽，斥退诸敌。迫使源自你的6格光环区域内拥有特定原型的生物进行1次意志豁免，失败则在奇迹持续期间陷入<轻度屈服：驱散>状态。被驱散的生物将会同时陷入恐慌与失能状态，并尽可能远离你。 \
【无心智】原型无法免疫此屈服状态，此状态会在目标受到伤害时提前结束，也会在你陷入失能・死亡时提前结束。
- 如果你拥有正能量源泉，本奇迹将会驱散拥有【虚能治疗】或【邪秽】原型的生物。
- 如果你拥有负能量源泉，本奇迹将会驱散拥有【命能治疗】或【圣洁】原型的生物。

##### 神力掌控
*一级 咒法* \
**施法时间**：附赠动作 \
**施法频率**：3次/长休 \
**持续时间**：立即 \
消耗1点愿力，你触碰圣徽进行祈祷，以此恢复［强度］点MP。

##### 恩惠
*一级 防护* \
**先决**：正能量源泉 \
**施法时间**：响应，施展【引导能量】时可用 \
**持续时间**：立即 \
准备本奇迹时，从以下状态中选择1项：魅惑、恐慌、中毒。 \
消耗触发奇迹给予的1点命能，解除该奇迹目标中所有命能生物承受的所选状态。 \
如果目标生物从多个来源承受所选状态，你需要分别为每个来源消耗1点命能才能解除其承受的状态。 \
**特殊**：你可以多次准备本奇迹，每次必须选择不同的状态。

<**强化_1**>：将下列状态加入本奇迹的可选状态列表：目盲、耳聋。 \
<**强化_2**>：将下列状态加入本奇迹的可选状态列表：束缚、震慑。 \
<**强化_3**>：将下列状态加入本奇迹的可选状态列表：石化、麻痹。

##### 暴行
*一级 惑控* \
**先决**：负能量源泉 \
**施法时间**：响应，施展【引导能量】时可用 \
**持续时间**：谐振 \
准备本奇迹时，从以下状态中选择1项：魅惑、恐慌、中毒。 \
消耗触发奇迹给予的1点虚能，在本奇迹持续期间，赋予触发奇迹目标中所有豁免失败的命能生物所选状态。 \
此状态会在目标受到伤害时提前结束，也会在你陷入失能・死亡时提前结束。 \
**特殊**：你可以多次准备本奇迹，每次必须选择不同的状态。

<**强化_1**>：将下列状态加入本奇迹的可选状态列表：目盲、耳聋。 \
<**强化_2**>：将下列状态加入本奇迹的可选状态列表：束缚、震慑。 \
<**强化_3**>：将下列状态加入本奇迹的可选状态列表：石化、麻痹。
