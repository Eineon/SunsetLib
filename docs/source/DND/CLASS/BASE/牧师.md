# 牧师

## 牧师核心特质
- **主要属性**：感知
- **生命值骰**：D6
- **职业熟练**：低阶APR、高阶SPR
- **豁免倾向**：反射_弱、强韧_强、意志_强
- **武器熟练**：简易武器
- **护甲受训**：轻甲

## 成为牧师

### 作为1级角色
- 获得「牧师核心特质」里的所有能力
- 获得牧师的1级特性

### 作为兼职角色
- 获得「牧师核心特质」里的以下能力：生命值骰，职业熟练，豁免倾向，轻甲的护甲受训
- 获得牧师的1级特性

### 选择子职业
你可以选择1种满足条件的牧师子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：神力源泉
经由神祇的祝福，你获得了一个能量池。从**正能量**与**负能量**之一作为神力源泉的性质。一旦做出选择，你就无法在没有神力介入的情况下更改神力源泉。 \
你的能量池中流淌着2点愿力，已消耗的愿力会在完成短休时恢复上限一半的次数、完成长休时恢复所有次数。4级起，以及之后每提升4级，你额外获得1点愿力。 \
若从多个职业获得神力源泉，将牧师等级加上其他职业等级决定本特性的效果。你必须为所有神力源泉选择同一种性质。

#### 正能量源泉
你始终准备着【疗伤术】。你能够消耗1点愿力，施展［⌈职业等级÷2⌉］环的【疗伤术】，并且无需法术位。 \
当你通过这种方式施法时，可以消耗1D8点治疗量，解除目标承受的中毒状态。 \
以此施展的【疗伤术】不会治疗拥有【负能量治疗】原型的生物，而是对其造成等量的正能量伤害。目标可以进行1次对抗法术DC的强韧豁免，成功则伤害减半。

#### 负能量源泉
你始终准备着【致伤术】。你能够消耗1点愿力，施展［⌈职业等级÷2⌉］环的【致伤术】，并且无需法术位。 \
当你通过这种方式施法时，可以消耗1D8点伤害值，使豁免失败的目标陷入中毒状态，持续1分钟。 \
以此施展的【致伤术】不会伤害拥有【负能量治疗】原型的生物，而是为其恢复等量的HP。

### 1级：施法
你通过祈祷，冥想与奉献习得如何施法。
- **魔法根源**：本职业的魔法根源由你选择的术法传承决定。
- **关键属性**：本职业的施法关键属性为**魅力**。
- **特殊法器**：若你空着主手，则可以使用自己的肉体作为本职法术的施法法器。
- **准备法术**：你可以从已知法术中准备不超过最大环阶的法术，它们会成为牧师的本职法术。 \
  「牧师特性」表中的**戏法**一列显示了你每职业等级能够准备的0术法术总数，这也被称为戏法。施展戏法不需要消耗法术位。 \
  「牧师特性」表中的**术法**一列显示了你每职业等级能够准备的1环及以上法术总数，这也被称为术法。
- **调整法术**：每次完成长休时，你都可以重新准备任意道以此准备的法术。
- **术法转换**：完成短休时，你可以重新准备最多［⌈职业等级÷2⌉］道本职术法。

### 1级：圣职
你让自己投身于以下一种由你自己选择的神圣职能：
- **保护者**：为战斗做足训练，你获得军用武器熟练与中甲、重甲和盾牌受训。将牧师生命骰视为D8、APR视为中阶。
- **修道者**：你获得1项技能与草药工具的熟练。若你已有草药工具的熟练，则自选1项工具获得熟练。 \
  你可以从已知法术中额外准备1道戏法。此外，你与神性的神秘链接使你在使用奥秘或宗教技能进行属性检定时，可以使用施法属性作为关键属性。 \
  2级起，你获得【万事通】特性，如同吟游诗人的同名特性。若从多个职业获得博学，将牧师等级加上其他职业等级决定博学的效果。

### 2级：魔法导师
你获得1项满足条件的魔导专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

### 2级：引导神力
你能从外层位面引导神圣能量来引动魔法效应。你获得以下引导神力选项。如果某个引导神力的效应要求豁免，使用你的法术DC作为豁免DC。
- **神圣火花**：当你通过【神力源泉】特性施展【疗伤术】或【致伤术】时，可以使法术影响源自你的6格光环区域内的所有生物，此时法术的治疗量与伤害值变为［[环阶]D6+SKM］点。你无法同时使用本选项与【接触导能】专长。
- **神言斥责**：以一个【魔法】动作消耗1点愿力，你展现圣徽，斥退诸敌。迫使源自你的6格光环区域内所有符合条件的生物进行1次意志豁免，失败则陷入恐慌与失能状态，持续1分钟。在这段持续时间内，被驱散的生物会尽可能远离你。此效应在其受到伤害时提前结束，你陷入失能状态或是死亡也会结束此效应。 \
  如果你的神力源泉为正能量，此效应会影响拥有【负能量治疗】原型的生物。反之则会影响能够被正能量治疗的生物。 \
  5级起，你可以对豁免失败的生物造成［[环阶]D6+SKM］点对应类型的伤害，这不会终止驱散效应。
- **神力掌控**：以一个附赠动作消耗1点愿力，你可以触碰圣徽进行祈祷，以此恢复［职业等级÷2］点MP。本选项在2次长休之间最多只能使用3次。 \
  若从多个职业获得本选项，则共享使用次数，并将牧师等级加上其他职业等级决定恢复的MP点数。

### 7级：受祝击
神圣的力量在战斗中注入你。你从以下特性中选择其一获得。
- **神圣打击**：每回合1次，当你使用武器的攻击命中了一名生物时，你可以使该生物额外受到1D8点伤害。 \
  此额外伤害的伤害类型与你的神力源泉性质相同，但无法造成治疗。
- **强力施法**：当你施展的本职戏法造成基础伤害时，你可以对其中1名目标额外造成［SKM］点相同类型的伤害。

### 10级：神圣干预
你能呼唤你所信奉的神祇或神系来代你降下干预。以一个【魔法】动作，你选择1道5环以内的已知法术 (施法时间不能为反应)，并将该法术作为执行此动作的一部分以5环施展，无需消耗法术位也无需对应的施法材料。直到你完成一次长休前，你都无法再次使用此特性。

### 14级：精通受祝击
你所选择的受祝击变得更加强大。
- **神圣打击**：神圣打击的额外伤害提升至2D8。
- **强力施法**：当你施展的本职戏法造成基础伤害时，你可以为自己或12格内可见的另一个生物注入活力，使其获得［SKM×2］(最低2) 点增强障壁。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

### 20级：进阶神圣干预
你能够呼唤更加强大的神圣干预。当你使用神圣干预时，你可以在选择法术时选择祈愿术。如果你这样做，你只有在完成2D4次长休后才能再次使用神圣干预。
