### 作为1级角色

### 作为兼职角色
- 获得「[职业名称]核心特质」里的以下能力：生命值骰、职业熟练、豁免倾向

### 选择子职业
你可以选择1种满足条件的[职业名称]子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。
