# 战士

## 战士核心特质
- **主要属性**：力量或敏捷
- **生命值骰**：D10
- **职业熟练**：高阶APR
- **豁免倾向**：反射_弱、强韧_强、意志_弱
- **武器熟练**：简易武器、军用武器
- **护甲受训**：轻甲、中甲、重甲、盾牌

## 成为战士

### 作为1级角色
- 获得「战士核心特质」里的所有能力
- 获得战士的1级特性

### 作为兼职角色
- 获得「战士核心特质」里的以下能力：生命值骰，职业熟练，豁免倾向，军用武器熟练，轻甲、中甲和盾牌的护甲受训
- 获得战士的1级特性

### 选择子职业
你可以选择1种满足条件的战士子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：战斗技巧
你的战斗技巧在战场上接受了磨砺，你习得战技并获得技力。
- **技力**：你获得［(APR+基本职业属性值-9)÷4］(最低1) 点技力，可以通过消耗技力使用战技。已消耗的技力会在完成短休・长休时恢复。
- **战技**：你可以习得战技。「战士特性」表中的**战技**一列显示了你每职业等级能够习得的战技总数。 \
  每次提升职业等级时，你都可以更换1项已习得的战技。如果战技的效果要求豁免，使用你的职业DC作为豁免DC。
- **卓越骰**：「战士特性」表中的**卓越骰**一列显示了你每职业等级的卓越骰尺寸。效果表述中的「**卓越_N**」代表投掷N枚卓越骰并将骰值相加。
- **卓越战斗**：在决定战斗专长的效果时，你可以使用卓越骰代替熟练骰。

### 1级：回气
你可以利用有限的精力与体力来重整旗鼓。以一个附赠动作，恢复［[卓越_2]+职业等级］点HP。 \
本特性可以使用2次，并且在完成短休后恢复1次使用次数，完成长休后恢复所有使用次数。4级、10级、16级起，分别增加1次使用次数。

### 1级：武备精通
选择合计2种武器与护甲激活精通词条。此后每提升5级，再选择2种武器或护甲激活精通词条。完成长休时，你可以重新选择合计2种武器与护甲。 \
若从多个职业获得武备精通，将战士等级加上其他职业等级决定本特性的效果。

### 2级：战斗风格
你获得1项满足条件的战斗专长。此后每提升3级，分别再获得1项满足条件的战斗专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

### 2级：战术思维
在战场内外，你都能够运用你的战术思维。你获得以下增益。
- **额外熟练**：你获得1项工匠工具的熟练，或是从下列技能中选择1项获得熟练：特技、驯兽、运动、历史、洞悉、威吓、游说、察觉、求生。
- **卓越技艺**：当你进行的属性检定失败时，可以消耗1次回气，使此次检定获得［卓越_1］点加值，这可能会让结果变为成功。如果检定依旧失败，不会消耗回气次数。

### 3级：动作如潮
你可以在短期内让自己突破身为凡骨的极限。你可以在自己的回合执行一个额外的标准动作。 \
9级起，你可以在使用本特性的回合进行第2次全力攻击，并且无需使用移动动作，但在决定其效果时，视APR比实际低4级。 \
使用此特性后，你必须在完成短休・长休才能再次使用。17级起，你可以在每次休息之间使用本特性2次，但每回合只能使用一次。

### 4级：千招百式
每轮1次，你可以使用移动动作消耗1次回气，选择1项已习得的战斗专长，将其临时变招为另一项你能够满足条件的战斗专长，持续时间如同「谐振」。 \
10级起，使用本特性改为附赠动作，并且可以同时选择最多2项战斗专长进行变招。 \
16级起，使用本特性无需任何动作，并且可以同时选择最多3项战斗专长进行变招。

> **》特殊交互** \
> 使用【千招百式】特性时，分配在所选专长上的聚能点不会因变招而改变。 \
> 通过变招获得的聚能专长不会为你提供聚能点。

### 5级：战术转进
使用【回气】特性时，你可以进行**X格即时移动**，X等于你最大速度的一半 (向上取整)。

### 6级：料敌机先
以一个附赠动作，你可以获知一名位于你6格内可见生物的长处和短处。你知晓该生物是否具有抗性、免疫或易伤，若有，则你知晓其具体内容是什么。 \
本特性一经使用，直至完成长休你都无法再次使用。你也可以消耗1点技力重置本特性的使用权。

### 7级：英勇
你可以在使用【踏入传奇】特性时不消耗英雄激励。 \
使用此特性后，你必须在完成短休・长休才能再次使用。15级起，你可以在每次休息之间使用本特性2次。

### 9级：战术主宰
每当你使用武器发动攻击，且已激活此武器的精通词条时，可以将此次攻击的精通词条改为推离、削弱、缓速中的一种。

### 12级：百战不殆
每当你投掷先攻时，恢复1点技力。

### 13级：究明攻击
你已深晓对手的一举一动，并且从每次攻击中吸取经验教训。如果你对一个生物发动攻击但失手，那么直到你的下个回合结束前，你对其发动的下次攻击检定具有优势。

### 18级：无拘反应
当一名敌人开始回合时，你会获得1个反应，持续至其回合结束。这个反应只能用于发动借机攻击、战技或战斗专长。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

### 20级：至高武艺
每回合1次，你可以在激活战技时不消耗技力。


## 战技

### 关键术语
- **强度**：战技拥有［⌈职业等级÷2⌉］级强度。
- **绝技**：当你激活拥有「绝技」标签的战技，或是发动1次攻击后，直到你的下个回合开始为止，无法再激活绝技。
- **架势**：激活持续时间为「架势」的战技时，之前激活的架势将提前结束。架势类似「谐振」，但无法在战斗之外持续。

### 战技条目
- **级别**：你能够习得［⌈职业等级÷2⌉］级以内的战技。
- **类别**：战技分为「迸放」与「派生」两种类别。
- **先决**：你只能习得满足先决条件的战技。若条件失效，你将暂时失去该战技，直到条件被重新满足。
- **需求**：你只能激活满足需求条件的战技。如果你不再满足正在持续的战技的需求条件，其效果将提前结束。
- **激活**：此处列出了激活战技所需使用的动作。你无法针对同一事件同时激活多个战技。 \
  如果记述为「N次攻击」，则需要你在自己的回合执行【攻击】动作，并消耗其中N次攻击。 \
  如果记述为「响应」，则会作为其他动作的一部分激活，不能在无法执行反应时激活。
- **频率**：此处列出了战技在特定时间内能够激活的次数上限。没有次数限制的战技不会显示本条目。

## 子职业

### 奥法骑士
奥法骑士将所有战士共通的高超武技与对魔法的精妙研究融合贯通。法术补足并拓展了他们的武技，为甲胄提供额外的保护，并使他们能够使用爆发性的魔法同时与多个敌人鏖战。

**先决条件**：你必须满足以下所有条件才能选择本子职。
1. 基本智力13+

#### 3级：战争联结
你习得一种让你和你的武器建立魔法联结的仪式。你执行持续一小时的仪式，且可在短休时进行。整个仪式过程中武器必须在你的触及范围内，直至仪式结束前，该武器必须一直在你触手可及的范围内，以完成联结。若该武器已与另一名战士建立联结，或是一把已被他人同调的魔法物品，仪式则会失败。 \
联结建立后，只要你不处于失能状态，使用联结武器时就不会被缴械。如果联结武器与你处于同一位面，则你可以用一个附赠动作使其立即传送到你手中。 \
你至多可以拥有两把建立联结的武器，但一个附赠动作只能召唤一把武器。如果你试图和第三把武器建立联结，则必须与已经联结的两把武器其中的一把取消联结。

#### 3级：施法
你学会了如何施展法术。
- **法术熟练**：本职业的SPR为**低阶**。
- **魔法根源**：本职业的魔法根源为**奥术**。
- **关键属性**：本职业的施法关键属性为**智力**，并将智力加入战士的主要属性。
- **特殊法器**：你可以使用连结武器作为本职法术的施法法器。
- **准备法术**：你可以从已知法术中准备最多［⌈(职业等级+SKM)÷2⌉］(最低1) 道1环及以上，且不超过最大环阶的法术，这也被称为术法，它们会成为奥法骑士的本职法术。
- **调整法术**：每次提升职业等级时，你都可以重新准备1道以此准备的术法。

#### 3级：战争魔法
你可以使用战技DC作为本职法术的豁免DC，这不会改变施法关键属性。

#### 3级：奥能战技
你可以习得奥能战技，并额外习得2项奥能战技。你可以将奥能战技作为法术准备，如同已习得般激活这些战技。

#### 7级：法术打击
你获得【法术打击】作为奖励专长。

#### 10级：奥能冲锋
你可以在使用【动作如潮】特性时传送至6各内一个你能看到的未被占据的空间。你可以在使用【动作如潮】提供的额外动作之前或之后进行传送。

#### 15级：重击施法
当你发动的武器攻击形成重击时，可以使用附赠动作代替【魔法】动作，发起1个敌对行为。 \
若为该行为消耗法术位，其环阶不能超过［最大环阶-2］环。

#### 奥能战技

##### 法术唤回
*一级 迸放* \
**消耗**：3X技力 \
**激活**：附赠动作 \
创造1枚X环 (最高不超过最大环阶) 的法术位，持续至回合结束。 \
使用该法术位施展的法术持续时间若到达10分钟，则变为「谐振」。


### 灵能武士
灵能武士唤醒他们心灵中的力量以增强自身的体魄，他们将这种被称为灵能的力量注入于刀兵之中，用念动力鞭笞他们的敌人，并以精神力铸造无形壁垒抵御攻击。

**先决条件**：你必须满足以下所有条件才能选择本子职。
1. 基本感知13+

#### 3级：灵能力量
你内心拥有一个灵能力量的源泉，为你提供下列增益。
- **主要属性**：将感知加入战士的主要属性。
- **灵能战技**：你会自动习得所有满足条件的灵能战技。

#### 灵能战技

触发攻击对目标造成的伤害减少［[卓越_1]+CKM］点。
