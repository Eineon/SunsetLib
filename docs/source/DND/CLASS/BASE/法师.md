# 法师

## 法师核心特质
- **主要属性**：智力
- **生命值骰**：D6
- **职业熟练**：低阶APR、高阶SPR
- **豁免倾向**：反射_弱、强韧_弱、意志_强
- **武器熟练**：简易武器
- **护甲受训**：无

## 成为法师

### 作为1级角色
- 获得「法师核心特质」里的所有能力
- 获得法师的1级特性（见法师特性表）

### 作为兼职角色
- 获得「法师核心特质」里的以下能力：生命值骰、职业熟练、豁免倾向
- 获得法师的1级特性（见法师特性表）

### 选择子职业
你可以选择1种满足条件的法师子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：施法
你学会了如何运用法力施展法术。
- **魔法根源**：本职业的魔法根源为**奥术**。
- **关键属性**：本职业的施法关键属性为**智力**。
- **戏法**：你可以从已知法术中准备0环法术，这也被称为戏法。施展戏法不需要消耗法术位。 \
  「法师特性」表中的**戏法**一列显示了你每职业等级能够准备的戏法总数。完成长休时，你可以重新准备戏法。
- **藏书**：藏书代表你在法师学徒阶段获取的所有成果。从已知法术中选择6道法术加入藏书列表。此后每提升1级，再选择2道［⌈职业等级÷2⌉］环以内的法术加入藏书列表。你永远无法将戏法加入藏书。
- **扩充藏书**：当你发现1道［⌈职业等级÷2⌉］环以内的已知法术时，可以消耗［(环阶+1)×环阶×5］分钟与［环阶］袋金币，将该法术加入藏书。这可以与【学习法术】同时进行。
- **法术书**：你可以将藏书列表中的任意道法术抄录在实体媒介上，使其成为你的法术书。 \
  每抄录1道法术都需要消耗［此法术环阶］分钟，也可以通过1小时的整备完成全部抄录。法术书只能被你自己或者施展了【鉴定术】的生物阅读。 \
  完成长休时，你可以阅览自己的法术书，从中选择任意道环阶不超过你的最大环阶的法术，将它们准备在**法术列队**中，成为法师的本职法术。
- **准备列队**：你可以在准备法术的同时创建・删改任意组法术列队。 \
  每组法术列队能够准备的最大法术数量由你的职业等级决定，如「法师特性」表中的**列队容量**一列所示。 \
  你需要从0开始，分别为每组法术列队分配一个逐渐增大的序列号。你只能使用法术位施展「0号列队」中的法术。一道法术可以同时准备在多个序列号不同的法术列队中。
- **调整列队**：在你的回合开始前，如果你正持握自己的法术书，则可以选择一个序列号，并减少［序列号×10］点先攻值，这会延迟你开始回合的时间。 \
  当你的回合开始时，所选序列号的法术列队将移动至序列0，所有低于该法术列队原本序列号的其他法术列队，序列号将分别+1。 \
  如果这使你的先攻值变为负值，你会失去这一轮的回合，随后将你的先攻值增加20。如果你的先攻值仍为负值，则需不断重复这一过程。 \
  如果在战斗外调整列队，你需要消耗［(序列号+1)×序列号×5］分钟的时间。
- **固有法术**：准备法术时，你还可以从法术书中选择最多［SKM］(最低0) 道法术作为固有法术进行准备，或调整已准备的固有法术。固有法术总是视为处于0号列队中，不会随着法术列队的更替而变化。 \
  如果其他法师特性给了你始终准备的1环及以上的法术，这些法术同时也视为固有法术，且不计入你能够准备的固有法术总数中。

### 1级：仪式学家
你可以通过阅览法术书，以仪式施展其中带有[仪式]标签的法术。

### 1级：奥术回想
2次长休之间仅1次，你可以在完成短休时，恢复最多［职业等级］点MP。

### 2级：学者
在学习法术之余，你还对一门学术领域有过专门的研究。 \
从下列技能中选择一项你具有熟练的技能：奥秘、历史、自然、宗教。你获得所选技能的专精。

### 2级：魔法导师
你获得1项满足条件的魔导专长。此后每提升5级，分别再获得1项满足条件的魔导专长。

### 4级：奥术论题
你可以通过10分钟的专注，对1道固有法术应用以下改动之一，持续至完成长休。在此期间，你必须将法术书摆放在随时能够阅览的位置。 \
你最多只能同时维持对1道法术的改动。此后每提升5级，分别可以多维持对1道法术改动。
- **成分**：将该法术的1项成分替换为姿势。如果替换消耗有价值耗材的材料成分，每次施展该法术你都会失去每环阶1D6点HP。 \
  材料价值每有1袋金币 (向上取整)，还会额外减损1点力量。解除属性减损的效应，每次只会解除1点该力量减损。
- **伤害类型**：6级起，向DM说明你的理论，若DM认为可行，则替换该法术包含的伤害类型，并对法术效果进行适当的调整。
- **专注**：8级起，若该法术需要专注，受到伤害不会打断其专注。
- **仪式**：10级起，若该法术的施法时间到达10分钟，且持续时间不为立即，则给予其仪式标签。重复以仪式施展该法术时，上次仪式施展的该法术会提前结束。
- **目标**：12级起，若该法术的施法距离不为自身且影响一个或更多生物，你可以为其设定一套不超过25个字的生效条件，例如“只影响你的敌人”、“只影响身高不超过1米的秃顶男性矮人”。

### 5级：即时调整
以1个附赠动作，你可以交换2个相邻的法术列队的序列号。你能够使用1次本特性，10级与15级起分别再增加1次使用次数。完成长休会恢复所有使用次数，完成短休会恢复1次使用次数。

### 9级：创造法术
当你通过【奥术论题】特性为1道固有法术应用改动时，可以通过1小时的整备并消耗［该法术环阶］箱金币，将改动后的法术作为一道新法术加入藏书列表，新法术的名称由你决定。

### 13级：法术精通
准备固有法术时，你可以额外各准备1道不超过［最大环阶-5］环与［最大环阶-6］环的法术作为固有法术。 \
以［最大环阶-5］环/［最大环阶-6］环施展这些法术时，你可以无需消耗法术位，此时该法术的持续时间若到达10分钟，则会变为「谐振」。以此施展的法术无法通过任何形式恢复HP。重复以此施展相同法术时，上次以此施展的该法术会提前结束。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

### 20级：传奇法术
通过8小时的整备并消耗10箱金币，你可以创造一道经过DM许可的全新法术，并将其加入藏书列表。
