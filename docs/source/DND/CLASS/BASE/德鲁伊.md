# 德鲁伊

## 德鲁伊核心特质
- **主要属性**：感知
- **生命值骰**：D6
- **职业熟练**：低阶APR、高阶SPR
- **豁免倾向**：反射_弱、强韧_弱、意志_强
- **武器熟练**：简易武器
- **护甲受训**：轻甲

## 成为德鲁伊

### 作为1级角色
- 获得「德鲁伊核心特质」里的所有能力
- 获得德鲁伊的1级特性

### 作为兼职角色
- 获得「德鲁伊核心特质」里的以下能力：生命值骰，职业熟练，豁免倾向，轻甲的护甲受训
- 获得德鲁伊的1级特性

### 选择子职业
你可以选择1种满足条件的德鲁伊子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：施法
你通过研究自然的神秘伟力学会了如何施展法术。
- **魔法根源**：本职业的魔法根源为**原能**。
- **关键属性**：本职业的施法关键属性为**感知**。
- **准备法术**：你可以从已知法术中准备不超过最大环阶的法术，它们会成为德鲁伊的本职法术。 \
  「德鲁伊特性」表中的**戏法**一列显示了你每职业等级能够准备的0环法术总数，这也被称为戏法。施展戏法不需要消耗法术位。 \
  「德鲁伊特性」表中的**术法**一列显示了你每职业等级能够准备的1环及以上法术总数，这也被称为术法。
- **调整法术**：每次完成长休时，你都可以重新准备任意道以此准备的法术。
- **术法转换**：完成短休时，你可以重新准备最多［⌈职业等级÷2⌉］道本职术法。

### 1级：德鲁伊语
你学会了德鲁伊语，一门德鲁伊之间的秘密语言。在学会这门古老语言的同时，你也解锁了和动物交谈的魔法：你始终准备着【动物交谈】法术。 \
你可以使用德鲁伊语来传递隐藏的信息。你和其他知晓这门语言的对象能够自动辨认出信息。其他人需要进行基于调查的察觉检定对抗你职业DC，检定成功才能意识到信息的存在，但不借助魔法则无法解读。

### 1级：原初职能
你将自己投身于所选择的以下一项神圣的角色之中：
- **卫士**：你获得军用武器熟练与中甲、重甲和盾牌受训。将德鲁伊生命骰尺寸提升至D8、APR提升至中阶、强韧倾向提升至强。如果你拥有德鲁伊之外的职业等级，则不会因此获得重甲受训。
- **术师**：你获得1项技能与草药工具的熟练。若你已有草药工具的熟练，则自选1项工具获得熟练。你可以从已知法术中额外准备1道戏法。 \
  2级起，你获得【万事通】特性，如同吟游诗人的同名特性。若从多个职业获得博学，将德鲁伊等级加上其他职业等级决定博学的效果。

### 2级：魔法导师
你获得1项满足条件的魔导专长。此后每提升5级，分别再获得1项满足条件的魔导专长。 \
每回合1次，你可以消耗1点技力代替移动动作激活以此获得的塑法专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

### 2级：自然纽带
从以下特性中选择1项获得。
- **动物伙伴**：你获得一头幼年动物伙伴的效忠，它会在冒险中陪同你一起旅行并尽其所能遵从你的任何简单指令。
- **荒野魔宠**：你获得【寻获魔宠】与【强化魔宠】作为奖励专长，并忽略其先决条件。你可以为魔宠选择【自然体】魔宠能力，且不计入你能够选择的能力总数中。 \
  此外，你可以在自己的回合命令魔宠使用其反应发动一次攻击，这无需动作，但你必须有能力下达命令。
- **领域法术**：选择1项牧师领域，将其领域法术加入你的已知法术列表。 \
  你可以消耗2点技力，以［⌈职业等级÷2⌉］环施展1道领域法术，并且无需消耗法术位。

### 4级：自然抗性
你对抗精类生物施展的法术和其他魔法效应时进行的豁免检定具有优势。 \
此优势也适用于对抗利用植物或目标为植物的法术和其他魔法效应时进行的豁免检定。

### 5级：自然复苏
你获得以下增益，使用这些增益无需动作。
1. 你随时可以消耗3点MP换取1点自然技力。自然技力只能通过德鲁伊职业特性使用。
2. 你可以消耗2点技力创造1枚最高［⌈职业等级÷4⌉］环的法术位，持续至你完成长休。此法术位只能用于施展本职法术。 \
   本增益在2次长休之间最多只能使用1次。20级起，增加1次使用次数，并在完成短休时恢复1次使用次数。

### 6级：自然视域
你可以消耗1点技力，无视12格内的灰、烟、雾以及类似环境造成的任何遮蔽，持续时间如同「谐振」。 \
10级起，本特性激活期间，你还可以看透相同距离内的叶片、藤条、灌木以及类似环境，如同它们是透明的一般。

### 7级：元素之怒
元素之力在你的身体里流淌。你从以下特性中选择其一获得。
- **强力施法**：当你施展的本职戏法造成基础伤害时，你可以对目标额外造成［SKM］点相同类型的伤害。
- **原力蛮击**：当你发动的武器攻击命中时，可以对目标额外造成1D8点寒冷、火焰、闪电或雷鸣伤害 (命中时选择)。一名生物每回合最多承受1次此伤害。

### 8级：毒素免疫
你免疫毒素伤害与中毒状态。

### 14级：千面万相
你可以随意施展【易容术】法术，且无需消耗法术位。

### 15级：元素狂怒
你选择的元素之怒特性变得愈发强大，如下所述。
- **强力施法**：当你施展一道施法距离至少为2格的本职戏法时，可以将施法距离提升60格。
- **原力蛮击**：原力蛮击的额外伤害提升至2D8。

### 18级：自然之心
投掷先攻时，你会获得1点自然技力，持续至下次投掷先攻。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

### 20级：不老身躯
你行使的原力魔法让你的衰老速度减缓。每经过10年的岁月，你的身体就仿佛只过了1年。你免疫魔法老化效应。


## 子职业

### 月亮结社

#### 3级：结社形态
*替换同名特性* \
你可以使用附赠动作消耗1点技力，变形为结社形态。你拥有5种结社形态，每种形态的外观由你首次变形时决定。 \
你能一直保持在该形态，直到你再次使用本特性变形、失去所有内在障壁、或死亡。你也可以使用附赠动作提前结束变形。 \
结社形态使用与荒野变形野兽形态类似的规则，但体型与速度之外的游戏数据不会被替换。你可以将原本作用于荒野变形野兽形态的增益赋予结社形态。
- **内在障壁**：变形为结社形态时，你会获得职业等级5倍的内在障壁。
- **月光愈合**：结社形态期间，你可以使用附赠动作消耗X点MP，将自身的内在障壁增加［5X］点，最高提升至［职业等级×5］点。
- **自然护甲**：结社形态期间，若未着装护甲，你可以使用［10+DEX+SKM］作为基础AC；若着装护甲，则可以使用SKM代替DEX加入AC。
- **无羁野性**：结社形态期间，你无法持用武器・盾牌。但在使用徒手打击时，将德鲁伊APR提升1阶。还可以使用施法属性作为徒手打击的攻击属性，并且可以改为造成穿刺或挥砍伤害。
- **兽形施法**：6级起，你能够在结社形态下施展法术，若为此消耗法术位，其环阶不能超过［职业等级÷2-2］(最高5) 环。
- **凶暴形态**：10级起，你的结社形态会获得强化。此外，结社形态下的徒手打击伤害骰提升1级。

  当你受到钝击、穿刺或挥砍伤害时，可以在伤害结算前使用反应消耗2点MP获得这三种伤害的抗性，持续至你的下个回合开始。
