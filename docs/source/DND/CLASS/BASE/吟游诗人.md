# 吟游诗人

## 吟游诗人核心特质
- **主要属性**：魅力
- **生命值骰**：D6
- **职业熟练**：低阶APR、高阶SPR
- **豁免倾向**：反射_强、强韧_弱、意志_强
- **武器熟练**：简易武器
- **护甲受训**：轻甲
- **额外熟练**：1项技能、3项艺人工具

## 成为吟游诗人

### 作为1级角色
- 获得「吟游诗人核心特质」里的所有能力
- 获得吟游诗人的1级特性

### 作为兼职角色
- 获得「吟游诗人核心特质」里的以下能力：生命值骰，职业熟练，豁免倾向，轻甲的护甲受训，1项技能熟练，1项艺人工具熟练
- 获得吟游诗人的1级特性

### 选择子职业
你可以选择1种满足条件的吟游诗人子职。此后你将获得所选子职的所有能力，只要其所需等级不超过你的职业等级。 \
若所选子职的先决条件失效，你将暂时失去子职提供的所有能力，直到条件被重新满足。

### 1级：施法
你从吟游艺术中学会了如何施展法术。
- **魔法根源**：本职业的魔法根源为**异能**与你选择的1种：奥术、神术、原能。
- **关键属性**：本职业的施法关键属性为**魅力**。
- **特殊法器**：你可以使用艺人工具作为本职法术的施法法器，并且可以通过演奏艺人工具来满足本职法术的姿势成分。
- **准备法术**：你可以从已知法术中准备不超过最大环阶的法术，它们会成为吟游诗人的本职法术。 \
  「吟游诗人特性」表中的**戏法**一列显示了你每职业等级能够准备的0环法术总数，这也被称为戏法。施展戏法不需要消耗法术位。 \
  「吟游诗人特性」表中的**术法**一列显示了你每职业等级能够准备的1环及以上法术总数，这也被称为术法。
- **调整法术**：每次提升职业等级时，你都可以调整以此准备的法术：重新准备1道戏法，再重新准备1道术法。

### 1级：组曲
你可以将表演融合到法术当中来创造名为组曲的独特法术。
- **准备组曲**：你可以将满足条件的组曲视为吟游诗人的已知法术进行准备。 \
  4级起，以及之后每提升4级，你分别可以选择1首满足条件的组曲始终准备。每次提升职业等级时，你可以重新选择1首组曲。
- **组曲成分**：所有组曲都拥有「S、F (艺人工具)」成分，无法免除。你必须通过演奏艺人工具满足这些成分。

#### 灵感
「吟游诗人特性」表中的**灵感骰**一列显示了你每职业等级的灵感骰尺寸。 \
效果表述中的「**灵感_N**」代表投掷N枚灵感骰并将骰值相加。 \
若从多个职业获得灵感，将吟游诗人等级加上其他职业等级决定灵感骰尺寸。

### 2级：魔法导师
你获得1项满足条件的魔导专长。 \
每次提升职业等级时，你都可以重新选择1项以此获得的专长。

	若从多个职业获得博学，将吟游诗人等级加上其他职业等级决定本特性的效果。

### 5级：激励之源
完成短休时，你会获得1点AP，持续至下次完成短休・长休。 \
此外，你随时可以消耗3点MP换取1点AP，这无需动作。 \
通过本特性获得的AP只能用于施展组曲。

### 7级：音律塑法
你获得【旋律法术】作为奖励专长。12级起，获得【和声组曲】；17级起，获得【对唱呼应】。 \
每回合1次，你可以消耗1AP代替移动动作激活以此获得的塑法专长。你可以在此时消耗那些通常只能用于施展组曲的AP。

### 9级：专精
选择2项已熟练的技能获得专精。

### 18级：先发激励
投掷先攻时，你会获得1点AP，持续至下次投掷先攻。此AP只能用于施展组曲。

### 19级：传奇恩惠
你获得1项传奇恩惠专长，或其他满足条件的专长。

### 20级：瞬念组曲
若你未陷入失能状态，作为投掷先攻的一部分，可以施展1道施法时间为「主要」或「次要」的组曲。


## 组曲

### 关键术语
- **强度**：组曲拥有［⌈职业等级÷2⌉］级强度。在对抗【解除魔法】以及其他类似效应时，使用强度作为组曲的法术环阶。
- **塑法**：拥有「塑法」标签的组曲会如同塑法专长般运作。

### 组曲条目
- **级别**：你能够准备［⌈职业等级÷2⌉］级以内的组曲。组曲视为戏法，施法时无需消耗法术位。
- **学派**：所有组曲都会散发魔法学派灵光，也会被侦测出属于组曲。
- **先决**：你只能准备满足先决条件的组曲。若条件失效，你将暂时失去该组曲，直到条件被重新满足。
- **感官**：此处列出了组曲的感官成分。无法通过相应感官感受组曲的生物不会受其影响。
- **施法时间**：此处列出了施展组曲所需使用的动作。 \
  如果记述为「主要」，则需要执行【魔法】动作施展；7级起，可以使用移动动作施展；13级起，可以使用附赠动作施展。 \
  如果记述为「次要」，则需要使用附赠动作施展。 \
  如果记述为「响应」，则会作为其他动作的一部分施展，不能在无法执行反应时施展。
- **持续时间**：施展持续时间到达1轮的组曲时，你需要为其维持专注。当你再次施展这样的组曲时，上一首组曲的专注会被打断。
