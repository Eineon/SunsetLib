# 专长

## 关键术语
- **典范**：你只能在每个4的倍数等级习得1项典范专长。如果在典范专长的位置习得战斗・魔导・通用专长，你可以选择1项属性使其初始值+1，最高20。
- **聚能**：部分专长会为你提供聚能点。完成长休时，你会恢复所有已消耗的聚能点，并且可以将聚能点分配在聚能专长上，或是重新分配聚能点。 \
  「**聚能_N**」后记述的效果会在为该专长分配的聚能点到达N点后激活。即使聚能点被消耗，也依然能够维持激活。已分配的聚能点不能被其他专长消耗。
- **塑法**：拥有塑法标签的专长能改变法术的属性。你可以在施展符合要求的法术时激活已习得的塑法专长。
- **导能**：拥有导能标签的专长能改变神力源泉的作用方式。你可以在通过神力源泉施展法术时激活已习得的导能专长。

## 基础调整
- 战斗风格专长更名为战斗专长。
- 背景不再提供起源专长。你会在1级时习得1项满足条件的起源专长。
- 每当人物等级提升至2的倍数时，你都会习得1项满足条件的专长。如果用于习得起源专长，你可以选择1项属性使其初始值+1，最高20。
- 除传奇恩惠专长之外，专长不再提供属性值提升。对于依赖属性值提升的效果，你可以从原本的属性值提升中选择1项属性决定效果。
- 除起源专长之外，原本不会提供属性值提升的专长分类变为「典范」。
- 原本的通用专长先决条件中对等级的要求减半。
- 原本的专长先决条件中的「战斗风格特性」替换为「APR2+」。
- 原本的专长先决条件中的「施法或契约魔法特性」替换为「SPR2+」。
- 你可以习得不满足APR条件的专长，但其只会在APR满足条件时生效。
- 已习得的专长的先决条件失效期间，你将暂时失去该专长，直到条件被重新满足。
- 原本的专长对移动速度的提升视为内在加值。
- 移除专长：投掷武器战斗、属性值提升、强健身心、中甲大师、重甲大师、巨武器大师、双持客、粉碎者、穿刺者、劈砍者、技艺专家、超魔法导师。

## 专长列表

### 个别调整
使用下列专长替换同名专长。

#### 起源专长

##### 警戒
*起源* \
你获得以下增益：
- **惊奇先攻**：你的先攻检定获得1点支援。
- **先攻互换**：当你掷完先攻，你可以立即与处于同一场战斗中的一名自愿的盟友交换先攻。如果你或那个盟友正处于失能状态，则不能进行交换。

##### 盲斗
*起源* \
你拥有察觉技能的熟练，并获得2格盲视。6级起，你会得知6格内对你不具全身掩护的生物占据的方格位置，但不会获知具体位置。 \
10级起，距离你6格内的不可见生物对你发动的攻击检定不会因此获得优势。14级起，移除此距离限制。

##### 凶蛮打手
*起源* \
你获得以下增益：
- **冒进打击**：激活**攻击姿态**时，你可以不承受攻击检定减值，改为使AC与反射豁免承受等量的减值，但也无法在此期间进行反击。
- **稳妥打击**：为武器攻击投掷伤害前，你可以将每枚基础伤害骰改为［(尺寸+1)÷2］点伤害，最后将小数向上取整。

##### 熟习
*起源* \
选择合计2项技能与工具获得熟练。8级起，再选择1项技能或工具获得熟练。 \
你可以将其中1项熟练改为获得1项已熟练技能的专精。

##### 酒馆斗殴者
*起源* \
你获得以下增益：
- **强化徒手打击**：你拥有徒手打击的熟练，并且其伤害骰级别获得1点内在加值。
- **临时武器专家**：你拥有临时武器的熟练，并且其伤害骰级别获得1点内在加值。
- **体术训练**：每回合1次，当你在自己的回合使用徒手・临时武器命中一名生物时，可以使用相同的攻击判定值对目标发动1次**推撞**或**擒拿**。

##### 健壮
*起源* \
你获得以下增益：
- **HP提升**：你的最大HP增加［人物等级×3］(最低10) 点。
- **坚忍**：你对抗自然环境影响的强韧豁免具有优势。
- **死硬**：你的死亡阈值提高1点。


#### 战斗专长

##### 箭术
*战斗* \
**先决**：APR2+，敏捷13+ \
当你持用1把远程武器且没有持用其他武器或盾牌时，你获得以下增益。
- **穿透攻击**：该武器的伤害骰级别在激活**攻击姿态**期间获得2点内在加值。
- **伤害重掷**：每轮你都可以重掷最多［DEX］(最低1) 枚该武器的基础伤害骰，并使用新的结果。

##### 防御
*战斗* \
**先决**：APR2+ \
若基础AC中包含的属性调整值不超过1项，你的AC获得2点内在加值。 \
作为【回避】动作的一部分，或是使用附赠动作，你可以将因此获得的内在加值添加在反射豁免上，持续至你的下个回合开始。

##### 对决
*战斗* \
**先决**：APR2+ \
当你单手持用一把近战武器且没有持用其他武器时，你的先攻检定获得1点支援，加入该武器伤害中的AKM (若有) 额外增加2点。

##### 巨武器战斗
*战斗* \
**先决**：APR2+，力量13+ \
若共持拥有「双手」或「多用」词条的武器，你获得以下增益。
- **猛力攻击**：该武器的伤害骰级别在激活**攻击姿态**期间获得2点内在加值。
- **伤害重掷**：每轮你都可以重掷最多［STR］(最低1) 枚该武器的基础伤害骰，并使用新的结果。

##### 拦截
*战斗* \
**先决**：APR2+ \
当你可见的生物发动攻击命中你的威胁范围内的生物时，你可以消耗1次借机攻击，使此次攻击造成的伤害减轻［1PD+APR÷2］点。 \
如果伤害被减轻至0，你可以对攻击者发动借机攻击。

##### 守护
*战斗* \
**先决**：APR2+ \
当你可见的生物对你的威胁范围内除你以外的生物发动攻击时，若你已持用盾牌，则可以消耗1次借机攻击，使触发本专长的攻击承受劣势。 \
只要被攻击生物仍在你的威胁范围内，其他所有对该生物发动的也具有劣势，持续至你的下个回合开始。

##### 双武器战斗
*战斗* \
**先决**：APR2+ \
进行双持攻击时，你可以将AKM加入本回合首次发动的副手攻击命中后的伤害检定，不超过正常能够加入的AKM。 \
如果你可以发动多次副手攻击，则会为其他副手攻击命中后的伤害检定加入比正常低2点的AKM (最低0)。

##### 徒手战斗
*战斗* \
**先决**：APR2+ \
你获得以下增益：
- **精通徒手打击**：你的徒手打击伤害骰提升至3级。如果发动徒手打击时没有持用其他武器或盾牌，则改为提升至4级。
- **徒手截击**：在你的威胁范围内，发动不具熟练的徒手打击的生物，将会引发你的借机攻击。
- **精通擒拿**：你发动擒拿时的攻击检定获得1点支援，对你发动擒拿时的攻击检定受到1点妨碍。 \
  在你的威胁范围内，不具**精通擒拿**特性的生物发动的擒拿，将会引发你的借机攻击。
- **精通紧勒**：你可以使用附赠动作代替移动动作进行紧勒，并且可以将AKM加入紧勒造成的伤害中。

##### 神射手
*战斗* \
**先决**：APR2+，敏捷13+ \
你获得以下增益：
- **抵近射击**：你不会因被威胁而在远程武器攻击上承受劣势。
- **绕过掩体**：以一个移动动作，你可以使本回合发动的远程武器攻击无视目标AC上的状态加值。
- **箭雨**：APR6级起，每回合1次，你可以在进行全力攻击时消耗其中1次攻击，随后指定武器射程内可见的一点，对源自该点的1格光环区域内任意数量的生物分别发动1次远程攻击，这些攻击检定受到2点减值。所消耗的攻击因全力攻击而承受的减值，也会作用在这些攻击检定上。此后每提升4级APR，就增加1格光环范围。

##### 哨兵
*战斗* \
**先决**：APR2+，力量或敏捷13+ \
你获得以下增益：
- **守护者**：威胁范围内你可见的生物执行【撤离】动作，或是对你以外的生物发动攻击，将会引发你的借机攻击。
- **阻拦**：因移动而触发你的借机攻击的生物被命中时，即便未形成重击也会打断移动。


#### 通用专长

##### 飙速跑者
*通用* \
**先决**：等级2+，敏捷或体质13+ \
你获得以下增益：
- **速度提升**：你的步行速度获得2点内在加值。
- **险地疾行**：当你在自己的回合执行【疾走】动作时，在这个回合余下的时间里，困难地形不再会额外消耗你的移动力。
- **灵活移动**：你因移动而触发借机攻击时，其攻击检定具有劣势。


#### 魔导专长

##### 法术射手
*魔导* \
**先决**：SPR2+ \
你获得以下增益：
- **抵近施法**：你不会因被威胁而在远程法术攻击上承受劣势。
- **绕过掩体**：以一个移动动作，你可以使本回合通过法术发动的远程攻击无视目标AC上的状态加值。
- **法术轰炸**：激活**攻击姿态**时，你可以令法术攻击检定也承受其减值。 \
  在此期间，只要你的法术攻击判定值比目标值低不超过4点，就可以选择源自攻击目标的1格光环区域内任意数量的生物作为第二目标。 \
  迫使所有第二目标进行1次对抗此法术DC的反射豁免，失败则受到［1PD］点相同类型的轰炸伤害。 \
  此后每提升6级APR，就增加1格光环范围，并增加［1PD］点轰炸伤害。 \
  你无法在进行反击时造成轰炸伤害，也无法同时造成轰炸伤害与擦掠伤害。


#### 典范专长

##### 战地施法者
*典范* \
**先决**：SPR2+ \
你获得以下增益： 
- **专注**：你为法术进行的专注豁免具有优势。
- **成分**：即使你的主手已持用武器・盾牌，也依然能够满足法术的姿势・材料成分。
- **法术反击**：你可以消耗通过反击发动的武器攻击代替【魔法】动作，发动1次法术攻击 (即便该动作通常能够发动多次攻击)。 \
  在决定法术反击能否命中时，将远程法术攻击的常规射程视为施法距离与6格中的较低者。 \
  该反击不会影响你与攻击者之外的对象。若为该反击消耗法术位，其环阶不能超过［最大环阶-2］(最低1) 环。
- **响应法术**：当一个生物因移动引发你的借机攻击时，可以改为消耗1次借机攻击代替【魔法】动作，对此生物发起1次敌对行为。 \
  该行为不会影响你与此生物之外的对象。若为该行为消耗法术位，其环阶不能超过［最大环阶-2］(最低1) 环。


### 新增专长

#### 起源专长

##### 属性强化
*起源* \
你获得1次**属性提升**。 \
**复选**：你可以多次选取本专长。

##### 夜视
*起源* \
你获得以下增益：
- **黑夜视觉**：你获得昏暗视觉。若已有则获得黑暗视觉。
- **敏锐感官**：你获得调查与察觉之一的熟练。

##### 即兴发挥
*起源* \
进行不具熟练的属性检定时，你依然会获得［人物等级÷2］点RB (在**熟练**一列标记为“☆”)。

##### 技能专攻
*起源* \
选择1项技能，使用所选技能的属性检定获得3点加值。10级起，提升至6点加值。 \
**复选**：你可以多次选取本专长，每次必须选择不同技能。

##### 豁免专攻
*起源* \
选择1种基础豁免或专注豁免，所选豁免检定获得2点加值。 \
**复选**：你可以多次选取本专长，每次必须选择不同豁免。

##### 宠物傍身
*起源* \
你拥有一只宠物，选择的一种微型动物，比如猫、鸟或老鼠。 \
宠物不会使用所选动物的数据卡，根据以下效应决定其游戏数据。
- **等级**：宠物的等级与你的人物等级相同。
- **谱系**：宠物的谱系为「凡」，并拥有【动物】原型。
- **生命值**：宠物的最大HP为［等级×5］点，并拥有等于其等级的D8生命骰。
- **护甲等级**：宠物的基础AC与你相同，并将PB降低2点加入其中，再获得［等级÷2］点RB。
- **检定修正**：宠物没有自己的属性，使用你的属性调整值。它的D20检定修正为［关键属性调整值+PB］点，再获得［等级÷2］点RB。宠物拥有特技、隐匿、察觉技能的专精。
- **感官**：宠物拥有昏暗视觉，并可通过宠物能力获得额外感官。
- **语言**：宠物理解你的语言但不会说。
- **速度**：宠物拥有6格步行速度。你可以改为选择一只水栖宠物，它只能在水中呼吸，没有步行速度，拥有6格游泳速度。
- **宠物能力**：获得宠物时，你从以下能力中选取两项。若你的宠物是生来就具有其中某项能力的动物（例如，猫头鹰本身就有飞行速度），则必须选取该能力。如果一种动物生来拥有的宠物能力数量超过上限，则不能成为你的宠物。
  + **两栖**：宠物同时拥有步行与游泳速度，都等于它的最高步行或游泳速度。宠物可以在空气和水中呼吸。
  + **掘地**：宠物获得1格掘穴速度。宠物可以在身后留下微型地道。
  + **蛛行**：宠物获得6格攀爬速度。宠物可以在难以攀爬的表面上攀爬，包括沿着天花板移动，且无需为此进行属性检定。
  + **飞掠**：宠物获得6格飞行速度。宠物飞行离开敌人的威胁范围时不会引发借机攻击。
  + **盲视**：宠物获得2格盲视。
  + **黑暗视觉**：宠物获得黑暗视觉。
  + **回声定位**：宠物获得4格盲视，无法在耳聋状态下使用。
  + **快速移动**：选择宠物的1种移动速度给予2点内在加值。
  + **心灵手巧**：宠物可以用最多两条肢体像手一样发起行为。
  + **敏锐嗅觉**：宠物进行依赖嗅觉的属性检定具有优势。
  + **健壮**：宠物的最大HP每等级增加3点。在投掷HD为其恢复HP时，可以取满HD。
- **战斗**：宠物是你和你盟友的盟友。战斗时，它在你的回合中行动，并遵从于你的口头指令 (无需动作)。如果你不发令，宠物将执行【回避】动作并远离危险。 \
  宠物没有也无法使用武器，因此无法发动攻击，也无法制造威胁范围。
- **重训**：作为一项休整期活动，你可以训练一只新的宠物，之前拥有的宠物将被释放，所需时间由DM决定。当你在拥有本专长时获得奖励本专长的特性，可以立即进行重训。

##### 灵敏移动
*起源* \
你获得以下增益：
- **腾跃**：每轮1次，当你使用移动动作时，如果需要消耗的移动力超过1格，则减少1格。
- **猛扑**：你可以在进行全力攻击时不消耗移动动作，但在决定其效果时，视APR比实际低6级。

##### 武器娴熟
*起源* \

- **致命灵巧**：执行【攻击】动作・发动借机攻击时，如果发动攻击属性为敏捷的攻击，其伤害骰级别获得1点内在加值。副手攻击不会获得本增益。

##### 战斗训练
*起源* \
你获得以下增益：
- **武器熟练**：选择1种军用武器获得熟练。
- **战技**：你习得1项满足条件的战技，并获得1点技力。将［人物等级÷3］(最低1) 作为战士等级来决定战技效果与卓越骰尺寸。
- **多元训练**：如果你拥有战士等级，则将［战士之外的职业等级÷3］加在战士等级之上，来决定所有的战技效果与卓越骰尺寸。


#### 战斗专长

##### 战斗专攻
*战斗* \
**先决**：APR2+ \
选择1种已熟练的武器，或是1道法术，你使用所选项目进行的攻击检定获得2点内在加值。 \
如果选择徒手・简易武器，则其伤害骰级别获得1点内在加值。 \

**复选**：你可以多次选取本专长，每次必须选择不同项目。

##### 披甲战斗
*战斗* \

*通用* \
**先决**：等级12+，岿然不动或英勇无畏 \
【岿然不动】的效果改为「只要你未失能，在受到强韧豁免成功仍能造成部分效应的效果时，豁免成功将不受任何效应，豁免失败则仅承受一半伤害 (若有)」。 \

#### 典范专长

##### 歼灭者
*典范* \
**先决**：力量或敏捷15+ \
你获得以下增益：
- **致命绝杀**：你的武器攻击造成伤害后，若目标处于垂危，你可以消耗1次借机攻击，迫使目标进行1次对抗攻击DC的反射豁免。 \
  若目标豁免失败，作为本次攻击的一部分，你可以处决目标，并且不会引发借机攻击。
- **百般武艺**：以一个附赠动作，你可以改变武器的握法或弹药。若其基础伤害属于下列之一，则改为另一种：钝击、穿刺、挥砍。APR6级起，你可以作为任意动作的一部分来使用本增益。
- **武器专精**：每回合1次，你可以在武器攻击命中时，根据其基础伤害的类型造成以下效果。
  + **钝击**：若目标的体型不比你大超过一级，则可以将其推动1格。若形成重击，直到你的下个回合开始前，任何以该生物为目标的攻击检定都会获得优势。 
  + **穿刺**：目标的下个回合结束时，还会受到［1WD］点穿刺伤害。若形成重击，重击骰枚数获得2点内在加值，并将本效果造成的伤害提升至［2W］。
  + **挥砍**：目标的所有速度受到2点减值，持续至你的下个回合开始。若形成重击，目标进行的所有攻击检定都会承受劣势，持续至你的下个回合开始。

##### 战斗反射
*典范* \
**先决**：力量13+，反射RB4+ \
你获得以下增益：
- **适时反击**：你因进行反击而承受的减值最高为［7-APR÷4］点。
- **刹那**：你发动武器攻击时，直到攻击结束为止，反射RB低于你的生物无法执行反应，因此也无法发动借机攻击・反击。
- **必中**：你发动武器攻击时，可以消耗1次借机攻击，将你掷出的1枚低于8的D20骰值提升至8。

##### 驭法攻击
*典范* \
**先决**：SPR4+ \
每回合1次，你在自己的回合执行【魔法】动作时，可以使用附赠动作额外以单手发动1次武器攻击。 \
这次额外攻击，以及通过【魔法】动作进行的攻击检定及豁免DC，都会承受2点减值。 \
你无法将AKM加入这次武器攻击的伤害检定，除非该调整值为负值。 \
**特殊**：使用激活「迅击」精通词条的武器发动这次额外攻击时，你可以将本次攻击作为【魔法】动作的一部分，无需使用附赠动作。
