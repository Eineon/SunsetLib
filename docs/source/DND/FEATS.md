# 专长

## 关键术语
- **典范**：你只能在每个4的倍数等级习得1项典范专长。如果在典范专长的位置习得通用专长，你可以选择1项属性初始值+1。
- **聚能**：部分专长会为你提供聚能点。完成长休时，你会恢复所有已消耗的聚能点，并且可以将聚能点分配在聚能专长上，或是重新分配聚能点。 \
  「**聚能_N**」后记述的效果会在为该专长分配的聚能点到达N点后激活。即使聚能点被消耗，也依然能够维持激活。一个专长可以消耗未分配给它的聚能点。
- **塑法**：拥有塑法标签的专长能够改变法术的属性。你可以响应特定事件激活已习得的塑法专长，无法在【sign:失能】期间激活。
- **导能**：拥有导能标签的专长能够改变奇迹的属性。你可以响应特定事件激活已习得的导能专长，无法在【sign:失能】期间激活。

## 基础调整
- 战斗风格专长更名为战斗专长。
- 背景不再提供起源专长。你会在1级时习得1项满足条件的起源专长。
- 每当人物等级提升至2的倍数时，你都会习得1项满足条件的专长。如果用于习得起源专长，你可以选择1项属性初始值+1。
- 除传奇恩惠专长之外，专长不再提供属性值提升。对于依赖属性值提升的效果，你可以从原本的属性值提升中选择1项属性决定效果。
- 除起源专长之外，原本不会提供属性值提升的专长分类变为「典范」。
- 原本的通用专长先决条件中对等级的要求减半。
- 原本的专长先决条件中的「战斗风格特性」替换为「WPR2+」。
- 原本的专长先决条件中的「施法或契约魔法特性」替换为「SPR2+」。
- 你可以习得不满足WPR条件的专长，但其只会在WPR满足条件时生效。
- 已习得的专长的先决条件失效期间，你将暂时失去该专长，直到条件被重新满足。
- 原本的专长对移动速度的提升视为内在加值。
- 移除专长：箭术、对决、巨武器战斗、投掷武器战斗、属性值提升、强健身心、中甲大师、重甲大师、巨武器大师、防御式决斗、双持客、粉碎者、穿刺者、劈砍者、技艺专家、超魔法导师。

## 专长列表

### 起源专长

#### 属性强化
*起源* \
你获得1次**属性提升**。 \
**复选**：你可以多次选取本专长。

#### 夜视
*起源* \
你获得以下增益：
- **黑夜视觉**：你获得昏暗视觉。若已有则获得黑暗视觉。
- **敏锐感官**：你获得察觉技能的熟练。若已有熟练，则自选1项技能获得熟练。

#### 即兴发挥
*起源* \
进行不具熟练的属性检定时，你依然会获得［人物等级÷2］点RB (在**熟练**一列标记为“☆”)。

#### 技能专攻
*起源* \
选择1项技能，使用所选技能的属性检定获得3点加值。10级起，提升至6点加值。 \
**复选**：你可以多次选取本专长，每次必须选择不同技能。

#### 豁免专攻
*起源* \
选择1种基础豁免，所选豁免检定获得2点加值。 \
**复选**：你可以多次选取本专长，每次必须选择不同豁免。

#### 灵敏移动
*起源* \
你获得以下增益：
- **闪避**：你因移动而触发借机攻击时，其攻击检定受到1点妨碍。
- **疾步**：作为一次至少2格的直线移动的一部分，你可以执行【cell:回避】。 \
  一回合内执行2次【cell:进行】，或通过**疾步**执行【cell:回避】的生物，能够在1轮内无视你以此获得的增益。

#### 武器娴熟
*起源* \
你获得以下增益：
- **优雅武艺**：拥有「轻型」词条的近战武器对你而言具有「灵巧」词条。
- **致命灵巧**：发动攻击属性为敏捷的武器攻击时，你可以在伤害检定中使用敏捷代替力量。 \
  只要你的敏捷到达15，便不会在力量不足15时因武器的「重型」词条承受妨碍。如果力量不足13，你依然无法使用重型武器。

#### 战斗训练
*起源* \
你获得以下增益：
- **武器熟练**：选择1种军用武器获得熟练。
- **战技**：你习得1项满足条件的战技，并获得1点技力。将［人物等级÷3］(最低1) 作为战士等级来决定战技效果与卓越骰尺寸。
- **多元训练**：如果你拥有战士等级，则将［战士之外的职业等级÷3］加在战士等级之上，来决定所有的战技效果与卓越骰尺寸。

#### 警戒
*起源* \
你获得以下增益：
- **惊奇先攻**：你用于决定先攻值的行动检定获得1点支援。
- **先攻互换**：决定先攻值后，若你未处于【sign:失能】，你可以立即与处于同一场战斗中，且同样未【sign:失能】的1名自愿的盟友交换先攻值。

#### 盲斗
*起源* \
你拥有察觉技能的熟练，再从以下特性选择一项获得。若已有察觉熟练，则自选1项技能获得熟练。
- **敏锐嗅觉**：你的嗅觉范围提升至6格，并获得2格基于嗅觉的盲视。
- **敏锐听觉**：你的听觉范围提升至12格，并获得4格基于听觉的盲视。

6级起，你会得知6格内对你不具全身掩护的生物占据的方格位置，但不会获知具体位置。 \
10级起，距离你6格内的不可见生物对你发动的攻击检定不会因此获得优势。14级起，移除此距离限制。

#### 魔法学徒
*起源* \
选择1种魔法根源。你获得以下增益：
- **本能戏法**：从所选根源中选择2道常见戏法作为本能法术。
- **本能术法**：从所选根源中选择1道1环常见法术作为本能法术，每天可以不消耗法术位施展此法术一次。
- **更换法术**：每次提升人物等级时，你都可以更换1道以此获得的本能法术。

**复选**：你可以多次选取本专长，每次必须选择不同的法术，或者不同根源的相同法术。

#### 凶蛮打手
*起源* \
你获得以下增益：
- **冒进打击**：激活**攻击姿态**时，你可以将攻击检定因此承受的减值转移至AC与反射豁免，但无法在此期间进行反击。
- **稳妥打击**：为武器攻击投掷伤害前，你可以将每枚基础伤害骰改为［(尺寸+1)÷2］点伤害，最后将小数向上取整。

#### 熟习
*起源* \
选择合计2项技能与工具获得熟练。8级起，再选择1项技能或工具获得熟练。 \
你可以将其中最多1项熟练改为获得1项已熟练技能或工具的专精。 \
**复选**：你可以多次选取本专长。

#### 酒馆斗殴者
*起源* \
你获得以下增益：
- **体术训练**：你拥有体术的熟练，并且其伤害档位获得1点内在加值。
- **临时武器专家**：你拥有临时武器的熟练，并且其伤害档位获得1点内在加值。
- **格斗专家**：每回合1次，当你在自己的回合使用体术・临时武器命中一名生物时，可以使用相同的攻击判定值对目标发动1次**推撞**或**擒拿**。

#### 健壮
*起源* \
你获得以下增益：
- **HP提升**：你的最大HP增加［人物等级×3］(最低10) 点。在投掷HD为你恢复HP时，可以取满HD。
- **坚忍**：你对抗自然环境影响的强韧豁免具有优势。
- **死硬**：你的死亡阈值获得1点加值。

**复选**：你可以多次选取本专长。

#### 宠物傍身
*起源* \
你拥有一只宠物，选择的一种微型的动物个体，比如猫、鸟或老鼠。 \
宠物不会使用所选动物的数据卡，根据以下规则决定其游戏数据。
- **等级**：宠物的等级与你的人物等级相同。
- **谱系**：宠物的谱系为「凡」，并拥有【动物】原型。
- **生命值**：宠物的最大HP为［等级×5］点，并拥有等于其等级的D8生命骰。
- **护甲等级**：宠物的AC与你相同，但不会受那些仅作用于你的临时修正影响。
- **检定修正**：宠物没有自己的属性，使用你的属性调整值。它的D20检定修正为［关键属性调整值+PB］点，再获得［等级÷2］点RB。宠物拥有特技、隐匿、察觉技能的专精。
- **感官**：宠物拥有昏暗视觉，并且可通过宠物能力获得额外感官。
- **语言**：宠物理解你的语言但不会说。
- **速度**：宠物拥有6格步行速度。你可以改为选择一只水栖宠物，它只能在水中呼吸，没有步行速度，拥有6格游泳速度。
- **宠物能力**：获得宠物时，为它选取2项【word:宠物能力】。若你的宠物生来具有其中某项能力 (例如，猫头鹰天生就有飞行速度)，则必须选取该能力。生来拥有的宠物能力数量超过2项的生物不能成为你的宠物。
- **战斗**：宠物是你和你盟友的盟友。战斗时，它在你的回合中行动，并遵从于你的口头指令 (无需动作)。如果你不发令，宠物将执行【cell:回避】并远离危险。 \
  宠物没有也无法使用武器，因此无法发动攻击，也无法制造威胁范围。
- **重训**：作为一项休整期活动，你可以训练一只新的宠物，之前拥有的宠物将被释放，所需时间由DM决定。当你在拥有本专长时获得奖励本专长的特性，可以立即进行重训。


### 通用专长

#### 飙速跑者
*通用* \
**先决**：等级2+，基本敏捷或体质13+ \
你获得以下增益：
- **速度提升**：你的步行速度获得2点内在加值。
- **险地疾行**：在同一回合执行2次【cell:行进】后，直到回合结束为止，你可以无视阻碍区域。
- **猛扑**：每回合1次，你可以使用标准动作进行全力攻击。此时在决定其效果时，视WPR比实际低6级。

#### 反射闪避
*通用* \
**先决**：等级2+ \
只要你未失能，在受到允许进行反射豁免以减半伤害的效应时，若豁免成功则不受伤害。

#### 精通反射闪避
*通用* \
**先决**：等级8+，反射闪避 \
使用【反射闪避】时，若你豁免失败则仅承受一半伤害。 \
12级起，【反射闪避】的效果改为「只要你未失能，在受到反射豁免成功仍能造成部分效应的效果时，豁免成功将不受任何效应，豁免失败则仅承受一半伤害 (若有)」。

#### 岿然不动
*通用* \
**先决**：等级2+ \
只要你未失能，在受到允许进行强韧豁免以减半伤害的效应时，若豁免成功则不受伤害。

#### 英勇无畏
*通用* \
**先决**：等级2+ \
只要你未失能，在受到允许进行意志豁免以减半伤害的效应时，若豁免成功则不受伤害。

#### 坚定防御
*通用* \
**先决**：等级12+，岿然不动或英勇无畏 \
【岿然不动】的效果改为「只要你未失能，在受到强韧豁免成功仍能造成部分效应的效果时，豁免成功将不受任何效应，豁免失败则仅承受一半伤害 (若有)」。 \
【英勇无畏】的效果改为「只要你未失能，在受到意志豁免成功仍能造成部分效应的效果时，豁免成功将不受任何效应，豁免失败则仅承受一半伤害 (若有)」。

#### 战斗专注
*通用 聚能* \
**先决**：等级2+ \
在战斗中，你可以使用全力动作恢复1点聚能点。每场战斗最多以此恢复3点聚能点。 \
每回合1次，你可以使用移动动作代替该全力动作。随后必须进行1次DC20的专注检定，失败则在3轮内无法再次使用本专长。

#### 额外聚能
*通用 聚能* \
**先决**：等级2+ \
你获得2点聚能点。6级与14级起，分别再获得1点。 \
**复选**：你可以多次选取本专长。

#### 决斗架势
*通用 战斗 架势* \
**先决**：WPR2+ \
当你单手持用一把武器且没有持用其他武器时，可以使用附赠动作进入**决斗架势**，持续时间如同「谐振」。在此期间，你获得以下增益。
- **决斗反应**：你的先攻检定获得1点支援。WPR8级起，提升至2点。
- **精确刺击**：使用该武器发动的攻击命中时，你可以将PB加在造成的伤害上。
- **决斗招架**：其他生物的近战攻击命中你时，你可以消耗1次借机攻击使你的AC获得专精，此举可能导致该攻击失手。此后直到本回合结束为止，你的AC在面对近战攻击时都将具有专精。 

#### 巨武架势
*通用 战斗 架势* \
**先决**：WPR2+，基本力量13+ \
当你共持拥有「双手」或「多用」词条的武器时，可以使用附赠动作进入**巨武架势**，持续时间如同「谐振」。在此期间，你获得以下增益。
- **猛力攻击**：你可以将［PB÷2］加在该武器的伤害档位上。
- **伤害重掷**：每轮你都可以重掷最多［STR］(最低1) 枚该武器的基础伤害骰，并使用新的结果。

#### 御射架势
*通用 战斗 架势* \
**先决**：WPR2+，基本敏捷13+ \
当你持用1把远程武器且没有持用其他武器或盾牌时，可以使用附赠动作进入**御射架势**，持续时间如同「谐振」。在此期间，你获得以下增益。
- **穿透射击**：使用该武器发动的攻击命中时，你可以选择源自攻击目标的［射程增量］格光环区域内1名你可见的生物作为第二目标，对目标造成［PB］点相同类型的伤害。
- **伤害重掷**：每轮你都可以重掷最多［DEX］(最低1) 枚该武器的基础伤害骰，并使用新的结果。

#### 战斗专攻
*通用 战斗* \
**先决**：WPR2+ \
选择1种已熟练的武器，或是1道法术，你使用所选项目进行的攻击检定获得1点支援。 \
如果选择徒手・简易武器，则其伤害档位获得1点内在加值。 \
WPR4级起，激活所选武器的精通词条。 \
**复选**：你可以多次选取本专长，每次必须选择不同项目。

#### 披甲战斗
*通用 战斗* \
**先决**：WPR2+ \
选择1种已受训的护甲，激活其精通词条。你还会在着装该护甲期间获得以下增益：
- **灵活着装**：你可以在着装该护甲时降低1点护甲值，增加1点敏捷上限；或是降低1点敏捷上限，增加1点护甲值。若该护甲的敏捷上限为0，则无法选择降低敏捷上限。
- **着甲移动**：若该护甲为中甲，移除其「嘈杂」词条 (若有)，并添加「柔韧」词条。
- **不落要塞**：若该护甲为重甲，你会在自己的回合开始时获得［2PD+STR］(最低1) 点状态韧性，持续至你的下个回合开始。

#### 精通战策
*通用 战斗* \
**先决**：WPR2+ \
选取本专长时，从以下特性中选择最多2项获得。
- **精通推撞**：你发动推撞时的攻击检定获得1点支援，对你发动推撞时的攻击检定受到1点妨碍。 \
  在你的威胁范围内，不具**精通推撞**特性的生物发动的推撞，将会引发你的借机攻击。
- **精通缴械**：你发动缴械时的攻击检定获得1点支援，对你发动缴械时的攻击检定受到1点妨碍。 \
  在你的威胁范围内，不具**精通缴械**特性的生物发动的缴械，将会引发你的借机攻击。
- **精通虚招**：你发动虚招时的属性检定获得1点支援，对你发动虚招时的属性检定受到1点妨碍。 \
  在你的威胁范围内，不具**精通虚招**特性的生物发动的虚招，将会引发你的借机攻击。 \
  WPR6级起，你可以使用附赠动作发动虚招。
- **精通兵胁**：你发动兵胁时的属性检定获得1点支援，对你发动兵胁时的属性检定受到1点妨碍。 \
  在你的威胁范围内，不具**精通兵胁**特性的生物发动的兵胁，将会引发你的借机攻击。 \
  WPR6级起，你可以使用附赠动作发动兵胁。

**复选**：你可以多次选取本专长，每次必须选择不同特性。

#### 即时备战
*通用 战斗* \
**先决**：WPR2+，基本敏捷13+ \
你可以将拔出武器作为先攻检定或任意动作的一部分，并且可以在拔出的回合将其重新收起。 \
即便没有拔出携带的武器，你也可以使用该武器制造威胁范围。 \
此外，你每轮可以额外发动［(当前敏捷值-9)÷4］(最低1) 次借机攻击。 \
WPR6级起，每回合1次，你可以在借机攻击命中时，再对相同目标发动1次攻击检定受到2点减值的借机攻击。

#### 坚毅
*通用 战斗* \

若对戏法激活本专长，可以使用移动动作免去聚能点消耗。

#### 强效法术
*通用 魔导 聚能 塑法* \
**先决**：SPR6+ \
你获得1点聚能点。SPR10级起再获得1点。 \
<**聚能_1**>：当你掷骰决定法术直接造成的伤害或治疗量时，可以消耗1点聚能点激活本专长，使本次掷骰获得［[法术环阶]D8］点加值。 \
若对戏法激活本专长，则会获得1D6点加值，并且可以使用附赠动作免去聚能点消耗。


### 典范专长

#### 歼灭者
*典范* \
**先决**：基本力量或敏捷15+ \
你获得以下增益：
- **粉碎防御**：你的武器攻击命中处于恐慌或失控状态的生物时，其对抗你使用武器发动的攻击具有破绽，持续至你的下个回合结束。
- **武器专精**：每回合1次，你可以在使用武器的攻击攻击命中时造成以下效果之一。
  1. 若目标的体型不比你大超过一级，则可以将其强制移动1格。若形成重击，直到你的下个回合开始前，任何以该生物为目标的攻击检定都会获得优势。 
  2. 目标的下个回合结束时，将受到［1WD］点武器伤害。若形成重击，目标的重击豁免受到1点妨碍，并将本效果造成的伤害提升至［2WD］。
  3. 目标的所有速度受到2点减值，持续至你的下个回合开始。若形成重击，目标进行的所有攻击检定都会承受劣势，持续至你的下个回合开始。

#### 战斗反射
*典范* \
**先决**：WPR4+，基本力量或敏捷13+，反射RB4+ \
你获得以下增益：
- **适时反击**：你因进行反击而在攻击检定上承受的减值最高为4点。WPR12级起，最高为2点。
- **刹那**：你发动武器攻击时，直到攻击结束为止，反射RB低于你的生物无法执行反应，因此也无法发动借机攻击・反击。
- **必中**：你发动武器攻击时，可以消耗1次借机攻击，将你在攻击检定中掷出的1枚低于8的D20骰值提升至8。WPR12级起，改为将低于10的骰值提升至10。

#### 跨界施法者
*典范* \
如果拥有至少2个满足以下条件的职业，在决定这些职业的最大环阶时，你的合计SPR获得2点加值。
- 在仅计算该职业最大环阶的情况下，能够以该职业的【施法】特性准备2环法术。

**复选**：你可以多次选取本专长，每次都会使上述效果所需的职业数量下限+1。 \
例如，第2次选择本专长时，你必须拥有至少3个满足条件的职业，才能再次获得该加值。

#### 战地施法者
*典范* \
你获得以下增益：
- **专注**：你为法术进行专注检定时，可以使用施法属性作为关键属性。
- **成分**：即使你的主手已持用武器・盾牌，也依然能够满足法术的姿势・材料成分。
- **法术反击**：你可以消耗通过反击发动的武器攻击来执行【魔法】动作，以此发动最多1次法术攻击 (即便该动作通常能够发动多次攻击)。 \
  在决定法术反击能否命中时，将远程法术攻击的常规射程视为施法距离与6格中的较低者。 \
  该反击不会影响你与攻击者之外的对象。若为该反击消耗法术位，其环阶不能超过［最大环阶-2］(最低1) 环。
- **响应法术**：当一个生物因移动引发你的借机攻击时，可以改为消耗1次借机攻击来执行【魔法】动作，以此对该生物发起1次敌对行为。 \
  该行为不会影响你与此生物之外的对象。若为该行为消耗法术位，其环阶不能超过［最大环阶-2］(最低1) 环。

#### 驭法攻击
*典范* \
**先决**：SPR4+ \
每回合1次，你在自己的回合执行【魔法】动作时，可以使用附赠动作额外以单手发动1次武器攻击。 \
这次额外攻击，以及通过【魔法】动作进行的攻击检定及豁免DC，都会承受2点减值。 \
你无法将AKM加入这次武器攻击的伤害检定，除非该调整值为负值。 \
**特殊**：使用激活「迅击」精通词条的武器发动这次额外攻击时，你可以将本次攻击作为【魔法】动作的一部分，无需使用附赠动作。
