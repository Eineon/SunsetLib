## 基础调整
- 防守与反击统称为「防御」
- 灵能力描述不再有字数限制
- 单项属性值最高不能超过 ⌊灵识÷2+1⌋
- 体魄属性到达4、6、8级时，分别获得1点技能点，只能加在运动、操作或隐秘上
- 心魂属性5级起，每级额外增加1点灵力值上限
- 补充道具点时，会恢复所有已消耗的道具点
- 当角色从当前场景进入下个场景时，会恢复为使用时髦动作而消耗的初始时髦值，最多为其总值的2/5
- 若角色的灵能力标签少于3个，则可以在激活/解放标签时将单个灵能力标签视为多个标签，总数依然不能超过3个
- 战斗中，消耗道具点创造道具可以作为任意行动的一部分
- 反击及连续反击不再减少CAT，但反击失败时检定结果视为0 (专长【反击风暴】能够为反击检定+1D4，并将反击失败时的检定结果视为10)
- 持续至战斗结束的效果，能够在战斗外持续10分钟

### 检定
进行检定时，在1D20中掷出的点数被称为**骰值**，骰值加上其他修正值 (包括加骰) 的结果被称为**判定值**，这些修正值的总和被称为**总修正**。 \
进行拥有关键属性的检定时，会将对应属性值作为修正值加在检定的骰值上。

#### 修正值
所有加值与减值都属于修正值，并总是为非0的自然数。未说明属于加值还是减值的修正则可以是负数。 \
检定获得的加值会与其他修正值相加，减值则会与其他修正值相减。未说明属于加值还是减值的修正，在为自然数时，视为加值处理；为负数时，将其绝对值视为减值处理。

#### 冠名修正值
同名的同类型修正值对同一项目的修正不会叠加，数值最高的修正值会压制其他修正值。在将属性值作为修正值时也会视其为与属性名称相同的冠名修正值。

#### 危害与抵抗
危害检定与抵抗检定是一种对抗检定，通常用于决定施加者对目标造成的危害程度。
- **危害检定**：进行1次攻击检定，并用8取代1D20的结果。危害检定的判定值也被称为**危害值**。
- **抵抗检定**：目标必须针对危害检定，使用由危害决定的关键属性进行1次抵抗检定。若抵抗检定的判定值不超过危害值，则代表抵抗失败，反之则为抵抗成功。

### 额外专精增益
根据灵能力的专精方面，角色可以从以下效果中选择1项获得。
1. 生命值上限+1
2. 灵力值上限+2
3. 道具点上限+1
4. 初始时髦值+5
5. 初始机运值+2
6. 技能点+2
7. 选择1种技艺系别，攻击检定+1

### 特质标签
特质标签分为3种类型：身份、性格、信念。在一开始，角色拥有每种特质标签各1个。
- **身份**：你在凡俗世界的表面身份。
- **性格**：你的人格中最鲜明的特点。
- **信念**：你之所以投身战斗的理由。

就像其他标签那样，角色可以尝试激活特质标签为行动检定提供加骰。根据DH的判断，特质标签有时会使行动检定承受减骰。 \
每有1个被激活的特质标签造成减骰，角色就会在检定结束后获得10点时髦值。

### 机运值
机运可以是负值，初始为0。角色会在行动检定大成功时获得1点机运，大失败时失去1点机运。大成功与大失败不会再改变CAT。 \
若 [机运>0]，角色可以消耗1点机运 (最低消耗至0)，对与自身相关的1次行动检定造成以下效果之一。 \
若 [机运<0]，与角色相关的行动检定发生时，DH、或是与本次检定相关的其他角色，可以使该角色提升1点机运 (最高提升至0)，令本次检定受到以下效果之一。 \
Ⅰ. 投掷1D20时，掷2次并采用较高的结果。 \
Ⅱ. 投掷1D20时，掷2次并采用较低的结果。 \
机运的效果不会叠加。每当间章结束时，所有角色的机运都会重置为初始值。

### 聚能值
角色的（聚能值:FP）上限为 ⌈(体魄+智慧)÷2⌉ 点。因进行【显能】、使用专长、或在防御中激活灵能力标签而消耗MP时，可以将其中一部分或全部替换为FP消耗。 \
当角色从当前场景进入下个场景时，会恢复所有已消耗的FP。

### 执念
每个恶灵拥有最多3个执念标签。当发生任何与恶灵相关的检定时，在掷骰之前，狩魂者可以激活任意个该恶灵的已知执念标签。 \
每激活1个执念标签，都会使CAT+1或-1，由激活标签的角色决定。

PL需要描述狩魂者使用执念标签的方式，例如：
- 唤起恶灵的人性
- 用敏感信息使其失去防备
- 激怒或分散其注意力
- 揭示其弱点

注意：相同的使用方式无法对同一个恶灵重复生效；一名角色每轮只能激活1次执念标签。


## 灵能
原本的【驱使灵能力】、【施展武技】与【施展术法】行动合并为【显能】行动。 \
角色可以将已习得的**技艺**编织为「术式」，再通过【显能】行动将其展现。

### 即兴显能
角色能够以一种更简单的方式进行【显能】，而无需使用术式，具体参考以下流程。
1. 描述你的动机和行为，并宣言任意种附魔，作为本次显能的愿景
2. 你的行为必须与愿景相匹配，否则显能将会失败，具体由DH裁定
3. 由DH根据你的行为决定进行什么检定，并为激活的每个灵能力标签支付1MP
4. 若愿景仅包含你的灵能力专精附魔，则检定获得1D4的专精加骰
5. 将CAT自由分配在愿景包含的附魔上，作为本次显能的效果

宣言附魔时，可以宣言任何通用附魔，也可以宣言包含在已习得技艺中的附魔。

在战斗之外，角色无法通过即兴显能为自己恢复的MP。

## 技艺
技艺分为3种系别，分别对应3种属性：（武术:体魄）、（法术:智慧）、（灵术:心魂）。「通用」会被同时视为所有系别。 \
在任意属性上每有1个等级，就可以习得1项对应系别的标准技艺。若属性到达7级，则可以改为习得对应系别的秘传技艺。 \
此外，角色会自动习得所有满足前置条件的基础技艺。 \
若无特别说明，技艺造成的附魔POW总是为1级。

### 编织术式
角色可以在间章将技艺编织为术式。术式由1项主要技艺与任意项次要技艺构成，所有技艺必须拥有相同的系别。 \
主要技艺无需术式即可通过【显能】直接展现，次要技艺则只能在处于术式内时生效。

#### 术式槽
角色拥有 [灵识-7] 格术式槽。每格术式槽都可以保存1道术式。只有保存在术式槽内的术式能够进行【显能】。角色可以在间章进行术式的保存与更换。

### 展现术式
展现拥有多种系别的主要技艺时，需要施术者从其中选择1种作为该技艺在本次显能期间的唯一系别。 \
主要技艺系别的对应属性会被作为本次显能的关键属性。这也会成为【显能】攻击检定的关键属性。
