## 基础调整
- 移除强度与专长
- 灵能力描述不再有字数限制
- 单项属性值最高不能超过 ⌊灵识÷2+1⌋
- 体魄属性到达3、5、7、9级时，分别获得1点技能点
- 心魂属性5级起，每级额外增加1点灵力值上限
- 单项技能等级最高不能超过 ⌊灵识÷2-2⌋
- 补充道具点时，会恢复所有已消耗的道具点
- 当角色从当前场景进入下个场景时，会恢复为使用时髦动作而消耗的初始SP，最多为其总值的2/5
- 若角色的灵能力标签少于3个，则可以在激活/解放标签时将单个灵能力标签视为多个标签，总数依然不能超过3个
- 对抗检定中，主动发起检定的一方被称为「挑战方」，被动卷入的一方被称为「应对方」
- 将CAT增加/减少X级的效果，等同于对判定值造成5X点加值/减值
- 战斗中，消耗道具点创造道具可以作为任意行动的一部分
- 战斗中，计算效果的时间时，自受到效果的回合结束起，当受到效果的角色回合结束时，视为经过1轮

### 检定
进行检定时，在1D20中掷出的点数被称为**骰值**，骰值加上其他修正值 (包括加骰) 的结果被称为**判定值**，这些修正值的总和被称为**总修正**。 \
进行拥有关键属性的检定时，会将对应属性值作为修正值加在检定的骰值上。

#### 修正值
所有加值与减值都属于修正值，并总是为非0的自然数。未说明属于加值还是减值的修正则可以是负数。 \
检定获得的加值会与其他修正值相加，减值则会与其他修正值相减。未说明属于加值还是减值的修正，在为自然数时，视为加值处理；为负数时，将其绝对值视为减值处理。

#### 冠名修正值
同名的同类型修正值对同一项目的修正不会叠加，数值最高的修正值会压制其他修正值。在将属性值作为修正值时也会视其为与属性名称相同的冠名修正值。

#### 格挡与反制
格挡检定与反击检定是一种对抗检定，通常用于决定应对方针对攻击的防御程度。 \
在防御时进行的格挡检定与反击检定，会被称为**防御检定**，其判定值则称为**防御值**。
- **格挡检定**：描述你的行为，进行1次与行为匹配的技能检定 (没有关键属性)，并用10取代1D20的骰值。
- **反制检定**：进行1次攻击检定。若防御值低于攻击检定的判定值，则将防御值视为0。若防御时进行的反制检定成功，则可以对攻击者瞬念显能【进攻】技艺，不进行攻击检定，直接使用本次反制的CAT。

#### 危害与抵抗
危害检定与抵抗检定是一种对抗检定，通常用于决定挑战方对目标造成的危害程度。
- **危害检定**：进行1次攻击检定，并用10取代1D20的骰值。危害检定的判定值也被称为**危害值**。若危害值与抵抗检定的判定值相等，则将危害检定的CAT视为1。
- **抵抗检定**：描述你的行为，进行1次与行为匹配的技能检定，关键属性由危害决定。若抵抗检定的判定值未超过危害值，则代表抵抗失败，反之则为抵抗成功。

### 技能
移除「狩魂学识」之外的所有技能，添加以下技能。技能不与任何属性绑定，由DH根据情景决定每次技能检定的关键属性。
- **执进**：身体机能的高效运用能力。
- **运筹**：战术规划与长期布局能力。
- **调查**：知识储备与逻辑推理能力。
- **洞悉**：基于经验形成的预知能力。
- **表达**：辐射影响与情绪传递能力。
- **共鸣**：情感共有与创伤抚慰能力。

### 生命值
狩魂者的HP上限调整为 ⌊1+(体魄+心魂)÷2⌋ 点。任何导致角色损失HP的效果，都视为使角色受到等量的**损伤**。 \
损伤可以超过HP上限，无论承受多少损伤，角色的行动都不会受到影响。 \
若在损伤到达HP上限时承受未标记为「非致命」的损伤，则会消除所有损伤，并在解体护符上划1斩痕。

#### 解体护符
解体护符会在创建角色时生成，容量为3斩。划在该护符上的每道斩痕都代表角色身心受到的直接伤害，例如骨折、失血、力竭、精神污染。 \
在解体护符上留下斩痕的损伤，若一次性造成的损伤值超过HP上限，则会在解体护符上额外划1斩痕。 \
根据划下斩痕的情景，每道斩痕都会成为**重创标签**，在被激活时损失1HP，并使检定的CAT减少1级。 \
当解体护符划满3斩时，角色将陷入无法行动的状态，此时承受任何损伤都会再扣除5倍的时髦值。
- **消除斩痕**：进入间章时，角色可以消除1道解体斩痕。角色还可以在间章花费1点经验消除1道解体斩痕。
- **经验奖励**：在2次间章之间，被激活1个不同的重创标签，角色就会获得1点经验。

### 额外专精增益
每次获得灵能力专精时，角色还可以从以下效果中选择1项获得。
1. 生命值上限+1
2. 灵力值上限+2
3. 道具点上限+1
4. 初始机运值+2
5. 初始时髦值+5
6. 技能点+2
7. 即兴显能时的行动检定+1
8. 选择1种技艺系别，攻击检定+1
9. 选择1种技艺系别，应对该系别的对抗检定+1

### 特质标签
PL可以从自己的角色设定中标记最多5个特质标签，例如身份、性格、目标等等。 \
角色可以尝试激活特质标签为行动检定提供1D4的加骰，或者也可以选择造成减骰。 \
特质标签提供的加骰会像时髦骰那样增加角色的SP。而每个由特质标签造成的减骰，都会使角色在检定结束后获得5SP。

PL可以在间章向DH描述自己角色的转变，随后修改特质标签。

### 机运值
机运可以是负值，初始为3。角色会在行动检定大成功时获得1点机运，大失败时失去1点机运。大成功与大失败不会再改变CAT。 \
若 [机运>0]，角色可以消耗1点机运 (最低消耗至0)，对与自身相关的1次行动检定造成以下效果之一。 \
若 [机运<0]，与角色相关的行动检定发生时，DH、或是与本次检定相关的其他角色，可以使该角色提升1点机运 (最高提升至0)，令本次检定受到以下效果之一。 \
Ⅰ. 投掷1D20时，掷2次并采用较高的结果。 \
Ⅱ. 投掷1D20时，掷2次并采用较低的结果。 \
角色可以在掷骰后，DH宣布CAT前使用机运。机运的效果不会叠加。若一次检定同时受到Ⅰ与Ⅱ效果影响，这次检定将免疫机运的效果。 \
每当间章结束时，所有角色的机运都会重置为初始值。

### 聚能值
角色的（聚能值:FP）上限为 ⌈(体魄+智慧)÷2⌉ 点。因使用技艺、或在防御中激活灵能力标签而消耗MP时，可以将其中一部分或全部替换为FP消耗。 \
当角色从当前场景进入下个场景时，会恢复所有已消耗的FP。

### 执念
每个恶灵拥有最多3个执念标签。当发生任何与恶灵相关的检定时，在掷骰之前，狩魂者可以激活任意个该恶灵的已知执念标签。 \
每激活1个执念标签，都会使CAT+1或-1，由激活标签的角色决定。

PL需要描述狩魂者使用执念标签的方式，例如：
- 唤起恶灵的人性
- 用敏感信息使其失去防备
- 激怒或分散其注意力
- 揭示其弱点

注意：相同的使用方式无法对同一个恶灵重复生效；一名角色每轮只能激活1次执念标签。

## 灵能
原本的【驱使灵能力】、【施展武技】与【施展术法】行动合并为【显能】行动。角色可以通过【显能】行动展现**技艺**。

### 即兴显能
角色能够以一种更简单的方式进行显能，而无需使用技艺，具体参考以下流程。
1. 描述你的动机和行为，并宣言任意个附魔，作为本次显能的愿景
2. 你的行为必须与愿景相匹配，否则显能将会失败，具体由DH裁定
3. 由DH根据你的行为决定进行什么检定，关键属性通常为心魂
4. 若愿景包含灵能力专精附魔，则检定获得1D4点专精加值
5. 将CAT自由分配在愿景包含的附魔上，所有持续附魔只会存在1轮
6. 至少为愿景包含的每个附魔分配1级CAT，否则将导致显能失败

宣言附魔时，可以宣言任何通用附魔，也可以宣言包含在已准备技艺中的附魔。 \
在战斗之外，角色无法通过即兴显能为自己恢复的MP。

即兴显能所能发挥的最大力量由其【叙事】POW决定，如下表所示。若【叙事】是愿景的唯一附魔，则可以在1D20的骰值低于11时视为11。

## 技艺
技艺分为2种系别，对应2种属性：（武术:体魄）与（法术:智慧）。「通用」会被同时视为所有系别。 \
在体魄或智慧属性上每有1个等级，就可以习得1项对应系别的标准技艺。若属性到达7级，则可以改为习得对应系别的进阶技艺。 \
此外，角色会自动习得所有满足前置条件的基础技艺。 \
若无特别说明，技艺造成的所有附魔都视为分配了 [CAT] 级CAT，若无CAT则将POW视为1级。

### 准备与展现技艺
游戏开始前，或是在间章中，角色可以准备任意项技艺，但这些技艺的**负荷**总和不能超过 [灵识-4] L。 \
角色只能展现已准备的技艺。展现拥有多种系别的技艺时，需要从其中选择1种作为本次显能期间的唯一系别。 \
技艺系别的对应属性会被作为本次显能的关键属性，也会成为其攻击检定的关键属性。

#### 强化技艺
展现任何技艺时，角色都可以尝试激活灵能力标签，并描述这些标签强化技艺的方式。 \
每激活1个灵能力标签用于强化技艺，都会使技艺的费用+1。不需要再额外为激活灵能力标签支付MP。

#### 效果叠加
无论技艺是否提供多种效果选项、或是否给予叠加性为「可」的附魔，同名持续性技艺的效果均不会叠加。 \
若不可叠加的效果会影响同一个事件，则效果最强、且最新造成的效果会压制其他效果，具体由DH裁定。
