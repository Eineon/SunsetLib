# 追加数据

## 附魔
所有附魔都拥有（附魔等级:POW），每在1个X阶附魔上分配X级CAT，就会提升1级该附魔的POW。

### 附魔词条

#### 时效
附魔的时效分为以下几种类型。
- **立即**：效果只会存在一瞬间。
- **永续**：效果会一直存在，直到你主动解除。
- **条件**：效果持续至条件被满足。
- **场景**：效果持续至当前场景结束。
- **N轮**：效果持续至经过N轮。自受到效果的回合结束起，当受到效果的角色回合结束时，视为经过1轮。附魔效果中的「轮」也依此计数。

##### 伤害与治疗
原本的伤害类型将作为属性标签附加在伤害上。伤害现在分为以下几种类型。复合伤害/治疗属于二阶附魔，属性伤害/治疗属于三阶附魔。 \
【生命治疗】取代【恢复】附魔、【灵力治疗】取代【增加灵力】附魔、【灵力伤害】取代【减少灵力】附魔。

## 技艺

### 技艺条目

#### 前置
角色无法习得未满足前置条件的技艺。若已习得的技艺前置条件失效，角色将暂时失去该技艺，直到条件被重新满足。

#### 费用
进行【显能】时，角色必须一次性消耗等于所用技艺费用总和的MP，否则将会显能失败。 \
若费用总和超过主要技艺系别的对应属性，也会导致显能失败。

#### 标签
- **维持**：记作 (维持_N)。每当你的回合开始时，必须为本技艺消耗N点MP，否则该技艺的效果将提前结束。
- **符箓**：可以消耗1KP代替需要因本技艺的费用而消耗的MP。
- **祷念**：指示类标签，无特殊效果。
- **通灵**：可以消耗任意点HP，代替双倍的因本技艺而消耗的MP。

### 技艺词条

#### 复制
复制特定技艺的效果，并在不支付代价的情况下发动。

#### 代价
【代价】词条后所述的文本，代表除了**费用**之外，发动本技艺所需额外支付的代价。 \
若无法支付代价，该技艺不会在本次显能中发动，但也不会导致显能失败。

#### 增幅
记作【增幅：N】。发动本技艺时，可以额外支付N (若为X则最低为1) 点MP作为代价，适用后方所述的效果。

#### 连携
记作【连携：条件】。所述条件被满足时，只要角色已习得该技艺，无论是否发动该技艺，都会立即适用后方的连携效果。 \
若满足条件的行为包含在【显能】的过程中，则其连携效果也会被附加到这次显能的效果内，成为整体效果的一部分。 \
若无特别说明，连携条件被重复满足时，连携效果也会被重复触发。

##### 连携条件
- **习得**：习得本技艺时满足条件。若失去本技艺，也会随之失去连携效果，直到重新获得本技艺。
- **斩杀**：通过本技艺的效果将不为【爪牙】的敌人HP降至0时满足条件。
- **咏唱_X**：根据技艺内容获得第X点咏唱时满足条件，分别计数不同技艺的咏唱点数。
- **自身_标签/附魔**：自身拥有所述标签/附魔期间满足条件。
- **目标_标签/附魔**：若本技艺的目标拥有所述标签/附魔则满足条件。

### 技艺列表

#### 基础技艺

##### 进攻
*主要技艺 武术* \
**费用**：0 \
对1名目标进行1次攻击检定，成功则造成【生命伤害(物理)】附魔。

##### 擒拿
*主要技艺 武术* \
**费用**：0 \
对1名目标进行1次危害检定，目标需要进行体魄抵抗。 \
若抵抗失败，则对目标造成【武术破绽_1】附魔。

##### 诅咒
*主要技艺 法术* \
**费用**：0 \
对1名目标进行1次危害检定，目标需要进行智慧抵抗。 \
若抵抗失败，则对目标造成【法术破绽_1】附魔。

##### 噪点
*主要技艺 灵术* \
**费用**：0 \
对1名目标进行1次危害检定，目标需要进行心魂抵抗。 \
若抵抗失败，则对目标造成【灵术破绽_1】附魔。

##### 灵迹
*主要技艺 法术* \
**费用**：0 \
在场景中留下对无灵力生物不可见的痕迹，或是定位身边的这种痕迹。 \
你也可以使用这项技艺抹除自身以此留下的痕迹。

##### 灵视
*主要技艺 法术/灵术* \
**费用**：0 \
你获得【灵视】附魔。 \
【灵视】：【正面|永续|一阶】你可以看见一些常人看不见的东西。进行调查/洞悉技能检定时，可以消耗2MP换取1D4点加值。

##### 爆裂
*主要技艺 法术/灵术 (符箓，祷念，通灵)* \
**费用**：0 \
对1名目标进行1次攻击检定，成功则造成【生命伤害】附魔，伤害属性由你展现本技艺的方式决定。

##### 医护
*主要技艺 通用* \
**费用**：1 \
选择1名自愿目标。描述你的行为，由DH依此决定你要进行什么检定。 \
若检定成功，则造成【治疗】附魔，治疗类型必须与行为匹配。

##### 支援
*主要技艺 通用* \
**费用**：1 \
选择1名自愿目标。描述你的行为，由DH依此决定你要进行什么检定。 \
若检定成功，则将CAT分配在与行为匹配的附魔上：强化○○、叙事、赋予抗性。

##### 妨碍
*主要技艺 通用* \
**费用**：1 \
选择1名目标。描述你的行为，由DH依此决定你要进行什么检定。 \
若检定成功，则将CAT分配在与行为匹配的附魔上：削弱○○、叙事、赋予易伤。
