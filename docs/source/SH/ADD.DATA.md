# 追加数据

## 附魔
所有附魔都拥有（附魔等级:POW），每在1个X阶附魔上分配X级CAT，就会提升1级该附魔的POW。 \
若附魔后方列出数字，则无法为其分配CAT，将列出的数字作为其POW。

### 附魔词条

#### 叠加
角色拥有多个叠加为「可」的同名附魔时，将这些附魔的POW相加来决定效果。 \
角色拥有多个叠加为「否」的同名附魔时，分别处理这些附魔的效果。若这些附魔会影响同一个事件，则POW最高、且最晚获得的附魔会压制其他附魔。

##### 伤害与治疗
原本的伤害类型将作为属性标签附加在伤害上。伤害现在分为以下几种类型。 \
若一次伤害同时拥有多个属性标签，造成伤害的角色必须决定本次伤害的唯一属性。 \
复合伤害/治疗属于二阶附魔，属性伤害/治疗属于三阶附魔。

## 技艺

### 技艺条目

#### 时机
- **主动**：主动技艺需要角色通过【显能】将其展现才能发动效果。
- **被动**：被动技艺无需角色进行【显能】也会自行发动效果。若无特别说明，角色无法主动终止被动技艺。

#### 标签
- **符箓**：可以消耗1KP代替费用产生的MP消耗。无法代替「增幅」产生的MP消耗。
- **祷念**：指示类标签，无特殊效果。
- **通灵**：可以消耗任意点HP，代替双倍的MP消耗。

#### 前置
角色无法习得未满足前置条件的技艺。若已习得的技艺前置条件失效，角色将暂时失去该技艺，直到条件被重新满足。

#### 费用
进行【显能】时，角色必须一次性消耗等于技艺费用的MP，否则将会显能失败。若费用超过技艺系别的对应属性，也会导致显能失败。
- **零与无**：为「0」的费用可以被修正值影响，为「无」则否。
- **X**：若无特别说明，为「X」的费用数值由角色在每次显能时自行决定，最低为1。影响费用的修正值会作用在决定数值之后。

#### 时效
持续时间超过一瞬间的技艺会拥有该条目。未显示时效条目的技艺，其时效视为「立即」。
- **立即**：效果只会存在一瞬间。
- **永续**：效果会一直存在，直到你主动解除。
- **条件**：效果持续至条件被满足。
- **场景**：效果持续至当前场景结束。
- **N轮**：效果持续至经过N轮。
- **维持**：记作「维持_N」。效果持续至场景结束。每当你的回合开始时，必须为本技艺消耗N点MP，否则效果将提前结束。

#### 触发
拥有该条目的技艺需要作为对特定事件的响应而展现，无法直接通过【显能】展现。 \
以此展现的技艺不会占用角色的行动次数，并且可以在回合外展现。

### 技艺词条

#### 复制
复制特定技艺的效果，并在不支付费用及代价的情况下发动。

#### 代价
「代价」词条后所述的文本，代表除了**费用**之外，发动本技艺所需额外支付的代价。若无法支付代价，将会显能失败。

#### 连携
记作「连携：条件」。所述条件被满足时，只要角色已习得该技艺，无论是否发动该技艺，都会立即适用后方的连携效果。 \
若满足条件的行为包含在【显能】的过程中，则其连携效果也会被附加到这次显能的效果内，成为整体效果的一部分。 \
若无特别说明，连携条件被重复满足时，连携效果也会被重复触发。

##### 连携条件
- **启动**：后方所述的文本激活连携效果。
- **习得**：习得本技艺时满足条件。若失去本技艺，也会随之失去连携效果，直到重新获得本技艺。
- **斩杀**：通过本技艺的效果将不为【爪牙】的敌人HP降至0时满足条件。
- **咏唱_X**：根据技艺内容获得第X点咏唱时满足条件，分别计数不同技艺的咏唱点数。
- **自身_标签/附魔**：自身拥有所述标签/附魔期间满足条件。
- **目标_标签/附魔**：若本技艺的目标拥有所述标签/附魔则满足条件。

#### 增幅
记作「增幅：N」。发动本技艺时，可以额外支付N (若为X则最低为1) 点MP作为代价，适用后方所述的效果。

### 技艺列表
